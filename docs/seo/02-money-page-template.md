# Money Page Template

ทุกหน้า money page ของ amphon.co.th ต้องมีโครงสร้างนี้

**Positioning:** เขียนเป็น **national intent / ทั่วประเทศ** — keyword หลักคือประเภทสินค้า (เช่น รับซื้อคอมสำนักงาน) ไม่ใช่พื้นที่ อุบลราชธานีใช้เฉพาะตอนพูดถึงหน้าร้านจริงเท่านั้น

## 1. SEO Title
- มี keyword หลัก (ประเภทสินค้า)
- มีจุดขายปลอดภัย เช่น ประเมินฟรี, ส่งรูปเช็กราคา, ประเมินตามสภาพจริง
- **ไม่บังคับ**ใส่อุบลราชธานีหรือภาคอีสานใน title
- ไม่ใช้คำ claim แรง

ตัวอย่างดี:
`รับซื้อคอมสำนักงาน ส่งรูปประเมินฟรี รับยกล็อต | อำพล เทรดดิ้ง`

ตัวอย่างหลีกเลี่ยง:
`รับซื้อคอมสำนักงาน อุบลราชธานี ภาคอีสาน ราคาสูง`

## 2. Meta Description
- ประมาณ 140-160 ตัวอักษร
- บอกว่ารับซื้ออะไร
- เหมาะกับใคร
- ลูกค้าทั่วประเทศส่งรูปประเมินได้
- มีหน้าร้านจริงที่อุบลราชธานี (trust proof สั้นๆ)

## 3. H1
- มี keyword หลัก
- H1 มีแค่ 1 อัน
- ไม่จำเป็นต้องมีชื่อจังหวัด

## 4. Hero Section
ต้องตอบทันทีว่า:
- รับซื้ออะไร
- ลูกค้ากลุ่มไหนเหมาะ (ทั่วประเทศ)
- ส่งรูปหรือรายการประเมินได้อย่างไร
- ทำไมควรไว้ใจร้าน (หน้าร้านจริง + ประเมินโปร่งใส)

## 5. Trust Bar
ใส่ 4-5 จุด:
- มีหน้าร้านจริงที่อุบลราชธานี
- ส่งรูปประเมินฟรี (ทั่วประเทศ)
- ประเมินตามสภาพจริง
- รับทั้งรายเครื่องและยกล็อต
- จ่ายเงินทันทีเมื่อตกลงราคา

## 6. รับซื้ออะไรบ้าง
แยก 4-6 หมวดย่อยตามสินค้านั้น เช่น:
- เครื่องใช้งานปกติ
- เครื่องมีตำหนิ
- เครื่องบริษัท
- เครื่องยกล็อต
- อุปกรณ์เสริม
- รุ่น/แบรนด์ที่พบบ่อย

## 7. ต้องส่งข้อมูลอะไรเพื่อประเมินราคา
ต้องมี:
- รูปสินค้า
- รุ่น/แบรนด์
- สเปก
- อาการหรือสภาพ
- จำนวนเครื่อง
- อุปกรณ์ที่มี
- พื้นที่ของลูกค้า (เพื่อสรุปขั้นตอนนัดรับ/จัดส่ง)

## 8. ขั้นตอนรับซื้อ
1. ส่งรูปหรือรายการสินค้า (LINE @webuy)
2. ประเมินราคาเบื้องต้น
3. นัดตรวจเช็ก / จัดส่ง / นำมาที่ร้าน (ตามประเภทสินค้า จำนวน และพื้นที่)
4. สรุปราคาและจ่ายเงิน

## 9. ราคาขึ้นกับอะไร
- รุ่น
- ปี
- สเปก
- สภาพ
- อุปกรณ์
- จำนวนเครื่อง
- ความต้องการของตลาด

## 10. ทำไมควรเลือกอำพล เทรดดิ้ง
- มีหน้าร้านจริงที่อุบลราชธานี — ตรวจสอบได้ ไม่ใช่เว็บไม่มีตัวตน
- เข้าใจสินค้าไอที
- ประเมินก่อน ไม่บังคับขาย
- เหมาะกับลูกค้าทั่วไปและองค์กรทั่วประเทศ
- ลูกค้าต่างจังหวัดส่งรูปหรือรายการประเมินได้
