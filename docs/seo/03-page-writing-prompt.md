# Page Writing Prompt — amphon.co.th Money Pages

ใช้ prompt นี้เมื่อสร้างหรือ rewrite หน้า money page ใน `src/content/services/`

---

## Context

amphon.co.th คือเว็บหลักของ **อำพล เทรดดิ้ง** — บริการรับซื้อสินค้าไอทีมือสอง **ระดับประเทศ**

- มีหน้าร้านจริง: 740/8 ถนนชยางกูร อ.เมือง อุบลราชธานี
- LINE: @webuy | โทร: 064-257-9353

## Positioning Rules (บังคับ)

1. **National intent** — เขียนให้ลูกค้าทุกจังหวัดรู้สึกว่าใช้บริการได้ ไม่ใช่เว็บรับเฉพาะอุบลหรือภาคอีสาน
2. **อุบลราชธานี = trust proof เท่านั้น** — ใช้เมื่อพูดถึงที่ตั้งหน้าร้านจริง สำนักงานใหญ่ หรือช่องทางนำสินค้ามาที่ร้าน
3. **อย่าเขียน headline หลักแบบ regional** — หลีกเลี่ยง title/H1/intro ที่เน้น "ภาคอีสาน" "อุบลราชธานี" เป็นคำหลัก
4. **ลูกค้าต่างจังหวัดส่งรูปหรือรายการประเมินได้** — ต้องมีใน intro หรือขั้นตอนประเมิน
5. **นัดรับ / จัดส่ง / ตรวจเช็ก** — ระบุว่าขึ้นกับประเภทสินค้า จำนวน และพื้นที่ ไม่สัญญาว่ารับทุกพื้นที่ทุกกรณี
6. **Claim-safe** — ดู `03-claim-style-guide.md` ห้ามใช้คำโอ้อวด

## Writing Prompt Template

```
เขียนหน้า money page สำหรับ amphon.co.th

หัวข้อ: [ชื่อบริการ / mainKeyword]
กลุ่มลูกค้า: [เช่น องค์กร / บุคคลทั่วไป / เกมเมอร์]

Positioning:
- เว็บรับซื้อสินค้าไอทีมือสองระดับประเทศ (national intent)
- มีหน้าร้านจริงที่อุบลราชธานี ใช้เป็น trust proof
- ลูกค้าทั่วประเทศส่งรูปหรือรายการประเมินทาง LINE @webuy ได้
- การนัดรับ/จัดส่ง/ตรวจเช็กขึ้นกับประเภทสินค้า จำนวน และพื้นที่

ห้าม:
- เขียนให้เหมือนรับเฉพาะอุบลหรือภาคอีสาน
- ใส่อุบล/อีสานใน title หรือ H1 เป็นคำหลัก
- ใช้ claim แรง (ดีที่สุด, ราคาสูงสุด, อันดับ 1, รับทุกรุ่นทุกสภาพ)

ต้องมี:
- frontmatter ตาม amphon's frontmatter guide
- โครงสร้างตาม 02-money-page-template.md
- FAQ อย่างน้อย 1 ข้อเรื่องลูกค้าต่างจังหวัด
- CTA: LINE @webuy, โทร 064-257-9353
```

## Geographic Copy — ตัวอย่างที่ใช้ได้

| สถานการณ์ | ตัวอย่างประโยค |
|-----------|----------------|
| Intro ระดับประเทศ | อำพล เทรดดิ้ง รับซื้อ [สินค้า] มือสองทั่วประเทศ ส่งรูปประเมินได้ก่อนตัดสินใจ |
| Trust proof | มีหน้าร้านจริงที่อุบลราชธานี ตรวจสอบได้ ไม่ใช่เว็บไม่มีตัวตน |
| ลูกค้าต่างจังหวัด | ลูกค้าต่างจังหวัดส่งรูปและรายละเอียดทาง LINE ได้ก่อน |
| นัดรับ/จัดส่ง | การนัดรับถึงที่หรือจัดส่งขึ้นกับประเภทสินค้า จำนวน และพื้นที่ — แจ้งรายละเอียดมาก่อน |
| หน้าร้าน | นำสินค้ามาตรวจเช็กที่ร้าน 740/8 ถนนชยางกูร อุบลราชธานี ได้ |

## Geographic Copy — หลีกเลี่ยง

- ❌ "รับซื้อ [สินค้า] ภาคอีสาน" (เป็น H1 หลัก)
- ❌ "บริการครอบคลุมทั่วภาคอีสาน"
- ❌ "ทีมงานวิ่งรับทั่วอีสาน" (โดยไม่มีเงื่อนไข)
- ❌ "เหมาะสำหรับลูกค้าในอุบลและจังหวัดใกล้เคียงเท่านั้น"
- ❌ ใส่รายชื่อจังหวัดอีสาน 8–10 จังหวัดเป็นหัวข้อหลักของ money page
