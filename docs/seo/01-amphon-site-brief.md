# Amphon.co.th Site Brief

## Brand
amphon.co.th คือเว็บหลักของ **อำพล เทรดดิ้ง** — บริการรับซื้อสินค้าไอทีมือสองระดับประเทศ ครอบคลุมคอมพิวเตอร์ โน๊ตบุ๊ค มือถือ iPad MacBook Server อุปกรณ์เครือข่าย และสินค้าไอทีองค์กร มีหน้าร้านจริงที่อุบลราชธานี (740/8 ถนนชยางกูร) ใช้เป็นจุดตรวจสอบและ trust proof

## Positioning
เว็บนี้ต้องวางตัวเป็น **เว็บรับซื้อสินค้าไอทีมือสองระดับประเทศ (national intent)** ไม่จำกัดเฉพาะอุบลราชธานีหรือภาคอีสาน

จุดขายหลัก:
- รับซื้อสินค้าไอทีมือสองทั่วประเทศ — ลูกค้าต่างจังหวัดส่งรูปหรือรายการประเมินได้
- มีหน้าร้านจริงที่อุบลราชธานี (ไม่ใช่เว็บ mass SEO ไม่มีตัวตน)
- ประเมินตามสภาพจริง ส่งรูปเช็กราคาเบื้องต้นได้
- จ่ายเงินทันทีเมื่อตกลงราคา

**อุบลราชธานี** = ที่ตั้งหน้าร้านจริง / สำนักงานใหญ่ / trust proof เท่านั้น  
**ไม่ใช่** = ข้อจำกัดว่ารับเฉพาะลูกค้าในอุบลหรือภาคอีสาน

| ช่องทาง | รายละเอียด |
|--------|------------|
| **ประเมินออนไลน์** | ลูกค้าทั่วประเทศส่งรูปหรือรายการสินค้าทาง LINE @webuy ได้ |
| **หน้าร้าน** | นำสินค้ามาที่ร้าน 740/8 ถนนชยางกูร อ.เมือง อุบลราชธานี |
| **นัดรับ / จัดส่ง / ตรวจเช็ก** | ขึ้นกับประเภทสินค้า จำนวน และพื้นที่ — แจ้งรายละเอียดก่อนเพื่อสรุปขั้นตอน |
| **งานองค์กร / ยกล็อต** | รับพิจารณาทั่วประเทศ นัดสำรวจหน้างานหรือจัดส่งตามขนาดล็อต |

พื้นที่ใกล้เคียงอุบล (ศรีสะเกษ ยโสธร อำนาจเจริญ สุรินทร์ ฯลฯ) อาจสะดวกสำหรับนัดรับถึงที่ — แต่**ไม่ใช่ขอบเขตบริการหลัก** อย่าเขียนให้ดูเหมือนรับเฉพาะภาคอีสาน

## Words to Avoid (Geographic Framing)

- ติดต่อสะดวกในภาคอีสาน (แทนที่จะเป็นทั่วประเทศ)
- ทีมงานวิ่งรับทั่วภาคอีสาน (ถ้าไม่มีเงื่อนไขชัดเจน)
