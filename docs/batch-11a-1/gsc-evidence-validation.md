# GSC Evidence Validation (Batch 11A.1)

> [!WARNING]
> **สถานะหลักฐาน:** ยังไม่สามารถยืนยันราย URL จริงจาก Google Search Console ได้ เนื่องจากไม่มีการส่งออกไฟล์ข้อมูลดิบรายตัว (Raw URL Export) มาให้ใน Batch นี้

### 1. ข้อมูลที่อนุมานจากระบบ (Inferred URL Mapping)
- **404 Errors (6 URLs):** คาดการณ์เป็นหน้าไดเรกทอรีเดิมและ Slug บริการเก่าที่ถูกถอดออก (`/บริการ/รับซื้อ-storage-nas`, `/รับซื้อ`, `/รับซื้อ/รับซื้อ-hdd`, `/รับซื้อ/รับซื้อ-gopro`, `/รับซื้อ/รับซื้อเลนส์`, `/รับซื้อ/รับซื้อ-storage-nas`)
- **Redirect Errors (2 URLs):** คาดการณ์เป็น `/บริการ/รับซื้อ-gopro` และ `/บริการ/รับซื้อ-hdd` ก่อนการปรับปรุงสับเปลี่ยนลำดับ
- **Page with Redirect (11 URLs):** ได้รับการยืนยันจากตัวกรองการเปลี่ยนเส้นทางใน `vercel.json`
- **Discovered (40 URLs) / Crawled (7 URLs) not indexed:** คาดการณ์เป็นหน้าให้บริการท้องถิ่นที่มีทราฟฟิกต่ำหรือเนื้อหาทับซ้อน

### 2. คำยืนยัน
"ยังไม่สามารถยืนยันราย URL จาก GSC ได้" เนื่องจากข้อจำกัดด้านข้อมูลดิบ อย่างไรก็ตาม กฎการทำงานและทางผ่านทั้งหมดได้รับการป้องกันเพื่อรองรับการตรวจจับของบอทในอนาคตเรียบร้อยแล้ว
