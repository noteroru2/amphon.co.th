## อาการเสียแบบไหนที่เราพิจารณารับซื้อ?

![รับซื้อโน๊ตบุ๊คเสีย เปิดไม่ติด ซากโน๊ตบุ๊ค - อำพล เทรดดิ้ง](/images/services/rub-sue-broken-notebook-amphon.png)

เรารับซื้อโน๊ตบุ๊คที่มีปัญหาทุกรูปแบบ ทุกแบรนด์ (Acer, Asus, Lenovo, HP, Dell, MSI):

- **เมนบอร์ดพัง (Motherboard Dead):** เปิดไม่ติด ไฟเข้าแต่ภาพไม่ขึ้น หรือเปิดแล้วดับ
- **จอแตก (Broken Screen):** จอแตก จอลาย จอมืด (ซ่อมจอราคาแพง ขายซากมักจะคุ้มกว่า)
- **เครื่องตกน้ำ (Water Damaged):** น้ำหกใส่ กาแฟหกใส่ คีย์บอร์ดรวน
- **แบตบวมจนดันเครื่องแตก:** แบตเตอรี่เสื่อมสภาพจนดันบอดี้หรือทัชแพดโก่งงอ
- **บานพับหัก (Hinge Broken):** เปิดปิดจอไม่ได้ กรอบพลาสติกแตกหัก

## อะไหล่ที่ยังมีมูลค่าแม้เครื่องเสีย

ถึงแม้เครื่องจะใช้งานไม่ได้แล้ว แต่ชิ้นส่วนเหล่านี้ยังมีราคาในตลาดอะไหล่:

| ชิ้นส่วน | ทำไมยังมีมูลค่า |
| :--- | :--- |
| **หน้าจอ (Display)** | จอโน๊ตบุ๊คราคาแพง 2,000–8,000 บาท ถ้าสภาพดีนำไปเป็นอะไหล่ซ่อมได้ |
| **RAM** | DDR4/DDR5 ยังใช้ได้ในเครื่องอื่น โดยเฉพาะถ้าเป็น 16GB+ |
| **SSD (M.2 / SATA)** | นำไปใช้ต่อได้เลยถ้าไม่ได้เสียหายจากน้ำหรือไฟ |
| **บอดี้ตัวเครื่อง** | รุ่นที่มีผู้ใช้งานมาก อะไหล่บอดี้ยังมีความต้องการ |
| **พัดลม / ระบบระบายความร้อน** | โดยเฉพาะ Gaming Notebook ที่ชิ้นส่วนระบายความร้อนมีราคา |

## ปัจจัยการประเมินราคาโน๊ตบุ๊คเสีย

1. **อายุและ Generation ของเครื่อง:** เครื่องรุ่นใหม่ (Intel Gen 10+ / Ryzen 4000+) ราคาซากจะสูงกว่า
2. **หน้าจอ:** ถ้าจอยังสมบูรณ์ดีทั้งที่บอร์ดเสีย จะได้ราคาสูงขึ้นมาก
3. **RAM และ SSD ภายใน:** ยิ่งความจุสูง ยิ่งบวกราคาเพิ่มได้มาก
4. **สภาพความเสียหาย:** เครื่องที่โดนน้ำหนักมักได้ราคาต่ำกว่าเครื่องที่บอร์ดพังอย่างเดียว
