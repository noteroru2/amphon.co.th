## อุปกรณ์ที่มักรับพิจารณา

### คอมและโน๊ตบุ๊ค

Desktop และ Notebook จากทุกแบรนด์หลัก ทั้ง Dell, HP, Lenovo, Acer, Asus และแบรนด์อื่น รับพิจารณาทั้งเครื่องที่ใช้งานได้ปกติและเครื่องที่มีอาการ สเปกตั้งแต่ Core i3 ขึ้นไปรับพิจารณาในหลายรุ่น เครื่องสเปกเก่าหรือ HDD ยังมีมูลค่าในตลาด refurbish ไม่จำเป็นต้องทิ้งก่อนถามราคา ถ้าล็อตหลักเป็นเดสก์ท็อปหลายเครื่อง ดูเพิ่มที่ [รับซื้อคอมพิวเตอร์ตั้งโต๊ะ](/บริการ/รับซื้อคอมพิวเตอร์ตั้งโต๊ะ) และ [รับซื้อ PC มือสอง](/บริการ/รับซื้อ-pc-มือสอง)

### จอและอุปกรณ์ต่อพ่วง

จอมอนิเตอร์ทุกขนาดและทุกแบรนด์ รับพิจารณาตามสภาพ รอยขีดข่วนเล็กน้อยหรือขอบมีรอยยังรับได้ พิกเซลเสียหรือจอมีปัญหามีผลต่อราคา รวมถึงอุปกรณ์ต่อพ่วงเช่น KVM Switch, Docking Station และ Port Hub ที่ยังใช้งานได้

### UPS และเครื่องสำรองไฟ

UPS ทุกขนาดตั้งแต่ขนาดเล็กสำหรับคอมตั้งโต๊ะไปจนถึง UPS ขนาดใหญ่สำหรับห้อง Server รับพิจารณาตามสเปกและสภาพแบตเตอรี่ เครื่องที่แบตเตอรี่เสื่อมแล้วยังมีมูลค่าชิ้นส่วนหลัก แจ้งยี่ห้อ รุ่น และ VA มาพร้อมรายการ

### Printer และ Scanner

Printer ทั้งแบบ Inkjet, Laser, และ Multifunction รับพิจารณาตามสภาพ Laser Printer ทำงานได้ปกติมักมีมูลค่าในตลาดมากกว่า Inkjet รุ่นเก่า Scanner แยกและ Scanner ในรูป Multifunction รับพิจารณาตามรุ่นและสภาพ

### Server, NAS และ Storage

Server ระดับ Enterprise จากแบรนด์ Dell, HP, IBM, Cisco มักมีมูลค่าในตลาดมือสอง รับพิจารณาตาม Generation และสเปก NAS สำหรับ SME จากแบรนด์ Synology, QNAP, Buffalo รับพิจารณาตามรุ่นและสภาพ รวมถึง Storage Array และ Tape Library บางรุ่น ถ้ามีเครื่องงานกราฟิกหรือเครื่องใช้ GPU หนักปะปนอยู่ด้วย ดูต่อที่ [รับซื้อ Workstation](/บริการ/รับซื้อ-workstation)

### Router, Switch และ Access Point

อุปกรณ์ Network จากแบรนด์ Cisco, Ubiquiti, MikroTik, HP, Netgear รับพิจารณาตามรุ่นและสภาพ อุปกรณ์ระดับ Enterprise หรือ Business Class มักมีมูลค่าสูงกว่า Consumer Grade ที่ออกแบบมาสำหรับบ้าน แจ้งรุ่นมาพร้อมรายการเพื่อประเมินได้ตรงขึ้น

### สายไฟ Rack และอุปกรณ์ห้อง IT

Rack Cabinet, PDU (Power Distribution Unit), Patch Panel, Structured Cabling และสายไฟจำนวนมากที่สะสมในห้อง IT รับพิจารณาตามสภาพและจำนวน Rack Cabinet สภาพดียังมีมูลค่าในตลาดมือสอง อุปกรณ์เบ็ดเตล็ดรับพิจารณารวมในล็อต

## กรณีไหนเหมาะกับบริการนี้

บริการเคลียร์อุปกรณ์ไอทีสำนักงานเหมาะที่สุดเมื่อมีของหลายประเภทปะปนกันและไม่ต้องการจัดการขายแยกเอง

**บริษัทหรือองค์กรที่เปลี่ยนระบบ IT ทั้งชุด**
เมื่ออัปเกรด Infrastructure ทั้งหมด อุปกรณ์เก่าทั้ง Server, Network Equipment และ UPS มักถูกเปลี่ยนพร้อมกัน การหาคนรับซื้อแยกทีละประเภทใช้เวลามาก บริการนี้รับพิจารณาทุกอย่างพร้อมกัน

**ห้อง IT ที่สะสมของเก่าหลายปี**
อุปกรณ์ที่ถูกแทนที่แต่ไม่เคยถูกจำหน่ายออก กองรวมกันจนเต็มห้อง รับพิจารณาทั้งหมดหรือบางส่วนตามที่ต้องการ ไม่บังคับขายทั้งหมด

**โรงเรียนหรือหน่วยงานที่เคลียร์ห้องคอมและห้อง IT**
อุปกรณ์จากห้องแล็บและห้อง IT ที่มีทั้งคอม จอ UPS และ Network Equipment ปะปน รับพิจารณาเป็นล็อตรวม

**เจ้าของกิจการที่ต้องการขายอุปกรณ์ไอทีทั้งหมดแต่ไม่มีเวลาจัดการเอง**
ถ่ายรูปส่งมาแล้วให้ทางร้านช่วยจัดการขั้นตอนต่อไป ไม่ต้องรู้รุ่นหรือราคาตลาดเอง

## ต้องส่งข้อมูลอะไรเพื่อประเมินราคา

อุปกรณ์หลายประเภทปะปนกันอาจดูน่าหนักใจ แต่ขั้นตอนเริ่มต้นทำได้ง่ายกว่าที่คิด

**รูปถ่ายภาพรวม**
ถ่ายภาพรวมของห้องหรือพื้นที่ที่มีอุปกรณ์อยู่ก่อน ไม่ต้องจัดเรียง ทางร้านดูได้จากรูปว่ามีอะไรอยู่บ้างโดยรวม

**รูปถ่ายแยกตามหมวดสินค้า**
ถ่ายเพิ่มแยกตามประเภทสินค้าที่มีจำนวนมาก เช่น กอง UPS แถว Server ใน Rack ชั้นวางคอม หรือกล่อง Network Equipment ที่ยังไม่ได้เปิด

**ยี่ห้อและรุ่นของอุปกรณ์หลัก**
สำหรับ Server, UPS ขนาดใหญ่ และ Network Equipment แจ้งยี่ห้อและรุ่นมาถ้าทราบ ถ้าไม่ทราบให้ถ่ายรูป label ที่ข้างเครื่อง ทางร้านช่วยระบุได้จากรูปในหลายกรณี

**จำนวนแต่ละประเภทโดยประมาณ**
ไม่ต้องนับแม่นยำ แค่ประมาณจำนวนพอได้ เช่น "คอมประมาณ 20 เครื่อง จอ 15 จอ UPS 5 ตัว Printer 3 เครื่อง"

**สภาพโดยรวมที่ทราบ**
แจ้งว่าอุปกรณ์ส่วนใหญ่ใช้งานได้หรือไม่ได้ หรือไม่แน่ใจเพราะเก็บไว้นาน ข้อมูลนี้ช่วยให้ประเมินเบื้องต้นได้ใกล้เคียงกว่าการไม่บอกอะไร

**พื้นที่ตั้งของสินค้า**
ระบุจังหวัดหรือพื้นที่คร่าวๆ เพื่อวางแผนขั้นตอนการรับของในภายหลัง

**UPS** — VA Rating ของเครื่อง, แบรนด์, อายุการใช้งาน และสภาพแบตเตอรี่ UPS ที่แบตเตอรี่ยังดีมีราคาสูงกว่า UPS ที่แบตเสื่อมแต่เครื่องยังดีก็ยังมีมูลค่า

**Printer** — เทคโนโลยีการพิมพ์ (Laser มักมีมูลค่าในตลาดมากกว่า Inkjet รุ่นเก่า), รุ่นและแบรนด์, สภาพหัวพิมพ์และตลับหมึก

**Server** — ยี่ห้อและ Platform (Dell PowerEdge, HP ProLiant มีตลาดชัดเจน), Generation ของ CPU และ RAM, จำนวน Drive และ RAID Configuration, สภาพโดยรวม

**Network Equipment** — ยี่ห้อ (Cisco มีตลาดมือสองในกลุ่ม Enterprise, Ubiquiti พบได้บ่อยใน SME), รุ่นและ Port Count, สภาพและ Firmware Version ที่รองรับ

## การคัดแยกอุปกรณ์ก่อนส่งรายการ

ไม่จำเป็นต้องคัดแยกให้สมบูรณ์แบบก่อนติดต่อ แต่การแยกอุปกรณ์ออกเป็นกลุ่มกว้างๆ ช่วยให้ประเมินได้เร็วและแม่นขึ้น

**แยกตามประเภทหลัก** คอม/โน๊ตบุ๊ค | จอ | UPS | Printer | Server และ Network — แยกออกเป็น 5 กลุ่มนี้ก็เพียงพอสำหรับการประเมินเบื้องต้น

**แยกตามสภาพ** ถ้าพอรู้สภาพ ให้แยกเป็นสองกลุ่มคือ "น่าจะใช้งานได้" กับ "ไม่แน่ใจหรือมีปัญหา" ไม่ต้องเปิดทดสอบทุกเครื่องก่อนส่งรายการ

**อุปกรณ์ที่ไม่รู้รุ่น** ถ่ายรูป label ข้างเครื่องมาได้เลย ทางร้านช่วยระบุรุ่นได้จากรูปในหลายกรณี ไม่จำเป็นต้องรู้รุ่นทุกชิ้นก่อนติดต่อ

**อุปกรณ์ที่ไม่แน่ใจว่าจะขาย** บอกมาในรายการว่า "ยังไม่แน่ใจ" ทางร้านประเมินให้ได้ว่ามีมูลค่าไหม แล้วค่อยตัดสินใจ

## สิ่งที่ต้องตรวจเช็กจริงก่อนสรุปราคา

ราคาเบื้องต้นที่ได้จากรายการหรือรูปถ่ายเป็นกรอบคร่าวๆ เท่านั้น อุปกรณ์ไอทีบางประเภทมีรายละเอียดที่ต้องดูจริงก่อนสรุปราคา

**คอมและโน๊ตบุ๊ค** — สภาพจอ แบตเตอรี่ และ Storage ต้องทดสอบจริง เครื่องที่รูปดูสมบูรณ์อาจมี bad sector หรือแบตเสื่อมหนักซึ่งมีผลต่อราคา

**UPS** — ต้องทดสอบ Runtime จริงหรืออ่านค่า Battery Health เครื่องที่แบตเตอรี่ยังพอมีอยู่กับเครื่องที่แบตตายสนิทราคาต่างกันชัดเจน

**Server** — จำนวน Drive ที่ยังอยู่, RAM ที่ติดตั้งจริง, สภาพ Power Supply และ RAID Card ต้องดูที่ตัวเครื่อง ไม่สามารถดูได้จากรูปภายนอกอย่างเดียว

**Network Equipment** — Cisco และอุปกรณ์ระดับ Enterprise บางรุ่น ราคาต่างกันมากตาม Firmware, License และ Module ที่ติดตั้ง ต้องดูที่ตัวเครื่องหรือ Console Output

**Printer** — ปริมาณหมึกที่เหลือ, จำนวน Page Count และสภาพหัวพิมพ์มีผลต่อราคา ข้อมูลเหล่านี้ได้จากการทดสอบพิมพ์หรือเปิดเมนู Maintenance

ทั้งหมดนี้ทางร้านตรวจเช็กตอนนัดดูของจริง ไม่หักราคาโดยไม่มีเหตุผล และถ้าราคาที่ได้หลังตรวจจริงต่างจากที่ประเมินเบื้องต้นมาก ลูกค้าตัดสินใจได้ว่าจะขายต่อหรือไม่
