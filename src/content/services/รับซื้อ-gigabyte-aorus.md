ถ้าเครื่องเปิดได้ ให้ถ่ายหน้า System Information หรือ Task Manager เพื่อยืนยัน CPU RAM SSD และ GPU ถ้าเปิดไม่ติด ให้ถ่ายอาการตอนเสียบชาร์จ เช่น ไฟชาร์จติดหรือไม่ พัดลมหมุนหรือไม่ และจอมีแสงหรือไม่
