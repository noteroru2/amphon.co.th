ถ้ามีอุปกรณ์เสริม เช่น Capture Card, จอ Monitor, SSD ภายนอก, HDD หลายลูก, Card Reader, UPS หรืออุปกรณ์ที่ใช้กับงานตัดต่อ สามารถส่งรายการมาประเมินร่วมได้ หากเครื่องเคยใช้ตัดต่อ 4K, งาน multicam, งานสี หรือ export งานบ่อย ควรแจ้งลักษณะงานไว้ด้วย

ให้ถ่ายรูปในที่สว่าง เห็นตัวเครื่องจริง ไม่ต้องแต่งภาพ และไม่ต้องปิดตำหนิ เพราะรอยใช้งานจริงมีผลต่อการประเมินตามสภาพจริง หากเป็นคอมตั้งโต๊ะหรือ Workstation แนะนำให้ถ่าย 4 มุมหลัก คือ ด้านหน้า ด้านหลัง ภายในเคส และภาพรวมอุปกรณ์ที่มากับเครื่อง

ถ้าเครื่องเปิดติด ให้ถ่ายหน้า System Information, Task Manager, Device Manager หรือหน้ารายละเอียดสเปกจากโปรแกรมที่ใช้อยู่ได้ หากมีใบเสร็จ กล่อง รายการอัปเกรด หรือรูปตอนใช้งานกับโปรแกรมงานจริง สามารถส่งเพิ่มได้ โดยเฉพาะเครื่องที่มีการ์ดจอเฉพาะทางหรืออุปกรณ์เสริมหลายชิ้น

## ราคาประเมินขึ้นกับอะไร

ราคาขึ้นกับรุ่น สเปก สภาพ อุปกรณ์ และจำนวนเครื่อง โดยคอมทำงานเฉพาะทางจะดูรายละเอียดมากกว่าคอมใช้งานทั่วไป เพราะบางเครื่องมีมูลค่าจากชิ้นส่วนหลัก เช่น CPU หลายคอร์ การ์ดจอ VRAM มาก RAM ความจุสูง SSD NVMe หลายลูก PSU กำลังวัตต์เหมาะสม หรือเคสที่รองรับงานหนักได้ดี

สำหรับคอมตัดต่อวิดีโอมือสอง ทีมจะพิจารณาทั้งความต้องการของตลาดมือสอง ความสมบูรณ์ของเครื่อง อายุการใช้งาน สภาพพอร์ต สภาพพัดลม ความร้อน อุปกรณ์ที่มาด้วย และความชัดเจนของสเปก หากลูกค้ามีรายการอัปเกรดหรือประวัติการใช้งานสามารถส่งประกอบได้

ลูกค้าสามารถส่งรูปเช็กราคาเบื้องต้นและเปรียบเทียบราคาก่อนได้ การประเมินสุดท้ายอาจปรับตามการตรวจเครื่องจริง โดยเฉพาะเครื่องที่มีอาการเสีย เปิดไม่ติด มีปัญหาการ์ดจอ หรือมีอุปกรณ์ไม่ครบ

## เครื่องมีตำหนิ เปิดไม่ติด หรือ GPU มีปัญหา ประเมินได้ไหม

เครื่องใช้งานได้หรือมีตำหนิส่งรูปประเมินเบื้องต้นได้ ไม่ว่าจะเป็นเปิดติดแต่ใช้งานช้า ไฟเข้าแต่ไม่ขึ้นภาพ เครื่องดับเอง พัดลมเสียงดัง การ์ดจอแสดงผลผิดปกติ พอร์ตบางช่องไม่ทำงาน SSD หาย RAM ไม่ครบ หรืออุปกรณ์ภายในถูกถอดไปบางส่วน

สำหรับเครื่องตัดต่อวิดีโอ ควรแจ้งอาการตามจริงตั้งแต่แรก เพราะเครื่องทำงานเฉพาะทางมักมีชิ้นส่วนที่มีผลต่อมูลค่าหลายจุด เช่น GPU, VRAM, PSU, RAM, Storage และระบบระบายความร้อน ถ้าส่งรูปจุดที่มีปัญหาได้ชัด จะช่วยให้ประเมินเบื้องต้นได้ใกล้เคียงกว่าเดิม

กรณีเครื่องเปิดไม่ติดหรือไม่สามารถเช็กสเปกจากระบบได้ ให้ถ่ายรูปสติ๊กเกอร์รุ่น เมนบอร์ด การ์ดจอ RAM SSD ป้าย PSU และภาพรวมภายในเคส ทีมจะรับข้อมูลไปพิจารณาตามสภาพจริงก่อนสรุปขั้นตอนต่อไป

หน้า [รับซื้อคอมตัดต่อวิดีโอ](/บริการ/รับซื้อคอมตัดต่อวิดีโอ/) ต่างจาก [รับซื้อคอมทำงานกราฟิก](/บริการ/รับซื้อคอมทำงานกราฟิก/) เพราะให้ความสำคัญกับไฟล์วิดีโอ ความเร็วในการอ่านเขียน storage ความจุพื้นที่เก็บงาน และการทำงานร่วมกับ GPU ใน workflow ตัดต่อ ส่วนหน้า [รับซื้อ Workstation](/บริการ/รับซื้อ-workstation/) เป็น hub รวมเครื่องทำงานหลายสาย

หน้านี้ยังต่างจากคอมสตรีมเกม เพราะคอมสตรีมเกมเน้นการเล่นเกมพร้อมออกอากาศและอุปกรณ์ live setup ส่วนคอมตัดต่อวิดีโอเน้น post-production, timeline, footage, render และ export งานวิดีโอเป็นหลัก
