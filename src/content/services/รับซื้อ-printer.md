## รับซื้อ Printer สำนักงานแบบไหนบ้าง

### Laser Printer ขาวดำและสี

เครื่องพิมพ์ Laser ที่ใช้ในสำนักงานมีตลาดมือสองที่ดีกว่า Inkjet ทั่วไป โดยเฉพาะรุ่นที่ยังพิมพ์ได้ปกติและ Drum/Toner ยังดีอยู่ รับพิจารณาทุกแบรนด์ทั้ง HP, Canon, Xerox, Brother และ Samsung

### Inkjet Printer สำหรับสำนักงาน

Inkjet รุ่นที่ออกแบบมาสำหรับงานสำนักงานมักมีมูลค่าในตลาดมือสองมากกว่ารุ่น Home ทั่วไป รับพิจารณาตามรุ่นและสภาพหัวพิมพ์ Inkjet ที่มีหมึกถัง (EcoTank, MegaTank) มักมีราคาดีกว่า Inkjet ระบบตลับ

### Multifunction (All-in-One)

เครื่องพิมพ์ที่มีทั้ง Print, Scan, Copy และบางรุ่น Fax รับพิจารณาตามว่าฟังก์ชันหลักยังทำงานได้หรือไม่ แจ้งมาด้วยว่าส่วนไหนใช้งานได้และส่วนไหนมีปัญหา

### Network Printer และ Printer สำหรับแผนก

Printer ที่ต่อ LAN หรือ Wi-Fi ได้และรองรับการพิมพ์จากหลายเครื่องพร้อมกัน มักมีมูลค่าดีกว่า USB-only Printer แจ้ง Network Feature มาด้วยถ้าทราบ

### Printer ยกล็อตจากสำนักงานหรือโรงเรียน

Printer หลายเครื่องที่มาจากสำนักงานเดียวกัน ขายพร้อมกันทั้งล็อต รับพิจารณาแม้จะผสมรุ่น ประเมินแยกรุ่นแล้วรวมเป็นราคาล็อตให้

## ข้อมูลที่ต้องส่งเพื่อประเมินราคา Printer

**รูปหน้าเครื่อง** — ถ่ายให้เห็น Control Panel, Display และ Paper Tray ด้านหน้า

**รูปด้านหลัง** — ถ่ายให้เห็น Port (USB/LAN), Power Input และช่องใส่ Cartridge/Toner ถ้าอยู่ด้านหลัง

**รูป Label รุ่นและ Serial** — มักอยู่ที่ด้านหลังหรือด้านล่างเครื่อง แสดงรุ่นและ Serial Number

**ยี่ห้อและรุ่น** — เช่น HP LaserJet Pro MFP M428fdn, Canon imageCLASS MF445dw, Brother HL-L2370DN

**ประเภทเครื่อง** — Laser / Inkjet / Multifunction / Network Printer

**สภาพการพิมพ์** — พิมพ์ได้ปกติ, พิมพ์ได้แต่คุณภาพไม่ดี, พิมพ์ไม่ออก, ไม่ได้ทดสอบ

**มีหมึก ตลับหรือถัง** — แจ้งว่ามีหมึกมาด้วยหรือไม่ ถ้ามีระบุว่าเต็มหรือเหลือน้อย

**ถาดกระดาษและ ADF** — ถาดกระดาษครบหรือไม่ ถ้ามี ADF (ป้อนกระดาษอัตโนมัติ) ยังทำงานได้ไหม

**สายไฟและ Cable** — มีสายไฟและสาย USB หรือ LAN มาด้วยหรือไม่

**จำนวนเครื่อง** — ถ้ามีหลายเครื่อง แจ้งจำนวนและว่ารุ่นเดียวกันหรือต่างรุ่น

**พื้นที่ตั้งสินค้า** — จังหวัดหรือพื้นที่คร่าวๆ

## วิธีถ่ายรูป Printer ให้ประเมินได้ไว

**รูปที่ต้องมีก่อน:**

ถ่ายด้านหน้าให้เห็น Panel และชิ้นส่วนหลักครบ ถ่าย Label รุ่นที่ด้านหลังหรือด้านล่างให้ชัด ถ่าย Display ถ้าเครื่องเปิดอยู่เพื่อให้เห็น Error Code ถ้ามี

**ถ้า Printer เปิดอยู่:**

พิมพ์ Configuration Page หรือ Test Page แล้วถ่ายรูปผลการพิมพ์มาด้วย ช่วยให้เห็นคุณภาพการพิมพ์และ Page Count ของเครื่องได้ทันที

**สำหรับ Multifunction:**

ถ่ายให้เห็น ADF (ถาดป้อนกระดาษด้านบน) และพื้นกระจก Scanner ว่ายังสมบูรณ์หรือไม่

## ราคา Printer มือสองขึ้นกับอะไร

### เทคโนโลยีการพิมพ์มีผลมากที่สุด

Laser มีราคาดีกว่า Inkjet ในขนาดงานเดียวกัน Inkjet ที่มีระบบถังหมึก (EcoTank, MegaTank, CISS) มีราคาดีกว่า Inkjet ระบบตลับ เพราะผู้ซื้อคำนวณต้นทุนหมึกต่อแผ่นได้ถูกกว่า

### Page Count และอายุการใช้งาน

Laser Printer ที่พิมพ์ไปน้อย (Page Count ต่ำ) มีราคาดีกว่าเครื่องที่พิมพ์ไปหลายหมื่นแผ่ว เพราะ Drum และ Fuser Unit มีอายุการใช้งานจำกัด Inkjet ราคาขึ้นกับสภาพหัวพิมพ์มากกว่า Page Count

### Network Capability

Printer ที่มี LAN หรือ Wi-Fi Built-in มีราคาดีกว่า USB-only ในรุ่นเดียวกัน

### สภาพและ Feature

Multifunction ที่ทุกฟังก์ชันใช้งานได้ปกติมีราคาดีกว่า Multifunction ที่บางส่วนเสีย Printer ที่มี Duplex (พิมพ์สองหน้าอัตโนมัติ) มีราคาดีกว่า Single-side Only ในรุ่นเดียวกัน

### หมึกที่มาด้วย

Printer ที่มีหมึกหรือ Toner มาพร้อมกันมีราคาดีกว่าเครื่องเปล่า โดยเฉพาะถ้าเป็น Toner ราคาสูงที่ยังเหลืออยู่มาก

## Printer เปิดไม่ติด หมึกตัน หัวพิมพ์มีปัญหา ยังประเมินได้ไหม

รับพิจารณาครับ แต่วิธีประเมินต่างกันตามอาการ

**Printer ที่หมึกตันหรือหัวพิมพ์สกปรก** — อาจยังมีมูลค่าถ้าตัวเครื่องยังดี ผู้ซื้อบางรายซื้อมาทำความสะอาดหัวพิมพ์เองหรือส่งซ่อม

**Printer ที่ Toner หมดแต่เครื่องยังดี** — รับพิจารณา ราคาของ Toner ไม่ได้กำหนดราคาตัวเครื่อง

**Printer ที่เปิดไม่ติด** — รับพิจารณาตามรุ่นและอาการ แจ้งรุ่นมาให้ดูก่อน

**Printer ที่ Paper Jam บ่อยหรือ Feed ไม่ขึ้น** — มีผลต่อราคา แต่ยังรับพิจารณา แจ้งอาการตรงๆ มาเลย

---


## Printer ที่ขายพร้อมกับคอมสำนักงานยกล็อต — ทำอย่างไรได้บ้าง

บริษัทที่เคลียร์สำนักงานพร้อมกันมักมี Printer ปะปนมากับคอมในล็อตเดียวกัน การแยกขายทีละชิ้นใช้เวลามาก การส่งรายการรวมมาในครั้งเดียวสะดวกกว่า

**ถ้า Printer มาพร้อมกับคอมในล็อตเดียว** แจ้งรายการรวมมาเลย ทางร้านประเมินแยกแล้วรวมเป็นราคาล็อตให้ ไม่ต้องแยกรายการเอง

**ถ้า Printer เป็น Network Printer ที่ใช้รวมทั้งสำนักงาน** แจ้งรุ่นและจำนวนมาด้วย Network Printer มักมีราคาดีกว่า USB-only ในรุ่นเดียวกัน

**ถ้ามี Toner หรือหมึกเหลือในเครื่อง** แจ้งมาด้วยว่าเหลือมากหรือน้อย มีผลต่อราคารวม โดยเฉพาะ Toner ที่มีราคาสูง

**ถ้า Printer บางเครื่องมีอาการ** แยกออกมาและแจ้งอาการแต่ละเครื่องในรายการ ทางร้านประเมินได้ตรงกว่าถ้าทราบว่าเครื่องไหนดีและเครื่องไหนมีปัญหา

---

## Printer ที่ขายพร้อมคอมสำนักงานหรือเคลียร์ออฟฟิศ

บริษัทหรือสำนักงานที่เคลียร์ของออกพร้อมกันมักมีทั้งคอมและ Printer ในล็อตเดียวกัน รับพิจารณาประเมินรวมได้ในครั้งเดียว ไม่ต้องแยกขายทีละชิ้น

แจ้งรายการรวมมาในครั้งเดียว ทั้งคอม โน๊ตบุ๊ค จอ UPS และ Printer ทางร้านประเมินแยกประเภทแล้วรวมเป็นราคาล็อตให้

## สิ่งที่ช่วยให้ประเมิน Printer ได้แม่นขึ้น

Page Count คือข้อมูลสำคัญที่สุดสำหรับ Laser Printer เข้าไปที่ Configuration Page หรือ Printer Status Page ซึ่งพิมพ์ออกมาได้จากเมนูของเครื่อง จะเห็น Total Page Count ชัดเจน

สำหรับ Inkjet ให้ทดสอบพิมพ์ Test Page และถ่ายรูปผลลัพธ์มา จะเห็นทันทีว่าหัวพิมพ์ยังสม่ำเสมอหรือมีหัวที่ตันบางส่วน

ถ้าไม่แน่ใจวิธีหาข้อมูลเหล่านี้ ส่งรุ่นมาก่อนแล้วสอบถามทางร้านได้ก่อนนัด ทางร้านช่วยแนะนำขั้นตอนได้

## สิ่งที่ลูกค้ามักสงสัยก่อนส่ง Printer มาประเมิน

**Inkjet สำนักงาน vs Home ราคาต่างกันไหม** — Inkjet ที่ออกแบบมาสำหรับงานสำนักงาน เช่น HP OfficeJet Pro หรือ Canon PIXMA Pro มีราคาดีกว่า Inkjet รุ่น Home-use ที่มีขนาดและ Speed ต่ำกว่า แม้จะเป็นแบรนด์เดียวกัน

**Toner เหลือมากมีผลไหม** — มีผลครับ Toner ที่ยังเหลืออยู่มากมีมูลค่าในตัวเอง เพราะ Toner Original สำหรับ Laser Printer ระดับ Business มักราคาสูง รวมกับตัวเครื่องแล้วราคาดีกว่าขายเครื่องเปล่า

**ไม่รู้ Page Count** — หา Configuration Page จากเมนูเครื่องพิมพ์ มักอยู่ใน Reports หรือ Information Pages ส่วน Inkjet ให้ทดสอบพิมพ์ Nozzle Check Pattern แล้วถ่ายรูปผลลัพธ์มา

**Printer ที่ไม่ได้ใช้มานานกว่า 1 ปี** — Inkjet อาจมีหัวพิมพ์ตัน ทดสอบพิมพ์ก่อนส่งรูปมา แต่ถ้าเปิดไม่ติดหรือทดสอบไม่ได้ ส่งรุ่นมาก่อนได้เลย ทางร้านประเมินตามสภาพที่แจ้ง

## สถานการณ์ที่มักพบในงานรับซื้อ Printer มือสอง

### สำนักงานย้ายไปใช้ Cloud Printing หรือ Outsource

บริษัทที่เปลี่ยนไปใช้บริการ Outsource Printing หรือ MPS (Managed Print Services) มักมี Printer เดิมค้างอยู่ในสำนักงาน รับพิจารณาทั้งล็อต

### โรงเรียนได้รับงบประมาณซื้อ Printer ใหม่

โรงเรียนที่ได้รับงบจัดซื้อ Printer ใหม่มักมีเครื่องเก่าที่ยังใช้งานได้รอการจำหน่าย รับพิจารณาและออกเอกสารให้ได้ถ้าแจ้งล่วงหน้า

### Printer ที่มาพร้อมกับคอมสำนักงานยกล็อต

การเคลียร์สำนักงานมักมี Printer ปะปนมากับคอม รับพิจารณาประเมินรวมได้ในครั้งเดียว
