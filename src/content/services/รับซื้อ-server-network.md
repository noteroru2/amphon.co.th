## รับซื้ออุปกรณ์โครงสร้างพื้นฐานไอที (IT Infrastructure)

อุปกรณ์ศูนย์ข้อมูล (Data Center) หรือห้องเซิร์ฟเวอร์ขนาดเล็กในออฟฟิศ เรารับเหมาทั้งหมด:
*   **รับซื้อ Server:** แบรนด์ดังอย่าง Dell PowerEdge, HP ProLiant, Lenovo ThinkSystem ทั้งแบบ Rackmount และ Tower
*   **รับซื้อ NAS / Storage:** อุปกรณ์เก็บข้อมูลเครือข่าย Synology, QNAP รวมถึงฮาร์ดดิสก์เกรด Enterprise (SAS/SATA)
*   **รับซื้อ Network Switch:** สวิตช์ฮับ (Hub Switch), PoE Switch, Managed/Unmanaged Switch
*   **รับซื้อ Router & Firewall:** อุปกรณ์จัดการเครือข่ายความปลอดภัยแบรนด์ต่างๆ
*   **รับซื้อ Access Point:** อุปกรณ์กระจายสัญญาณ Wi-Fi สำหรับองค์กร (เช่น Ubiquiti UniFi, Aruba)

## ข้อควรระวังด้านความปลอดภัยสำหรับ Server

สิ่งสำคัญที่สุดในการขายเซิร์ฟเวอร์และสตอเรจคือ "ข้อมูล" หากบริษัทของคุณยังไม่อนุญาตให้นำหน่วยความจำออกจากพื้นที่ สามารถ **ถอด HDD/SSD เก็บไว้และขายเฉพาะตัวเครื่องให้เราได้** หรือหากต้องการขายรวม HDD รบกวนให้ฝ่าย IT ทำการ Wipe Data (เขียนทับด้วยเลขศูนย์ระดับ Low-level) ก่อนส่งมอบครับ
