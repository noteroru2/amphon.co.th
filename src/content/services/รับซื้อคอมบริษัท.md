## รับซื้อคอมบริษัทแบบไหนบ้าง

หน้านี้ออกแบบสำหรับองค์กรที่ต้องจัดการสินทรัพย์ไอทีโดยเฉพาะ — ไม่ใช่แค่ขายคอมเครื่องเดียว แต่เป็นการเคลียร์คอมพนักงาน โน๊ตบุ๊คที่ถูกเรียกคืน หรืออุปกรณ์ไอทีหลายรายการพร้อมกัน

### คอมพนักงานที่ครบอายุการใช้งาน

คอม Desktop ที่บริษัทใช้มา 3–5 ปี ถึงเวลาเปลี่ยนรุ่นใหม่ แต่เครื่องเก่ายังสภาพดีหรือใช้งานได้บางส่วน รับพิจารณาจาก Dell, HP, Lenovo, Acer และแบรนด์อื่น ไม่ว่าจะเป็น OptiPlex, EliteDesk หรือ ThinkCentre หากเป็นชุดเดสก์ท็อปล้วนหรือเครื่องออฟฟิศหลายชุด สามารถดูแนวทางเพิ่มที่ [รับซื้อคอมพิวเตอร์ตั้งโต๊ะ](/บริการ/รับซื้อคอมพิวเตอร์ตั้งโต๊ะ) และ [รับซื้อ Desktop PC](/บริการ/รับซื้อ-desktop-pc)

### โน๊ตบุ๊คองค์กรที่พนักงานส่งคืน

โน๊ตบุ๊คสายธุรกิจอย่าง Dell Latitude, HP EliteBook, Lenovo ThinkPad หรือ HP ProBook ที่พนักงานใช้งานแล้วส่งคืน หรือที่บริษัทรวบรวมไว้รอจำหน่าย รับพิจารณาตามสภาพจริงทุกเครื่อง ไม่ตีราคารวมแบบเหมาล็อตโดยไม่อธิบาย

### คอมที่เสียหรือซ่อมไม่คุ้ม

เครื่องที่บูตไม่ติด จอเสีย หรือซ่อมแล้วต้นทุนเกินมูลค่า บางเครื่องยังมีมูลค่าชิ้นส่วนหรือใช้ refurbish ได้ ส่งอาการมาให้ดูก่อน ประเมินตามสภาพจริง ไม่คาดเดา

### ชุดคอมบริษัทที่ซื้อพร้อมกันทั้งล็อต

ถ้าบริษัทซื้อคอมมาพร้อมกันทั้งชุด รุ่นเดียวกัน สเปกเดียวกัน การขายยกล็อตประเมินได้ง่ายกว่า และจบในครั้งเดียว ไม่ต้องจัดการแยกเครื่อง ถ้าในล็อตมีทั้งเครื่องใช้งานทั่วไปและเครื่องแรงสำหรับงานกราฟิกหรือวิเคราะห์ข้อมูล ดูแนวทางเพิ่มที่ [รับซื้อ PC มือสอง](/บริการ/รับซื้อ-pc-มือสอง), [รับซื้อ Workstation](/บริการ/รับซื้อ-workstation) และ [รับซื้อคอมสเปกสูง](/บริการ/รับซื้อคอมสเปกสูง)

### อุปกรณ์ไอทีที่มาพร้อมกับชุดคอม

จอมอนิเตอร์ คีย์บอร์ด เมาส์ UPS และอุปกรณ์เสริมอื่นที่มาพร้อมกับชุดคอมบริษัท รับพิจารณาได้ด้วย แจ้งรายการมาพร้อมกันเพื่อประเมินรวมในคราวเดียว ถ้ามีอุปกรณ์เครือข่ายหรือของห้อง IT ปนมาด้วย ดูรายละเอียดที่ [รับซื้ออุปกรณ์ไอทีบริษัท](/บริการ/รับซื้ออุปกรณ์ไอทีบริษัท)

## เหมาะกับบริษัทที่กำลังเปลี่ยนเครื่องหรือเคลียร์ทรัพย์สิน

งานรับซื้อคอมบริษัทมักเกิดจากสถานการณ์เฉพาะขององค์กร ไม่ใช่แค่ "อยากขายคอมเก่า" แต่ต้องปิดงานให้ครบทั้งด้านเทคนิคและเอกสาร

### บริษัทเปลี่ยนชุดคอมพนักงานพร้อมกัน

องค์กรที่อัปเกรด PC พนักงานทั้งสำนักงานในคราวเดียว มักมีคอมเก่า 20–100 เครื่องรออยู่ การขายยกล็อตช่วยให้จัดการได้รวดเร็วและไม่ต้องแบกรับภาระสินทรัพย์ที่ไม่ได้ใช้แล้ว

### โรงเรียน โรงพยาบาล และหน่วยงานราชการ

ห้องปฏิบัติการคอมพิวเตอร์หรือครุภัณฑ์ที่ใช้งานมาหลายปี มักมีเครื่องหลายรุ่นปะปนกัน ทางร้านรับพิจารณาแม้รุ่นจะปะปน แจ้งรายการมาก่อนเพื่อประเมินเบื้องต้น และออกใบรับซื้อได้ถ้าแจ้งล่วงหน้า

### SME ที่ปิดสาขาหรือลดจำนวนพนักงาน

คอมพนักงานที่เหลือจากการลดขนาดองค์กร รับพิจารณาเป็นล็อตหรือรายเครื่องได้ ไม่มีขั้นต่ำตายตัว ขึ้นกับสภาพและประเภทสินค้า

### บริษัทที่ปิดกิจการหรือย้ายสำนักงาน

เมื่อต้องเคลียร์ทรัพย์สินออกจากสถานที่ภายในเวลาจำกัด การส่ง Inventory List มาก่อนช่วยให้วางแผนนัดตรวจเช็กและรับมอบได้เร็วขึ้น

## ส่ง Inventory List หรือรายการเครื่องเพื่อประเมินราคา

จุดแตกต่างสำคัญของงานบริษัทคือ ไม่จำเป็นต้องถ่ายรูปทุกเครื่อง ถ้ามีรายการเครื่องที่ครบถ้วนอยู่แล้ว

### กรณีมี Inventory List หรือ Asset List

ส่งไฟล์ Excel, CSV หรือรายการ text มาทาง LINE @webuy ได้เลย ควรมีข้อมูลต่อเครื่องดังนี้:

- แบรนด์และรุ่น เช่น Dell OptiPlex 7060 หรือ Lenovo ThinkPad E580
- CPU และ Generation เช่น Core i5-8400, Core i7-8550U
- RAM (GB) และประเภท Storage (HDD / SSD)
- สภาพโดยรวม เช่น ใช้งานได้ปกติ / มีอาการ / เปิดไม่ติด
- จำนวนเครื่องในแต่ละรุ่น
- Serial Number (ถ้ามีในระบบ asset management)

ถ้าฝ่าย IT export รายการจากระบบจัดการทรัพย์สินได้ จะช่วยให้ประเมินเบื้องต้นได้เร็วมาก โดยไม่ต้องเดินตรวจทุกเครื่องก่อนติดต่อ

### กรณีไม่มี List

ถ่ายรูปตัวแทนสัก 2–3 เครื่อง เน้นที่ label สเปกด้านหลังหรือด้านล่างเครื่อง แจ้งจำนวนรวมและรุ่นคร่าวๆ ก็เพียงพอสำหรับการประเมินเบื้องต้นได้

### ข้อมูลเพิ่มเติมที่ช่วยให้ประเมินแม่นขึ้น

- พื้นที่ของลูกค้า (จังหวัด/เขต) เพื่อสรุปขั้นตอนนัดรับหรือจัดส่ง
- วันที่ต้องการเคลียร์ออกจากสถานที่
- รูปแบบเอกสารที่ฝ่ายบัญชีต้องการ (ถ้ามี)

## ราคาประเมินขึ้นกับอะไร

ราคาต่อเครื่องของคอมบริษัทขึ้นกับปัจจัยเดียวกับคอมทั่วไป แต่มีบางจุดที่งานบริษัทต่างออกไป

**ปัจจัยที่มีผลโดยตรง:**

- **รุ่นและ Generation ของ CPU** — Core i5/i7 Gen 6 ขึ้นไปยังเป็นที่ต้องการในตลาด refurbish

- **สภาพตัวเครื่อง** — รอยใช้งานปกติไม่มีผลมาก แต่บวม แตก หรือเสียหายหนักมีผล
- **ความสม่ำเสมอของล็อต** — ล็อตที่รุ่นเดียวกันทั้งหมดประเมินและขายต่อได้ง่ายกว่า
- **สภาพ HDD/SSD** — เครื่องที่ถูก Format แล้วทำงานได้ปกติมีมูลค่าสูงกว่าเครื่องที่มี bad sector
- **จำนวนเครื่อง** — ยกล็อตมีความยืดหยุ่นในการเจรจาต่างจากรายเครื่อง

**สิ่งที่ไม่ทำให้มูลค่าหายไปทั้งหมด:**

- ไม่มีกล่องหรืออุปกรณ์ครบ — ลดมูลค่าเล็กน้อยแต่ยังรับพิจารณา
- ไม่มี Windows License — ราคาขึ้นกับสเปกและสภาพฮาร์ดแวร์เป็นหลัก
- เครื่องมีรอยตามการใช้งาน — ประเมินตามสภาพจริง ไม่ตัดราคาเกินจริง

### คอมบริษัทแยกกลุ่มสภาพอย่างไร

**กลุ่มที่ 1 — เครื่องใช้งานได้ปกติ สเปกยังดี**
คอมหรือโน๊ตบุ๊คที่ยังเปิดติด ทำงานได้ปกติ มี SSD และ RAM ตั้งแต่ 8 GB ขึ้นไป CPU Gen 8 ขึ้นไป กลุ่มนี้มีมูลค่าประเมินสูงสุดในล็อต

**กลุ่มที่ 2 — เครื่องใช้งานได้ แต่สเปกเก่าหรือ HDD**
คอม Core i3–i5 Gen 4–7 หรือเครื่องที่ยัง HDD อยู่ ยังขายได้แต่มูลค่าประเมินต่ำกว่ากลุ่มแรก เหมาะสำหรับตลาด refurbish

**กลุ่มที่ 3 — เครื่องมีอาการหรือซ่อมบางส่วน**
เครื่องที่จอมีปัญหา แบตเสื่อมมาก คีย์บอร์ดพัง หรือ HDD มี bad sector บางส่วน มูลค่าประเมินจะต่ำกว่าปกติ แต่ยังมีมูลค่าชิ้นส่วน

**กลุ่มที่ 4 — เครื่องเปิดไม่ติดหรือเสียหายหนัก**
บางเครื่องในล็อตอาจบูตไม่ได้เลย รับพิจารณาตามสภาพ ถ้าเป็นส่วนน้อยของล็อต ไม่ทำให้ล็อตรวมเสียมูลค่ามากนัก

## ข้อมูลในเครื่องและเอกสารบริษัทต้องเตรียมอะไร

การขายทรัพย์สินบริษัทไม่ได้จบที่ราคาอย่างเดียว มีรายละเอียดด้านข้อมูลและเอกสารที่ฝ่าย IT และฝ่ายบัญชีต้องรู้ก่อน

### เรื่องข้อมูลและ License ในเครื่อง

ก่อนส่งมอบคอมบริษัท ฝ่าย IT ควรดำเนินการเหล่านี้:

- **Deactivate Windows License** โดยเฉพาะถ้าเป็น Volume License ที่ผูกกับ MAK หรือ KMS ขององค์กร
- **ออกจาก Domain** ถ้าเครื่องอยู่ใน Active Directory
- **ลบข้อมูลพนักงานและข้อมูลบริษัท** ด้วย Format หรือ Reset เครื่อง
- **ถอน MDM / Device Management** ถ้ามีระบบจัดการอุปกรณ์จากส่วนกลาง

ไม่จำเป็นต้องติดตั้ง Windows ใหม่ก่อนส่ง แต่ควรเคลียร์ข้อมูลให้เรียบร้อยก่อนส่งมอบจริง ทางร้านรับเครื่องที่ลูกค้าเตรียมพร้อมแล้วเท่านั้น ความรับผิดชอบในการลบข้อมูลก่อนส่งมอบเป็นของลูกค้า

ถ้าทีม IT ไม่แน่ใจขั้นตอน สามารถสอบถามทางร้านได้ก่อนนัด

### เรื่องเอกสารสำหรับฝ่ายบัญชี

ทางร้านออกใบรับซื้อให้ได้ถ้าแจ้งล่วงหน้าตอนติดต่อครั้งแรก ควรแจ้งด้วยว่าต้องการเอกสารในรูปแบบอะไร เช่น:

- ใบรับซื้อสำหรับตัดจำหน่ายทรัพย์สิน
- ใบเสร็จรับเงิน
- เอกสารยืนยันการรับมอบสินค้า

จ่ายเงินทันทีเมื่อตกลงราคาและตรวจเช็กเครื่องแล้ว ไม่ต้องรอโอน

### เตรียมเครื่องก่อนส่งมอบ — Checklist สำหรับฝ่าย IT

1. **รวบรวมและตรวจนับเครื่อง** — export asset list ถ้ามี
2. **ออกจาก Domain และถอน MDM**
3. **Deactivate License และลบข้อมูล** — Format หรือ Reset
4. **เช็กสภาพเบื้องต้น** — แยกเครื่องที่มีอาการออกจากเครื่องปกติ
5. **รวบรวมอุปกรณ์ที่มา** — สายไฟ อแดปเตอร์ กล่อง
