## รุ่นซีพียูที่เราเปิดรับซื้อ

![รับซื้อซีพียู CPU Intel AMD มือสอง - อำพล เทรดดิ้ง](/images/services/rub-sue-cpu-intel-amd-amphon.png)

เรารับซื้อซีพียูคอมพิวเตอร์เดสก์ท็อป (Desktop Processor) ทุกรุ่น:

- **Intel Core Series:** รับตั้งแต่ Gen 8 จนถึงรุ่นใหม่ล่าสุด Gen 14 ทั้ง Core i9, Core i7, Core i5 และ Core i3 (รับทั้งรุ่นมี F, K, KS)
- **AMD Ryzen Series:** รับตระกูล AM4 และ AM5 ทั้ง Ryzen 9, Ryzen 7, Ryzen 5 และ Ryzen 3
- **Intel Xeon / AMD Threadripper:** ซีพียูระดับ Workstation และ Server เราก็รับซื้อ ให้ราคาตามสเปค

*(ซีพียูรุ่นที่เก่ากว่า Gen 8 (เช่น Gen 6, Gen 7) เรายังรับประเมินอยู่ครับ แต่อาจจะราคาไม่สูงมากตามความต้องการของตลาด)*

## สภาพแบบไหนที่เราพิจารณารับซื้อ?

1. **ใช้งานได้ปกติ สภาพสวย:** หน้าสัมผัสไม่เป็นรอยไหม้ ใช้งานได้ปกติทุกคอร์ (Core) และเทรด (Thread) ได้ราคาสูงสุด
2. **ขางอ (Pin Bent) เล็กน้อย:** สำหรับ CPU AMD หรือซีพียูรุ่นเก่าที่มีขาพินที่ตัว CPU หากขางอเรายังรับซื้อ แต่จะหักค่าบริการดัดขากลับครับ
3. **ไม่มีพัดลมแถม:** เรารับซื้อเฉพาะตัวซีพียูเปล่าๆ ก็ได้ ไม่ซีเรียสเรื่องพัดลมแถมครับ
4. **ซีพียูเสีย (Dead CPU):** เปิดไม่ติด บูทไม่ขึ้น เครื่องค้างจอฟ้า เราตีราคาเป็นซาก/อะไหล่ได้ครับ

## ปัจจัยหลักที่มีผลต่อราคาประเมินซีพียู

1. **เจเนอเรชันและซีรีส์:** Gen ยิ่งใหม่ ซีรีส์ยิ่งสูง (i9, Ryzen 9) ราคาประเมินยิ่งสูงตามความแรง
2. **การรับประกัน (Warranty):** ซีพียูส่วนใหญ่ประกันศูนย์ 3 ปี หากยังมีประกันศูนย์ไทยเหลือ (Synnex, WPG, Advice, JIB) จะช่วยเพิ่มราคาประเมินได้
3. **อุปกรณ์ครบกล่อง (Boxed vs Tray):** ซีพียูแบบ Box ที่มาพร้อมกล่องเดิมและพัดลมแถม (Heatsink) จะได้ราคาดีกว่าแบบตัวเปล่า (Tray) ถาดพลาสติก
4. **ร่องรอยการผ่ากระดอง (Delidding):** หากซีพียูมีร่องรอยการงัดแงะ หรือผ่ากระดองเพื่อใส่ Liquid Metal จะหมดประกันทันที และราคาตกเยอะครับ

ขั้นตอนง่ายๆ ไม่ยุ่งยาก:
1. **ถ่ายรูปตัวซีพียู:** ถ่ายสกรีนตัวหนังสือบนหน้ากระดอง (Heat Spreader) ให้ชัดเจนว่าระบุรุ่นอะไร
