amphon.co.th รับพิจารณารับซื้อโน๊ตบุ๊คไม่มี Adapterจากลูกค้าทั่วประเทศ ลูกค้าสามารถส่งรูปสินค้า รุ่น สเปก อาการเสีย จำนวนเครื่อง หรือ Inventory List มาให้ประเมินเบื้องต้นได้ทาง LINE: **@webuy** ก่อนตัดสินใจขาย บริการนี้วางตำแหน่งเป็นงานรับซื้อสินค้าไอทีมือสองระดับประเทศ ไม่ใช่บริการที่จำกัดเฉพาะจังหวัดใดจังหวัดหนึ่ง

เครื่องไม่มี Adapter ไม่มีที่ชาร์จ อุปกรณ์ไม่ครบ หรือทดสอบเปิดเครื่องไม่ได้เพราะสายชาร์จหาย คือแกนหลักของหน้านี้ เนื้อหาจึงเน้นการประเมินตามสภาพจริง การแจ้งอาการให้ตรง และการส่งข้อมูลให้ครบตั้งแต่ต้น เพื่อให้ทีมงานคุยรายละเอียดได้ตรงกับสภาพเครื่องจริง amphon.co.th มีหน้าร้านจริงที่จังหวัดอุบลราชธานี ใช้เป็นหลักฐานให้ลูกค้าตรวจสอบตัวตนและความน่าเชื่อถือได้ แต่ลูกค้าทั่วประเทศสามารถเริ่มจากการส่งรูปและข้อมูลเพื่อประเมินเบื้องต้นได้เหมือนกัน

## รับซื้อกรณีไหนบ้าง

กรณีที่เหมาะกับหน้านี้ ได้แก่

- มีแต่ตัวเครื่อง ไม่มี Adapter
- Adapter หายหลังย้ายบ้านหรือย้ายสำนักงาน
- เครื่องบริษัทคืนมาแต่สายชาร์จไม่ครบ
- มี Adapter แต่ไม่แน่ใจว่าตรงรุ่นหรือไม่
- มีเครื่อง USB-C แต่ไม่รู้ว่าหัวชาร์จที่มีใช้ได้หรือไม่

นอกจากรายการข้างต้น หากเครื่องมีหลายอาการร่วมกัน เช่น ไม่มี Adapter ร่วมกับเปิดไม่ติด หรือจอแตกพร้อมบานพับหัก ก็ยังส่งข้อมูลให้ประเมินเบื้องต้นได้ โดยควรแจ้งอาการทั้งหมดที่ทราบ ไม่ต้องซ่อมก่อนและไม่ต้องแกะเครื่องเอง

## เหมาะกับลูกค้ากลุ่มไหน

หน้านี้เหมาะกับลูกค้าทั่วไป บริษัท โรงเรียน สำนักงาน ร้านค้ามือสอง และผู้ที่ไม่ต้องการซื้อ Adapter ใหม่เพื่อทดสอบก่อนขาย รวมถึงลูกค้าที่ต้องการเปรียบเทียบราคาก่อนได้ก่อนตัดสินใจว่าจะขาย ซ่อม หรือเก็บเครื่องไว้ใช้งานต่อ

กรณีเป็นเครื่องส่วนตัว เครื่องบริษัท เครื่องจากสำนักงาน เครื่องที่เก็บไว้นาน หรือเครื่องที่รวบรวมมาเป็นล็อต สามารถส่งข้อมูลให้ทีมงานช่วยประเมินเบื้องต้นได้ จุดสำคัญคือข้อมูลต้องชัดเจนพอ เช่น รุ่น สเปก อาการ จำนวนเครื่อง และพื้นที่ตั้งสินค้า

## ต้องส่งข้อมูลอะไรเพื่อประเมินราคา

เพื่อให้ประเมินเบื้องต้นได้ไว แนะนำให้ส่งข้อมูลต่อไปนี้ทาง LINE: **@webuy**

- รูปด้านหน้าเครื่อง
- รูปด้านหลังเครื่อง
- รูปสเปกหรือหน้า About / System Information ถ้าเปิดได้
- ยี่ห้อ รุ่น และซีรีส์
- CPU / RAM / SSD ถ้าทราบ
- อาการเสียหรือตำหนิ
- อุปกรณ์ที่มี เช่น Adapter, กล่อง, กระเป๋า
- จำนวนเครื่อง
- พื้นที่ตั้งสินค้า
- กรณีบริษัทให้ส่ง Inventory List ได้

ถ้าเครื่องเปิดไม่ได้หรือไม่สามารถเข้าหน้า System Information ได้ ให้ถ่ายรูปป้ายรุ่นใต้เครื่อง สติ๊กเกอร์ Serial Number หรือรูปทรงเครื่องหลายมุมแทน หากมีเอกสารสเปกเดิม ใบเสนอราคาเดิม หรือรายการทรัพย์สินเดิม สามารถส่งประกอบได้

## วิธีถ่ายรูปเครื่องให้ประเมินได้ไว

ถ่ายรูปในที่สว่าง เห็นรายละเอียดชัด ไม่เบลอ และไม่ควรถ่ายเฉพาะมุมสวย เพราะการประเมินต้องดูสภาพจริง ควรถ่ายฝาเครื่อง ด้านใน ด้านหลัง ป้ายรุ่น พอร์ตสำคัญ และอุปกรณ์ที่มี เช่น Adapter กล่อง หรือกระเป๋า

ถ่ายพอร์ตชาร์จให้ชัดเป็นพิเศษ และถ้ามี Adapter ที่ไม่แน่ใจว่าใช่หรือไม่ ให้ถ่ายสเปกบน Adapter กับรูปหัวชาร์จมาด้วย ถ้าเป็น USB-C ให้แจ้งกำลังวัตต์ของหัวชาร์จถ้าทราบ หากมีหลายเครื่อง ให้ถ่ายภาพรวมก่อนหนึ่งชุด แล้วเลือกเครื่องตัวอย่างของแต่ละกลุ่มมาถ่ายละเอียดเพิ่ม เช่น กลุ่มเปิดติด กลุ่มมีตำหนิ กลุ่มเปิดไม่ติด และกลุ่มไม่มี Adapter วิธีนี้ช่วยให้ประเมินเบื้องต้นได้เป็นระบบมากกว่าการส่งรูปปะปนกันโดยไม่มีคำอธิบาย

## ราคาประเมินขึ้นกับอะไร

ราคาขึ้นกับรุ่น สเปก สภาพ อุปกรณ์ จำนวนเครื่อง อาการที่พบ และความชัดเจนของข้อมูลที่ส่งมา เครื่องที่มี CPU รุ่นใหม่ RAM เพียงพอ SSD ใช้งานได้ สภาพบอดี้ดี และมี Adapter ครบ จะประเมินต่างจากเครื่องที่มีตำหนิหลายจุดหรือทดสอบไม่ได้

เครื่องไม่มี Adapter จะประเมินต่างจากเครื่องที่มี Adapter ครบ เพราะอาจยังทดสอบไม่ได้ ถ้าตรวจจริงแล้วพบอาการอื่น เช่น เปิดไม่ติด จอเสีย หรือแบตมีปัญหา การประเมินจะยึดสภาพจริงของเครื่อง การประเมินจากรูปเป็นการประเมินเบื้องต้นเท่านั้น หากต้องตรวจเครื่องจริง ราคาหรือเงื่อนไขอาจเปลี่ยนตามสภาพที่พบจริง จึงควรแจ้งข้อมูลตรงไปตรงมาเพื่อให้การประเมินใกล้เคียงที่สุด

## เครื่องบริษัท / เครื่องยกล็อต / Inventory List ถ้าเกี่ยวข้อง

ในล็อตหลายเครื่องควรแยกคอลัมน์ Adapter ว่ามี ไม่มี ไม่แน่ใจ หรือไม่ตรงรุ่น เพื่อให้ประเมินตามอุปกรณ์ที่มีจริง ถ้าเป็นบริษัทหรือมีหลายเครื่อง แนะนำส่งรายการในรูปแบบตาราง เช่น ลำดับ, ยี่ห้อ, รุ่น, CPU, RAM, SSD, อาการ, Adapter, จำนวน และหมายเหตุ

Inventory List ไม่จำเป็นต้องซับซ้อนมาก แต่ควรแยกข้อมูลให้ชัด หากมี Asset Tag หรือหมายเลขทรัพย์สิน ให้ใส่เพิ่มได้ โดยเฉพาะงานบริษัท โรงเรียน หน่วยงาน หรือสำนักงานที่ต้องการตรวจเทียบรายการก่อนส่งมอบเครื่อง

## อาการเสียหรือตำหนิต้องแจ้งอะไรบ้าง

ไม่มี Adapter ไม่ได้แปลว่าเครื่องเสียเสมอไป แต่ทำให้การทดสอบจำกัด จึงควรแจ้งว่าเปิดทดสอบไม่ได้เพราะไม่มีที่ชาร์จ อาการที่ควรแจ้ง เช่น เปิดติดหรือไม่ ชาร์จเข้าไหม จอแสดงผลปกติไหม จอแตกหรือจอเป็นเส้นหรือไม่ แบตเตอรี่เก็บไฟไหม คีย์บอร์ดกดครบไหม บานพับแน่นไหม เครื่องเคยตก เคยโดนน้ำ หรือเคยซ่อมมาก่อนหรือไม่ หากมีการถอด SSD, RAM หรือแบตเตอรี่ ควรแจ้งด้วย

## การนัดตรวจ จัดส่ง หรือประเมินตามเงื่อนไข

การนัดตรวจ การจัดส่ง หรือการรับสินค้า ต้องพิจารณาตามจำนวนสินค้า ประเภทสินค้า อาการ และพื้นที่ตั้งสินค้า กรณีเครื่องจำนวนน้อย อาจเริ่มจากส่งรูปและข้อมูลเพื่อประเมินเบื้องต้นก่อน แล้วค่อยคุยขั้นตอนต่อไป

กรณีมีหลายเครื่อง มีเครื่องเสียหลายอาการ หรือมีสินค้าไอทีประเภทอื่นรวมอยู่ด้วย ควรส่งภาพรวมและรายการเครื่องก่อน เพื่อให้ทีมงานพิจารณาแนวทางที่เหมาะสม ไม่ควรส่งสินค้าโดยยังไม่ได้คุยรายละเอียด โดยเฉพาะเครื่องที่เปิดไม่ติด แบตบวม หรือเสียหายหนัก
