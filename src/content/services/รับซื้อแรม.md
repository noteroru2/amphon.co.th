## ประเภทของแรม (RAM) ที่เรารับซื้อและเป็นที่ต้องการของตลาด

เรารับซื้อหน่วยความจำคอมพิวเตอร์ทุกฟอร์มแฟกเตอร์ ครอบคลุมทั้งลูกค้าทั่วไปและองค์กรที่ต้องการขายเหมา:

### 1. แรมคอมพิวเตอร์ตั้งโต๊ะ (U-DIMM / Desktop RAM)
แรมขนาดมาตรฐานที่ใส่ในเคสคอมพิวเตอร์ทั่วไปและคอมประกอบเกมมิ่ง:
* **รับซื้อแรม DDR5:** แรมเทคโนโลยีล่าสุด (บัส 5200, 5600, 6000MHz ขึ้นไป) เป็นที่ต้องการของตลาดสายจัดสเปคคอมแรงและเครื่องรุ่นใหม่
* **รับซื้อแรม DDR4:** แรมยอดนิยมที่คนใช้งานเยอะที่สุด (บัส 2666, 3200, 3600MHz) ซื้อง่ายขายคล่อง 
* **รูปแบบการดีไซน์:** เรารับทั้ง **แรมเกมมิ่ง (Gaming RAM)** ที่มีซิงค์ระบายความร้อนอะลูมิเนียม หรือมีไฟ RGB อลังการ และ **แรมเปลือย (Value RAM)** ที่เป็นแผงวงจรสีเขียว/ดำธรรมดา

### 2. แรมโน๊ตบุ๊ค และ Mini PC (SO-DIMM / Laptop RAM)
แรมแผงสั้นที่ใช้ในโน๊ตบุ๊ค ออลอินวัน (AIO) และมินิพีซี:
* เรารับซื้อแรมโน๊ตบุ๊ค DDR4 และ DDR5 ทั้งความจุ 8GB, 16GB และ 32GB
* ส่วนใหญ่โน๊ตบุ๊คมักจะแถมแรมมาแผงเดียว เมื่อลูกค้าซื้อแรมมาเพิ่มเป็นคู่ ก็มักจะนำแรมแผงเก่าที่ติดเครื่องมาขาย เรายินดีรับซื้อครับ!

### 3. แรมเซิร์ฟเวอร์ และ Workstation (ECC Registered / Server RAM)
สำหรับลูกค้าองค์กร (B2B) และ Data Center เรามีทีมงานรับเหมาประมูลแรม ECC สำหรับ Server เช่น DDR4 ECC RDIMM เพื่อตัดจำหน่ายออกจากบริษัทอย่างเป็นระบบ

### 4. แบรนด์แรมชั้นนำที่ตลาดต้องการสูง
* **Corsair:** ซีรีส์ Vengeance (LPX / RGB Pro / DDR5), Dominator Platinum
* **Kingston:** ซีรีส์ FURY (Beast / Renegade) รุ่นเก่าๆ อย่าง HyperX เราก็รับ
* **G.Skill:** ซีรีส์ Trident Z (Neo / Royal), Ripjaws
* **แบรนด์อื่นๆ:** Adata (XPG), TeamGroup (T-Force / T-Create), Crucial, PNY, Thermaltake, KLEVV, Apacer, Silicon Power

## รับซื้อ RAM ตาม Generation ที่คนค้นจริง

### DDR3
แรม DDR3 ยังพบได้บ่อยในคอมตั้งโต๊ะรุ่นเก่า โน๊ตบุ๊คเก่า และเครื่ององค์กรที่ยังไม่อัปเกรดทั้งล็อต หากเป็นแรมใช้งานได้ปกติและมีจำนวนหลายแผง เรายังรับพิจารณาตามรุ่น ความจุ และความครบของล็อตได้

### DDR4
DDR4 เป็นกลุ่มที่ยังมีการซื้อขายต่อเนื่องที่สุด ทั้งแรมโน๊ตบุ๊ค แรมคอมประกอบ และแรมจากเครื่องบริษัทเก่า ความจุ 8GB, 16GB และชุดคู่ที่สเปกตรงกัน มักประเมินได้ง่ายกว่าแผงเดี่ยวข้อมูลไม่ครบ

### DDR5
DDR5 เหมาะกับกลุ่มเครื่องรุ่นใหม่ คอมประกอบ และเครื่องงานที่อัปเกรดสเปกมาแล้ว เวลาส่งประเมินควรถ่ายรูปสติกเกอร์รุ่น บัส ความจุ และแจ้งว่าขายเป็นแผงเดี่ยวหรือเป็นคู่ เพื่อให้ตีราคาเบื้องต้นได้ใกล้เคียงขึ้น

## สภาพไหนที่เราพิจารณารับซื้อ? (บอกตรงๆ ไม่หมกเม็ด)

เพื่อความโปร่งใส นี่คือเกณฑ์การพิจารณาสภาพแรมของเรา:

1. **สภาพสวยกริ๊บ ใช้งานสมบูรณ์ (ราคาดีสุด):** ขาทองเหลือง (Gold Finger) สะอาด ซิงค์ระบายความร้อนไม่มีรอยบิ่น สติ๊กเกอร์วอยด์ครบ กล่องตรง 
2. **มีร่องรอยการใช้งานหนัก:** ซิงค์อะลูมิเนียมมีคราบออกไซด์ (Oxide) หรือคราบเกลือ/สนิม สติ๊กเกอร์ประกันลอกขาด เรารับซื้อครับ แต่อาจจะต้องหักลบราคาประเมินลงบ้าง เนื่องจากอาจจะหมดประกันศูนย์
3. **ไม่มีกล่อง:** รับซื้อปกติครับ ลูกค้าแค่ห่อบับเบิ้ลกันกระแทกให้หนาๆ เวลาจัดส่งมาให้เรา
4. **แรมพัง จอฟ้า เปิดไม่ติด (Dead RAM):** หากคุณนำแรมไปเสียบแล้วเมนบอร์ดร้องเตือน หรือบูท Windows แล้วจอฟ้า เรารับซื้อเป็น **ราคาซากอะไหล่อิเล็กทรอนิกส์ (E-Waste)** ชั่งกิโลกรัมเพื่อนำไปรีไซเคิลครับ

## 4 ปัจจัยหลักที่ใช้กำหนดราคาประเมิน RAM

อยากรู้ว่าแรมของคุณได้ราคาเท่าไหร่? เช็คสเปคตาม 4 ข้อนี้เลย:
1. **รุ่นเทคโนโลยี (DDR5 vs DDR4):** DDR5 เป็นของใหม่ ราคาย่อมสูงกว่า DDR4 มาก 
2. **ความจุ (Capacity):** ความจุคือสิ่งสำคัญที่สุด แรมแผงละ 16GB หรือคู่ละ 32GB (16x2) จะได้ราคาดีกว่าการขายแรมแผงละ 8GB หลายๆ แผง 
3. **ความเร็วบัส (Bus Speed) และค่า CL:** ยิ่งแรมมีบัสสูง (เช่น DDR5 บัส 6400MHz หรือ DDR4 บัส 3600MHz) และมีค่า CAS Latency (CL) ต่ำๆ ยิ่งดึงดูดสายโอเวอร์คล็อก และขายได้ราคาแพง
4. **ประกัน Life Time ศูนย์ไทย:** แรมส่วนมากมีประกัน "ตลอดอายุการใช้งาน" หากสติ๊กเกอร์ผู้นำเข้า (Synnex, Ascenti, JIB, Advice, WPG) อยู่ครบถ้วน มูลค่าของแรมจะแทบไม่ตกเลย
