# รับซื้อแรมมือสอง (RAM) DDR4, DDR5 ให้ราคาดีสุด ตีราคาไว

คุณเพิ่งอัปเกรดแรมใหม่ให้แรงขึ้น หรือถอดแรมจากคอมพิวเตอร์ที่เสียแล้วออกมาใช่ไหม? แรม (RAM - Random Access Memory) เป็นชิ้นส่วนที่เสียยากมาก และเป็นที่ต้องการของตลาดอยู่เสมอ! **อำพล เทรดดิ้ง** รับซื้อแรมมือสองทุกประเภท ทั้งแรมสำหรับคอมพิวเตอร์ตั้งโต๊ะ (Desktop PC) และแรมสำหรับโน๊ตบุ๊ค (Notebook) ตั้งแต่ยุค DDR4 จนถึง DDR5 รุ่นล่าสุด ประเมินราคาอย่างเป็นธรรม จ่ายเงินสดทันที!

## ประเภทแรมที่เรารับซื้อ

![รับซื้อแรม RAM มือสอง DDR4 DDR5 - อำพล เทรดดิ้ง](/images/services/rub-sue-ram-ddr4-ddr5-amphon.png)

เรารับซื้อแรมจากคอมพิวเตอร์หลากหลายประเภท แบรนด์ดัง และความจุ:

- **ชนิดของแรม:** DDR5 (ล่าสุด) และ DDR4 (ยอดฮิต) ส่วนรุ่นเก่าอย่าง DDR3 เรายังรับเหมาในราคาเหมาจ่ายครับ
- **สำหรับเครื่อง PC (U-DIMM):** รับทั้งแรมบ้านๆ (แรมเขียว) และแรมเกมมิ่งที่มีซิงค์ระบายความร้อน หรือมีไฟ RGB สวยงาม
- **สำหรับโน๊ตบุ๊ค (SO-DIMM):** แรมโน๊ตบุ๊คแบบถอดได้ เราก็รับซื้อครับ
- **แบรนด์ยอดนิยม:** Corsair, Kingston (Fury/HyperX), G.Skill, Adata, XPG, TeamGroup, Crucial, PNY, Thermaltake
- **ความจุ:** รับซื้อตั้งแต่แผงละ 4GB, 8GB, 16GB, 32GB และ 48GB ขึ้นไป

## สภาพแบบไหนที่เราพิจารณารับซื้อ?

1. **ใช้งานได้ปกติ สภาพสวย:** ตัวแรมไม่มีรอยไหม้ ทองเหลืองไม่ลอกหรือหลุด ใช้งานได้ปกติ ไม่จอฟ้า (Blue Screen) ได้ราคาดีที่สุด
2. **มีรอยคราบอ็อกไซด์ (Oxide):** หากมีคราบเกาะที่ทองเหลืองแต่ยังขัดออกและใช้งานได้ เรารับซื้อครับ แต่อาจหักลบราคาประเมินเล็กน้อย
3. **สติ๊กเกอร์รับประกันหลุดหาย:** แรมที่มีสติ๊กเกอร์ประกันหาย หรือวอยด์ขาด จะถูกตีเป็นของหมดประกัน (ไม่มีประกัน Life Time แล้ว) ซึ่งราคารับซื้อจะลดลงครับ
4. **แรมเสีย (Dead RAM):** สำหรับแรมที่ช็อต หรือเปิดไม่ติด เรายังรับซื้อในราคาซากอิเล็กทรอนิกส์ (เหมาเป็นกิโลกรัม) สำหรับงานรีไซเคิลครับ

## ปัจจัยหลักที่มีผลต่อราคาประเมิน RAM

1. **เจเนอเรชันและความเร็ว (BUS):** แรม DDR5 จะมีราคาสูงกว่า DDR4 และความเร็วบัส (BUS Speed) ที่สูงกว่า เช่น บัส 6000MHz จะแพงกว่าบัส 5200MHz หรือ 3200MHz
2. **ความจุ (Capacity):** ความจุรวมต่อคู่ หรือต่อแผง ยิ่งเยอะราคายิ่งสูง
3. **การออกแบบ (Heatsink & RGB):** แรมสายเกมเมอร์ที่มีซิงค์อะลูมิเนียมระบายความร้อน หรือมีไฟ RGB จะได้ราคาประเมินสูงกว่าแรมเปลือย (Bare PCB)
4. **การรับประกัน (Warranty):** แรมส่วนใหญ่มีการรับประกันแบบตลอดอายุการใช้งาน (Lifetime Warranty) หากสติ๊กเกอร์ประกันศูนย์ไทย (Synnex, Ascenti, JIB) อยู่ครบถ้วน จะปล่อยขายง่ายและได้ราคาดีมาก

| :--- | :--- | :--- |
| **DDR5 คู่ 32GB (16GBx2) RGB** | สภาพสวย ประกัน LT | 2,500 - 4,000+ บาท |
| **DDR5 แผงเดี่ยว 16GB / 8GB** | สภาพสวย ประกัน LT | 800 - 1,500+ บาท |
| **DDR4 คู่ 32GB (16GBx2) RGB** | ใช้งานปกติ ประกัน LT | 1,500 - 2,500 บาท |
| **DDR4 คู่ 16GB (8GBx2) RGB** | ใช้งานปกติ ประกัน LT | 800 - 1,500 บาท |
| **แรมโน๊ตบุ๊ค DDR4 8GB / 16GB** | ใช้งานปกติ ประกัน LT | 300 - 800+ บาท |
| **แรมเก่า DDR3 / แรมเสีย** | ตีเป็นราคาอะไหล่ | เหมากิโลกรัมละ 200+ บาท |

## ขั้นตอนการส่งข้อมูลเพื่อประเมินราคา

ขั้นตอนง่ายๆ ไม่ยุ่งยาก:
1. **ถ่ายรูปตัวแรม:** ถ่ายรูปชัดๆ ที่สติ๊กเกอร์สเปคบนตัวแรม (จะระบุว่าเป็น DDR อะไร, ความจุเท่าไหร่, ความเร็ว Bus เท่าไหร่)
2. **ถ่ายรูปสติ๊กเกอร์รับประกัน:** เพื่อเช็คว่าประกันจากศูนย์ (Distributor) ไหน และยังไม่ฉีกขาด
3. **ระบุจำนวน:** แจ้งว่ามีกี่แผง เป็นคู่ที่มาด้วยกันแต่แรก (Kit) หรือแยกชิ้นมา
