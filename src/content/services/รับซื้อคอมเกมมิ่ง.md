## รับซื้อคอมกลุ่มนี้แบบไหนบ้าง

คอมเกมมิ่งที่เหมาะกับการส่งมาประเมินอาจเป็นคอมประกอบเล่นเกม คอมมีไฟ RGB คอมมีเคสกระจก คอมที่ใส่การ์ดจอ GTX, RTX หรือ Radeon คอมที่ใช้ CPU Intel Core หรือ AMD Ryzen และเครื่องที่เคยใช้งานกับเกมออนไลน์ เกม AAA โปรแกรมสตรีม หรือโปรแกรมตัดต่อเบื้องต้น

ไม่จำเป็นต้องเป็นเครื่องใหม่หรือเครื่องสเปกแรงมากเท่านั้น เครื่องเล่นเกมรุ่นเก่าก็ส่งรูปมาให้ดูได้ เช่น เครื่องที่ใช้ GTX 1060, GTX 1660, RTX 2060, RTX 3060, RX 580, RX 6600 หรือรุ่นใกล้เคียง การประเมินจะดูองค์ประกอบทั้งชุด ไม่ได้ดูแค่ชื่อการ์ดจออย่างเดียว

หากเป็นเครื่องประกอบที่เปลี่ยนอะไหล่มาหลายรอบ ควรแจ้งด้วยว่าอะไหล่ชิ้นใดเป็นของเดิมและชิ้นใดเคยเปลี่ยน เช่น เพิ่ม RAM เปลี่ยน SSD เปลี่ยน PSU หรือถอด HDD ออกแล้ว ข้อมูลเหล่านี้ช่วยให้ประเมินตามสภาพจริงได้ชัดขึ้น

## เหมาะกับลูกค้ากลุ่มไหน

หน้านี้เหมาะกับนักเรียน นักศึกษา คนทำงาน เกมเมอร์ทั่วไป ผู้ปกครองที่ต้องการขายเครื่องของลูก เจ้าของร้านเล็ก ๆ หรือคนที่อัปเกรดคอมใหม่แล้วมีเครื่องเก่าเหลืออยู่ ถ้าไม่อยากแยกขายการ์ดจอ CPU RAM SSD ทีละชิ้น สามารถส่งรายละเอียดคอมทั้งชุดมาให้ประเมินได้

ยังเหมาะกับคนที่มีคอมเกมมิ่งแต่ไม่ได้ใช้งานนานแล้ว เช่น เปิดติดบ้างไม่ติดบ้าง ฝุ่นเยอะ พัดลมดัง ระบบ Windows มีปัญหา หรือไม่แน่ใจว่าเครื่องยังเล่นเกมได้ดีแค่ไหน ให้แจ้งอาการตามจริง ไม่ต้องพยายามซ่อมหรือแกะอะไหล่ก่อนส่งข้อมูล

## ต้องส่งข้อมูลอะไรเพื่อประเมิน

ข้อมูลที่ควรส่งสำหรับคอมเกมมิ่งมีดังนี้

- สเปก CPU / Mainboard / RAM / SSD / HDD
- การ์ดจอ เช่น GTX / RTX / Radeon ถ้ามี
- PSU / Case / ชุดระบายความร้อน ถ้าทราบ
- อาการเสียหรือตำหนิ เช่น เปิดช้า ค้าง ดับเอง จอฟ้า ภาพลาย
- อุปกรณ์ที่มี เช่น จอ คีย์บอร์ด เมาส์ สายไฟ กล่อง
- จำนวนเครื่อง
- พื้นที่ตั้งสินค้า

## ราคาประเมินขึ้นกับอะไร

ราคาขึ้นกับรุ่น สเปก สภาพ อุปกรณ์ และจำนวนเครื่อง โดยคอมเกมมิ่งจะดูหลายจุด เช่น รุ่นการ์ดจอ ความจุ RAM ประเภท SSD รุ่น CPU อายุการใช้งาน PSU เคส ระบบระบายความร้อน ความสะอาดภายใน สภาพพอร์ต และอาการขณะใช้งานจริง

เครื่องที่มีสเปกชัดเจน มีอุปกรณ์ครบตามที่แจ้ง และทดสอบเปิดใช้งานได้มักประเมินได้ตรงกว่าเครื่องที่ข้อมูลไม่ครบ ส่วนเครื่องที่มีตำหนิ เช่น บานฝาข้างหาย ไฟ RGB ไม่ติด พัดลมดัง หรือ Windows เสีย ยังส่งรูปมาให้พิจารณาเบื้องต้นได้ แต่ต้องแจ้งรายละเอียดตามจริง

## เครื่องมีตำหนิ เปิดไม่ติด หรืออุปกรณ์ไม่ครบ ประเมินได้ไหม

กรณีไม่แน่ใจว่าเสียที่การ์ดจอ เมนบอร์ด หรือ PSU อย่าเพิ่งสรุปอาการเอง ให้แจ้งอาการที่เห็นจริง เช่น “กดเปิดแล้วพัดลมหมุนแต่ไม่ขึ้นภาพ” หรือ “เล่นเกมแล้วดับ” เพราะอาการเดียวกันอาจเกิดจากหลายสาเหตุและต้องตรวจจริงก่อนสรุป

ถ้าคุณมีคอมเกมมิ่งมือสองที่ต้องการขาย ให้ส่งรูปเครื่อง สเปก อาการ และอุปกรณ์ที่มีมาทาง LINE: @webuy ได้ ทีมงานจะประเมินเบื้องต้นตามข้อมูลจริง และแจ้งแนวทางต่อไปตามจำนวนเครื่อง ประเภทสินค้า อาการ และพื้นที่ตั้งสินค้า
