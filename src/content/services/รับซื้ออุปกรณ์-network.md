## รับซื้ออุปกรณ์ Network แบบไหนบ้าง

### Router และ Core Router

Router ที่ใช้เชื่อมต่อระบบเครือข่ายภายในกับ Internet หรือเชื่อมระหว่าง Site รับพิจารณาทั้ง Router สำหรับ SME และ Router ระดับ Enterprise พร้อม Module เสริมถ้ามี

### Managed Switch ทุกระดับ

Switch ระดับ L2 และ L3 ทั้ง Access Layer, Distribution Layer และ Core Switch รับพิจารณาตาม Port Count, Speed (1G, 10G, 25G) และ PoE Capability อุปกรณ์ที่ Support Stack หรือมี SFP+ Uplink มีมูลค่าสูงกว่า Switch ทั่วไปในจำนวน Port เท่ากัน

### Firewall และ UTM

Firewall แบบ Hardware Appliance จากทุกแบรนด์ รับพิจารณาตามรุ่นและ Throughput Hardware ยังมีมูลค่าแม้ Subscription License หมดแล้ว เพราะผู้ซื้อบางรายต่อ Renewal เองหรือใช้ในงานที่ไม่ต้องการ Subscription

### Access Point และ Wireless Controller

Access Point ทั้งแบบ Ceiling Mount, Wall Mount และ Outdoor Access Point รับพิจารณาตามรุ่นและ Wi-Fi Standard (Wi-Fi 5, Wi-Fi 6, Wi-Fi 6E) Wireless Controller แบบ Hardware รับพิจารณาตามรุ่นและจำนวน AP ที่จัดการได้

### Media Converter และ Transceiver

Media Converter สำหรับแปลง Fiber เป็น Ethernet รับพิจารณาตามประเภทและ Speed SFP, SFP+ และ QSFP Transceiver ที่ยังดีรับพิจารณา โดยเฉพาะ Cisco-compatible Optic Module ที่มีตลาดในตัว

## ข้อมูลที่ต้องส่งเพื่อประเมินราคาอุปกรณ์ Network

อุปกรณ์ Network มี Part Number และ Model ที่กำหนดราคาได้ชัดกว่าอุปกรณ์ประเภทอื่น ยิ่งแจ้งรุ่นได้ตรง ยิ่งประเมินได้เร็วและแม่นขึ้น

**รูปหน้าเครื่อง** — ถ่ายให้เห็น Port Panel ทั้งหมด สถานะ LED และ Model Name ที่หน้าตัวเครื่อง

**รูปหลังเครื่อง** — ถ่ายให้เห็น Uplink Port, Power Input, Console Port และ Module Slot ถ้ามี

**รูป Label รุ่นและ Serial** — สติกเกอร์ด้านล่างหรือด้านหลังเครื่อง มักมี Model Number, Part Number (P/N) และ Serial Number ครบ

**รุ่นและแบรนด์** — เช่น Cisco Catalyst WS-C2960X-24PS-L, Fortigate FG-60F, Ubiquiti UAP-AC-PRO, MikroTik CCR1009-7G-1C-1S+

**จำนวน Port** — แจ้งจำนวน Port ทั้งหมดและ Speed เช่น 24x 1G + 4x SFP+

**PoE หรือไม่** — ถ้าเป็น PoE Switch แจ้ง Total PoE Budget ถ้าทราบ เพราะมีผลต่อราคามาก

**Adapter หรือสายไฟ** — อุปกรณ์ Network Desktop มักใช้ Adapter แยก แจ้งว่ามีมาด้วยหรือไม่

**Rack Mount Kit** — ถ้าเป็น Rack-mount รุ่น แจ้งว่ามี Rack Ear มาด้วยหรือไม่

**จำนวนเครื่อง** — ถ้ามีหลายชิ้น แจ้งจำนวนรวมและว่ารุ่นเดียวกันหรือต่างรุ่น

**สภาพการใช้งาน** — ทำงานได้ปกติ, Port บางตัวไม่ทำงาน, LED มีปัญหา หรือไม่ได้ทดสอบ

**พื้นที่ตั้งสินค้า** — จังหวัดหรือพื้นที่คร่าวๆ เพื่อวางแผนการรับสินค้า

## วิธีถ่ายรูปอุปกรณ์ Network ให้ประเมินได้ไว

อุปกรณ์ Network เล็กและเบา แต่ข้อมูลสำคัญกระจายอยู่หลายจุด

**3 รูปที่ต้องมีก่อน:**

ถ่ายด้านหน้าให้เห็น Port Panel และ LED Status ทั้งหมด ถ่ายด้านล่างหรือด้านหลังที่มี Label แสดง Model, P/N และ Serial ถ่ายรวมถ้ามีหลายชิ้นวางอยู่ด้วยกัน

**สิ่งที่ช่วยให้ประเมินได้แม่นขึ้น:**

ถ้าอุปกรณ์มี Module เสริม เช่น SFP Module หรือ Line Card ถ่ายให้เห็นด้วย สำหรับ Access Point ถ่ายป้ายที่ด้านหลังหรือด้านล่างของตัวเครื่องจะมีรุ่นครบ

**สำหรับล็อตหลายชิ้น:**

ถ่ายภาพรวมให้เห็นจำนวนก่อน จากนั้นถ่าย Label ของตัวแทน 1–2 ชิ้นในแต่ละรุ่น ถ้ารุ่นเดียวกันทั้งหมดไม่ต้องถ่ายทุกชิ้น

## ราคาอุปกรณ์ Network มือสองขึ้นกับอะไร

### แบรนด์และ Market Position

แบรนด์มีผลต่อราคามากที่สุดในอุปกรณ์ Network Cisco มีมูลค่าในตลาดมือสองสูงกว่าแบรนด์อื่นอย่างชัดเจน ตามด้วย Juniper, Fortinet และ Aruba ที่มีตลาด Enterprise รองรับ Ubiquiti และ MikroTik มีตลาดของตัวเองที่ชัดเจนในกลุ่ม SME และผู้ดูแลระบบ TP-Link Omada และ Netgear Managed มักมีราคาต่ำกว่าในจำนวน Port เท่ากัน แต่ยังมีตลาดในกลุ่มลูกค้าที่งบประมาณจำกัด

### รุ่นและ Feature Set

Switch ที่รองรับ Layer 3 Routing มีราคาสูงกว่า L2 Only ใน Port Count เดียวกัน PoE Switch มีราคาดีกว่า Non-PoE ชัดเจน โดยเฉพาะถ้า PoE Budget สูง Firewall ที่มี Throughput สูงและรองรับ UTM Feature ครบมีราคาดีกว่า Basic Firewall

### Speed และ Port Count

10GbE Switch ยังมีราคาสูงกว่า 1GbE อย่างชัดเจน จำนวน Port มีผลตรง Switch 48-Port ราคาดีกว่า 24-Port ในสัดส่วนที่ไม่เท่ากัน

### สถานะ License และ Support

Cisco SmartNet, Fortinet FortiCare และ Juniper JNAPT ที่ยังมีอยู่ช่วยเพิ่มมูลค่าได้ แต่อุปกรณ์ที่ License หมดแล้วยังมีราคาในตลาดมือสองที่ต้องการซื้อ Hardware อย่างเดียว

### สภาพ Port และ Cosmetic

Port ที่ไม่ทำงานหรือ LED ผิดปกติมีผลต่อราคา แต่รอยขีดข่วนเล็กน้อยบน Body ไม่ส่งผลมากนักถ้าอุปกรณ์ทำงานได้ปกติ

## อุปกรณ์ที่ Reset Config แล้วหรือยังไม่ได้ Reset — ควรทำอย่างไร

### ทำไมต้อง Reset ก่อนส่งมอบ

Config ใน Switch และ Router มักมีข้อมูลโครงสร้างเครือข่ายขององค์กร รวมถึง VLAN, Password, SNMP Community String และ Management IP ที่ควรลบออกก่อนส่งมอบ Firewall มักมี Policy Rule, VPN Credential และ User Account ที่เป็นข้อมูลสำคัญขององค์กร

### วิธี Reset โดยประเภทอุปกรณ์

สำหรับ Cisco Switch และ Router: ใช้คำสั่ง `write erase` แล้ว Reload หรือกด Reset Button ค้างตามคู่มือรุ่นนั้น สำหรับ Fortigate: ไปที่ System → Reset หรือใช้คำสั่ง `execute factoryreset` ผ่าน Console สำหรับ Ubiquiti: Reset Button ค้าง 10 วินาทีหรือใช้ UniFi Controller ถ้ายังเชื่อมต่ออยู่ สำหรับ MikroTik: ใช้ Netinstall หรือกด Reset Button ตามคู่มือรุ่น

### ถ้ายัง Reset ไม่ได้

แจ้งมาให้ทางร้านทราบก่อนนัด ทางร้านจะช่วยประเมินว่าสถานการณ์นั้นมีผลต่อการรับซื้ออย่างไร อย่างน้อยควรลบ Password เดิมออกหรือตั้งเป็น Default ก่อน ถ้าทำได้

## สถานการณ์ที่มักพบในงานรับซื้ออุปกรณ์ Network

### บริษัทอัปเกรดจาก Unmanaged เป็น Managed Switch

เมื่อบริษัทเติบโตและต้องการแบ่ง VLAN หรือควบคุม Traffic มักซื้อ Managed Switch ใหม่และมี Switch เก่าที่ยังใช้งานได้เหลืออยู่ รับพิจารณาทั้งล็อต

### เปลี่ยน Wi-Fi Infrastructure ทั้งสำนักงาน

โรงแรมหรือสำนักงานที่อัปเกรดจาก Access Point เก่าเป็น Wi-Fi 6 มักมี AP เดิมหลายสิบตัวที่ยังสภาพดี แจ้งรุ่นและจำนวนมาก่อน

### เปลี่ยน Firewall เนื่องจาก License หมดและไม่ต่ออายุ

กรณีที่พบบ่อย บริษัทซื้อ Firewall ใหม่แทนที่จะต่อ Renewal Hardware เก่ายังมีมูลค่า แจ้งรุ่นและปีที่ซื้อมาก่อน

### ปิดสาขาหรือย้ายที่ทำการ

เมื่อปิดสาขา อุปกรณ์ Network จากสาขานั้นมักไม่ได้ย้ายตามเพราะสาขาใหม่มีระบบของตัวเอง รับพิจารณาทั้งล็อต อุปกรณ์ Network เบาและขนสะดวกกว่า Server มาก

## รับซื้ออุปกรณ์ Network จากลูกค้าทั่วประเทศ

อุปกรณ์ Network ที่เบาและส่งได้ง่ายกว่า Server หรือ NAS ทำให้ลูกค้าจากต่างจังหวัดมีตัวเลือกมากกว่าในการส่งสินค้ามา แต่ขั้นตอนแรกคือส่งรุ่นและรูปมาประเมินก่อนเสมอ

อำพล เทรดดิ้ง มีหน้าร้านจริงที่อุบลราชธานีเป็นหลักฐานว่ามีตัวตนที่ตรวจสอบได้ สำหรับอุปกรณ์ Network มูลค่าสูงหรือล็อตใหญ่ ลูกค้าสามารถตรวจสอบข้อมูลร้านก่อนดำเนินการได้เสมอ

## ความต่างระหว่างอุปกรณ์ Network, Server, NAS และ UPS

อุปกรณ์ทั้งสี่ประเภทนี้มักอยู่ในห้อง IT เดียวกัน แต่มีวิธีประเมินและปัจจัยราคาต่างกันอย่างชัดเจน

| | **Network Equipment** | **Server** | **NAS** | **UPS** |
|---|---|---|---|---|
| **จุดประสงค์** | เชื่อมต่อเครือข่าย | Compute / Application | จัดเก็บและแชร์ข้อมูล | สำรองไฟฟ้า |
| **ตัวอย่าง** | Cisco, Fortigate, Ubiquiti | Dell PowerEdge, HP ProLiant | Synology, QNAP | APC, Eaton, CyberPower |
| **ปัจจัยราคาหลัก** | แบรนด์, L2/L3, PoE, Port Speed | Generation, CPU, RAM, Drive | แบรนด์, Bay, CPU, HDD | VA Rating, แบตเตอรี่ |
| **สิ่งสำคัญก่อนขาย** | Factory Reset ลบ Config/Password | Secure Erase Drive, Reset iDRAC | Factory Reset / ถอด Drive | เช็กสภาพแบตเตอรี่ |
| **น้ำหนัก/ขนส่ง** | เบา ส่งพัสดุได้ | หนัก วางแผนต้องการ | เบา-กลาง ส่งได้ | กลาง-หนักตามขนาด |
| **License** | มีผลบางแบรนด์ | OS License แยก | ไม่ค่อยมี | ไม่มี |

ทางร้านรับพิจารณาทุกประเภท ส่งรายการรวมมาในครั้งเดียวได้เลย
