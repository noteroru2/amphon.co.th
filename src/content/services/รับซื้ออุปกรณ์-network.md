## อุปกรณ์เครือข่ายและแบรนด์ที่เรารับซื้อ

![รับซื้ออุปกรณ์ Network Cisco Mikrotik Aruba - อำพล เทรดดิ้ง อุบลราชธานี](/images/services/rub-sue-network-cisco-amphon.png)

เรารับซื้ออุปกรณ์เน็ตเวิร์ค (Network Equipments) ทุกประเภท ครอบคลุมตั้งแต่อุปกรณ์สำหรับบ้านพักอาศัย, องค์กรขนาดเล็ก (SMB) ไปจนถึงระดับ Enterprise (Data Center):

**1. Network Switch (สวิตช์ฮับ)**
- **แบรนด์ชั้นนำ:** Cisco (Catalyst, Nexus), Aruba, Juniper, Fortinet
- **แบรนด์ทั่วไป:** Mikrotik, Ubiquiti (UniFi), TP-Link, D-Link, Zyxel, Ruijie, Netgear
- **ประเภทที่รับ:** L2/L3 Managed Switch, PoE Switch (Power over Ethernet), 10G/40G Fiber Switch

**2. Router & Firewall (เร้าเตอร์และไฟร์วอลล์)**
- **Firewall/Security:** Fortinet (FortiGate Series), Palo Alto, Check Point, Cisco ASA, Sophos, WatchGuard
- **Router:** Cisco ISR/ASR Series, Mikrotik Cloud Core Router (CCR)

**3. Wireless Access Point (ตัวกระจายสัญญาณ WiFi)**
- Ubiquiti (UniFi AP ทุกรุ่น), Aruba IAP, Cisco Aironet / Catalyst AP, Ruckus, EnGenius, Ruijie (Reyee)

**4. อุปกรณ์สื่อสารและ VoIP**
- IP Phone (Cisco, Grandstream, Yealink), Video Conference System, PABX ตู้สาขาโทรศัพท์

**5. อุปกรณ์เสริมตู้ Rack และสายสัญญาณ**
- สาย LAN (CAT5e, CAT6) ที่รื้อถอนแล้ว, สาย Fiber Optic Patch Cord, SFP Module (1G/10G/40G)
- รางปลั๊ก (PDU), ถาดรอง Rack, KVM Switch

## สภาพแบบไหนที่เราประเมินราคาให้?

- **สภาพดี ใช้งานได้ปกติ:** อุปกรณ์ที่ยังทำงานได้ทุกพอร์ต ไม่เคยซ่อม สภาพภายนอกสวย ได้ราคาสูงสุด
- **เก่าเก็บ ตกรุ่น:** อุปกรณ์ที่ความเร็ว 10/100 หรือ Switch รุ่นเก่าๆ ที่องค์กรไม่ได้ใช้แล้ว เรารับเหมาซื้อทั้งหมด
- **เสีย เปิดไม่ติด พอร์ตพัง:** อุปกรณ์ที่โดนฟ้าผ่า เปิดไม่ติด หรือ PoE จ่ายไฟไม่ได้ เรารับซื้อเป็นราคาซากอิเล็กทรอนิกส์ เพื่อนำไปคัดแยกอะไหล่และส่งโรงงานรีไซเคิลต่อไป
- **รับเหมาทั้งโครงการ (Turnkey Project):** เรารับเหมารื้อถอนระบบ Network ทั้งตึก สายไฟ สายแลน ตู้แร็ค รับเหมาซื้อในครั้งเดียว

## ปัจจัยที่มีผลต่อการประเมินราคาอุปกรณ์ Network

1. **แบรนด์และรุ่น:** Cisco, Fortinet, Aruba และ Mikrotik เป็นแบรนด์ยอดนิยมในตลาดมือสอง ทำให้ได้ราคาประเมินที่ดีและขายง่าย
2. **ความเร็วและพอร์ต:** Switch ระดับ Gigabit (10/100/1000) หรือ 10G Uplink จะได้ราคาดีกว่า Switch 10/100 แบบเก่า
3. **ฟังก์ชัน PoE (Power over Ethernet):** Switch ที่จ่ายไฟได้ (PoE) มีความต้องการสูงมากสำหรับงานติดตั้งกล้องวงจรปิด ทำให้ราคาดีกว่า Switch ธรรมดา
4. **License และ Subscription:** สำหรับอุปกรณ์อย่าง Firewall (เช่น FortiGate) หากยังมี License หรือ Subscription เหลืออยู่ จะเพิ่มมูลค่าได้มหาศาล
5. **ความสมบูรณ์ของอุปกรณ์เสริม:** อแดปเตอร์แท้, หูจับเข้าตู้แร็ค (Rack Mount Kit), สาย Console ถือเป็นส่วนสำคัญที่ผู้ซื้อต่อต้องการ

## บริการรื้อถอนและทำลายข้อมูล (ITAD Service)

สำหรับหน่วยงานราชการ ธนาคาร หรือองค์กรขนาดใหญ่ที่มีนโยบายความปลอดภัยของข้อมูลเคร่งครัด:

## ทำไมบริษัทต่างๆ ถึงเลือกใช้บริการ อำพล เทรดดิ้ง?

- **รับซื้อจริง มีเงินทุนพร้อม:** เรามีประสบการณ์รับเหมาโปรเจกต์ขนาดใหญ่ จ่ายเงินหลักแสนถึงหลักล้านด้วยเงินสดหรือโอนเข้าบัญชีบริษัทได้ทันที
- **ให้ราคายุติธรรม:** เราตีราคาตามกลไกตลาดและ Part Number ของอุปกรณ์ ไม่ได้ตีเหมาเป็นเศษเหล็กแบบร้านรับซื้อของเก่าทั่วไป
- **มืออาชีพ เชื่อถือได้:** เราเข้าใจระบบ Network และระเบียบการจัดซื้อจัดจ้างขององค์กร สามารถออกเอกสารได้ครบถ้วน
- **มีรถรับถึงที่ ทั่วอีสาน:** หมดปัญหาเรื่องการขนย้าย เราจัดการให้ตั้งแต่ยกของจนถึงเคลียร์พื้นที่ทำความสะอาดหลังการขาย
