amphon.co.th รับพิจารณารับซื้อโน๊ตบุ๊คบริษัทจากลูกค้าทั่วประเทศ ลูกค้าสามารถส่งรูปสินค้า รุ่น สเปก อาการเสีย จำนวนเครื่อง หรือ Inventory List มาให้ประเมินเบื้องต้นได้ทาง LINE: **@webuy** ก่อนตัดสินใจขาย บริการนี้วางตำแหน่งเป็นงานรับซื้อสินค้าไอทีมือสองระดับประเทศ ไม่ใช่บริการที่จำกัดเฉพาะจังหวัดใดจังหวัดหนึ่ง

เครื่องพนักงานเก่า เครื่ององค์กร เครื่องบริษัทปลดระวาง เครื่องมี Asset Tag และ Inventory List คือแกนหลักของหน้านี้ เนื้อหาจึงเน้นการประเมินตามสภาพจริง การแจ้งอาการให้ตรง และการส่งข้อมูลให้ครบตั้งแต่ต้น เพื่อให้ทีมงานคุยรายละเอียดได้ตรงกับสภาพเครื่องจริง amphon.co.th มีหน้าร้านจริงที่จังหวัดอุบลราชธานี ใช้เป็นหลักฐานให้ลูกค้าตรวจสอบตัวตนและความน่าเชื่อถือได้ แต่ลูกค้าทั่วประเทศสามารถเริ่มจากการส่งรูปและข้อมูลเพื่อประเมินเบื้องต้นได้เหมือนกัน

## รับซื้อกรณีไหนบ้าง

กรณีที่เหมาะกับหน้านี้ ได้แก่

- โน๊ตบุ๊คพนักงานเก่าหลังเปลี่ยนรอบอุปกรณ์
- เครื่องจากฝ่าย IT ที่รอจำหน่ายออกจากทะเบียน
- เครื่องผู้บริหาร ฝ่ายขาย บัญชี หรือทีมปฏิบัติงาน
- เครื่องที่มี Asset Tag, Serial Number, Docking หรือ Adapter แยกเป็นรายการ
- เครื่องบริษัทปลดระวางที่มีหลายแผนกหรือหลายสาขา

นอกจากรายการข้างต้น หากเครื่องมีหลายอาการร่วมกัน เช่น ไม่มี Adapter ร่วมกับเปิดไม่ติด หรือจอแตกพร้อมบานพับหัก ก็ยังส่งข้อมูลให้ประเมินเบื้องต้นได้ โดยควรแจ้งอาการทั้งหมดที่ทราบ ไม่ต้องซ่อมก่อนและไม่ต้องแกะเครื่องเอง

## เหมาะกับลูกค้ากลุ่มไหน

หน้านี้เหมาะกับฝ่าย IT ฝ่ายจัดซื้อ ฝ่ายบัญชี เจ้าของกิจการ ผู้ดูแลทรัพย์สิน และบริษัทที่ต้องการจัดการเครื่องพนักงานเก่าอย่างเป็นระบบ รวมถึงลูกค้าที่ต้องการเปรียบเทียบราคาก่อนได้ก่อนตัดสินใจว่าจะขาย ซ่อม หรือเก็บเครื่องไว้ใช้งานต่อ

กรณีเป็นเครื่องส่วนตัว เครื่องบริษัท เครื่องจากสำนักงาน เครื่องที่เก็บไว้นาน หรือเครื่องที่รวบรวมมาเป็นล็อต สามารถส่งข้อมูลให้ทีมงานช่วยประเมินเบื้องต้นได้ จุดสำคัญคือข้อมูลต้องชัดเจนพอ เช่น รุ่น สเปก อาการ จำนวนเครื่อง และพื้นที่ตั้งสินค้า

## ต้องส่งข้อมูลอะไรเพื่อประเมินราคา

เพื่อให้ประเมินเบื้องต้นได้ไว แนะนำให้ส่งข้อมูลต่อไปนี้ทาง LINE: **@webuy**

- รูปด้านหน้าเครื่อง
- รูปด้านหลังเครื่อง
- รูปสเปกหรือหน้า About / System Information ถ้าเปิดได้
- ยี่ห้อ รุ่น และซีรีส์
- CPU / RAM / SSD ถ้าทราบ
- อาการเสียหรือตำหนิ
- อุปกรณ์ที่มี เช่น Adapter, กล่อง, กระเป๋า
- จำนวนเครื่อง
- พื้นที่ตั้งสินค้า
- กรณีบริษัทให้ส่ง Inventory List ได้

ถ้าเครื่องเปิดไม่ได้หรือไม่สามารถเข้าหน้า System Information ได้ ให้ถ่ายรูปป้ายรุ่นใต้เครื่อง สติ๊กเกอร์ Serial Number หรือรูปทรงเครื่องหลายมุมแทน หากมีเอกสารสเปกเดิม ใบเสนอราคาเดิม หรือรายการทรัพย์สินเดิม สามารถส่งประกอบได้

## วิธีถ่ายรูปเครื่องให้ประเมินได้ไว

ถ่ายรูปในที่สว่าง เห็นรายละเอียดชัด ไม่เบลอ และไม่ควรถ่ายเฉพาะมุมสวย เพราะการประเมินต้องดูสภาพจริง ควรถ่ายฝาเครื่อง ด้านใน ด้านหลัง ป้ายรุ่น พอร์ตสำคัญ และอุปกรณ์ที่มี เช่น Adapter กล่อง หรือกระเป๋า

ถ้ามี Asset Tag หรือ Serial Number ให้ถ่ายเพิ่มทุกเครื่องที่เป็นตัวอย่าง หากมี Docking หรือ Adapter แยกชุด ให้ถ่ายรวมกับเครื่องหรือแยกตามหมายเลขรายการ เพื่อให้เทียบกับ Inventory List ได้ง่าย หากมีหลายเครื่อง ให้ถ่ายภาพรวมก่อนหนึ่งชุด แล้วเลือกเครื่องตัวอย่างของแต่ละกลุ่มมาถ่ายละเอียดเพิ่ม เช่น กลุ่มเปิดติด กลุ่มมีตำหนิ กลุ่มเปิดไม่ติด และกลุ่มไม่มี Adapter วิธีนี้ช่วยให้ประเมินเบื้องต้นได้เป็นระบบมากกว่าการส่งรูปปะปนกันโดยไม่มีคำอธิบาย

## ราคาประเมินขึ้นกับอะไร

ราคาขึ้นกับรุ่น สเปก สภาพ อุปกรณ์ จำนวนเครื่อง อาการที่พบ และความชัดเจนของข้อมูลที่ส่งมา เครื่องที่มี CPU รุ่นใหม่ RAM เพียงพอ SSD ใช้งานได้ สภาพบอดี้ดี และมี Adapter ครบ จะประเมินต่างจากเครื่องที่มีตำหนิหลายจุดหรือทดสอบไม่ได้

สำหรับเครื่ององค์กร ความสม่ำเสมอของล็อตและเอกสารรายการเครื่องมีผลต่อความเร็วในการประเมิน รุ่นเดียวกันหลายเครื่องมักจัดกลุ่มได้ง่ายกว่าเครื่องหลายรุ่นปนกัน แต่ล็อตปนกันก็ส่งพิจารณาได้หากข้อมูลชัดเจน การประเมินจากรูปเป็นการประเมินเบื้องต้นเท่านั้น หากต้องตรวจเครื่องจริง ราคาหรือเงื่อนไขอาจเปลี่ยนตามสภาพที่พบจริง จึงควรแจ้งข้อมูลตรงไปตรงมาเพื่อให้การประเมินใกล้เคียงที่สุด

## เครื่องบริษัท / เครื่องยกล็อต / Inventory List ถ้าเกี่ยวข้อง

หน้านี้ควรใช้ Inventory List เป็นแกนหลัก เพราะลูกค้าบริษัทมักมีผู้เกี่ยวข้องหลายฝ่ายและต้องการรายการชัดเจนก่อนตัดสินใจ ถ้าเป็นบริษัทหรือมีหลายเครื่อง แนะนำส่งรายการในรูปแบบตาราง เช่น ลำดับ, ยี่ห้อ, รุ่น, CPU, RAM, SSD, อาการ, Adapter, จำนวน และหมายเหตุ

Inventory List ไม่จำเป็นต้องซับซ้อนมาก แต่ควรแยกข้อมูลให้ชัด หากมี Asset Tag หรือหมายเลขทรัพย์สิน ให้ใส่เพิ่มได้ โดยเฉพาะงานบริษัท โรงเรียน หน่วยงาน หรือสำนักงานที่ต้องการตรวจเทียบรายการก่อนส่งมอบเครื่อง หากเน้นปริมาณหลายเครื่องโดยไม่จำกัดบริบทองค์กร ดูหน้า [รับซื้อโน๊ตบุ๊คยกล็อต](/บริการ/รับซื้อโน๊ตบุ๊คยกล็อต)

## อาการเสียหรือตำหนิต้องแจ้งอะไรบ้าง

แจ้งสถานะรายเครื่อง เช่น เปิดติด เปิดไม่ติด แบตเสื่อม ไม่มี Adapter จอมีปัญหา หรือถูกถอด SSD/RAM แล้ว เพื่อให้ประเมินตามสภาพจริง อาการที่ควรแจ้ง เช่น เปิดติดหรือไม่ ชาร์จเข้าไหม จอแสดงผลปกติไหม จอแตกหรือจอเป็นเส้นหรือไม่ แบตเตอรี่เก็บไฟไหม คีย์บอร์ดกดครบไหม บานพับแน่นไหม เครื่องเคยตก เคยโดนน้ำ หรือเคยซ่อมมาก่อนหรือไม่ หากมีการถอด SSD, RAM หรือแบตเตอรี่ ควรแจ้งด้วย

## การนัดตรวจ จัดส่ง หรือประเมินตามเงื่อนไข

การนัดตรวจ การจัดส่ง หรือการรับสินค้า ต้องพิจารณาตามจำนวนสินค้า ประเภทสินค้า อาการ และพื้นที่ตั้งสินค้า กรณีเครื่องจำนวนน้อย อาจเริ่มจากส่งรูปและข้อมูลเพื่อประเมินเบื้องต้นก่อน แล้วค่อยคุยขั้นตอนต่อไป

กรณีมีหลายเครื่อง มีเครื่องเสียหลายอาการ หรือมีสินค้าไอทีประเภทอื่นรวมอยู่ด้วย ควรส่งภาพรวมและรายการเครื่องก่อน เพื่อให้ทีมงานพิจารณาแนวทางที่เหมาะสม ไม่ควรส่งสินค้าโดยยังไม่ได้คุยรายละเอียด โดยเฉพาะเครื่องที่เปิดไม่ติด แบตบวม หรือเสียหายหนัก
