UPS ขนาดเล็กถึงกลางที่ใช้กับคอมตั้งโต๊ะและอุปกรณ์สำนักงาน ขนาด 650VA ถึง 2000VA รับพิจารณาตามรุ่นและสภาพ แบรนด์ที่พบบ่อยคือ APC Back-UPS, Syndome, Cleanline และ CyberPower

UPS ขนาดใหญ่ที่ใช้ในห้อง Server หรือห้อง IT ขนาด 2000VA ขึ้นไป รับพิจารณาตามรุ่นและสภาพ APC Smart-UPS, Eaton 5P/9PX และ Schneider Electric ระดับ SME มีตลาดที่ชัดเจน

### Rackmount UPS สำหรับตู้ Rack

UPS แบบติดตั้งใน Rack Cabinet ที่มักใช้ในระบบ Data Center และห้อง Server ขนาดใหญ่ รับพิจารณาตามรุ่นและ VA Rating

## ข้อมูลที่ต้องส่งเพื่อประเมินราคา UPS

UPS มีปัจจัยราคาที่เชื่อมโยงกับสภาพแบตเตอรี่โดยตรง ข้อมูลที่ครบช่วยให้ประเมินได้ตรงกับมูลค่าจริง

**รูปหน้าเครื่อง** — ถ่ายให้เห็น Display Panel, LED Status และ Model Name ที่หน้าตัวเครื่อง

**รูปด้านหลัง** — ถ่ายให้เห็น Output Socket, Input, Communication Port (USB/Serial) และ Battery Compartment ถ้าเห็นได้

**รูป Label รุ่นและ Serial** — สติกเกอร์ด้านหลังหรือด้านล่างเครื่อง มักมีรุ่น VA/Watt Rating และ Serial Number ครบ

**ยี่ห้อและรุ่น** — เช่น APC Smart-UPS SUA1500, Eaton 5P1550, CyberPower PR2200ELCD

**VA และ Watt ถ้าทราบ** — VA Rating บอกขนาด UPS, Watt Rating บอก Real Power ที่รองรับได้

**อายุการใช้งานโดยประมาณ** — UPS ที่ใช้มา 2–3 ปีมีแบตดีกว่า UPS ที่ใช้มา 5+ ปีอย่างชัดเจน

**สภาพแบตเตอรี่** — แบตยังใช้งานได้ปกติ / Beep เตือนบ่อย / Runtime สั้นลงมาก / ไม่ทราบสภาพ

**จำนวนเครื่อง** — ถ้ามีหลายเครื่อง แจ้งว่ารุ่นเดียวกันหรือต่างรุ่น

**สภาพโดยรวม** — เปิดติดปกติ, มีอาการ, เปิดไม่ติด หรือไม่ได้ทดสอบ

**พื้นที่ตั้งสินค้า** — จังหวัดหรือพื้นที่คร่าวๆ เพราะ UPS ขนาดใหญ่ต้องวางแผนการขนย้าย

**ถ้า UPS ขนาดใหญ่ทำให้ถ่ายยาก:**

ถ่ายภาพรวมก่อนแล้วถ่ายเฉพาะ Label จุดสำคัญ ไม่ต้องย้ายตัวเครื่องก่อนถ่ายรูป

## ราคา UPS มือสองขึ้นกับอะไร

### สภาพแบตเตอรี่มีผลมากที่สุด

UPS ที่แบตยังดีและ Runtime ตามสเปกราคาสูงกว่า UPS ที่แบตเสื่อมอย่างชัดเจน ต่างได้ถึง 30–50% ในรุ่นเดียวกัน UPS ที่เพิ่งเปลี่ยนแบตใหม่มีมูลค่าใกล้เคียง UPS ที่แบตยังดี

### แบรนด์และซีรีส์

APC Smart-UPS มีราคาดีกว่า APC Back-UPS ใน VA Rating เดียวกัน Eaton และ Schneider ระดับ Enterprise มีราคาสูงกว่า CyberPower ระดับ SME

### VA Rating และ Output Power

UPS ขนาดใหญ่มีราคาสูงกว่าขนาดเล็กตามสัดส่วน UPS ที่มี True Online Double Conversion มีมูลค่าสูงกว่า Line-Interactive และ Offline/Standby ในขนาดเดียวกัน

### อายุการใช้งานและอายุแบต

แบตเตอรี่ Lead-Acid มีอายุการใช้งาน 2–4 ปี UPS ที่อายุเกิน 5 ปีและไม่เคยเปลี่ยนแบตมักมีมูลค่าต่ำกว่าอย่างชัดเจน

### อุปกรณ์ที่มาด้วย

สายไฟ, Communication Cable (USB/Serial), Software License และ Manual ที่มาพร้อมกันช่วยเพิ่มมูลค่าเล็กน้อย แต่ไม่มีก็ยังรับพิจารณา

## UPS แบตเสื่อม เปิดไม่ติด หรือใช้งานมานาน ยังประเมินได้ไหม

**UPS แบตเสื่อมแต่วงจรยังดี**
รับพิจารณาครับ วงจร Inverter, Charger Circuit และ Transfer Switch ที่ยังดีมีมูลค่า ผู้ซื้อบางรายซื้อ UPS ตัวเปล่ามาเปลี่ยนแบตเองเพราะถูกกว่าซื้อใหม่ราคาต่ำกว่า UPS แบตยังดีชัดเจนแต่ยังขายได้

**UPS ที่ใช้งานมาเกิน 5 ปี**
ยังรับพิจารณา แต่ราคาต่ำกว่า UPS รุ่นเดียวกันที่อายุน้อยกว่า แจ้งปีที่ซื้อหรืออายุโดยประมาณมาด้วย

**UPS เปิดไม่ติดหรือมีอาการ**
รับพิจารณาตามรุ่น UPS Enterprise บางรุ่นที่เปิดไม่ติดยังมีมูลค่าชิ้นส่วน ส่งรุ่นและอาการมาให้ดูก่อน ทางร้านจะแจ้งว่ารับพิจารณาได้หรือไม่

## การจัดการแบตเตอรี่และความปลอดภัยก่อนขาย

แบตเตอรี่ใน UPS เป็น Lead-Acid ที่มีน้ำกรดและอาจมีน้ำหนักมาก มีบางเรื่องที่ควรรู้ก่อนขาย

**ไม่ต้องถอดแบตออกก่อนขาย**
ส่งมาพร้อมแบตได้เลย ทางร้านจัดการต่อเอง แต่ถ้าต้องการถอดเองก็ทำได้ แจ้งให้ทราบตอนติดต่อว่าถอดแบตออกแล้ว

**UPS ที่แบตบวมหรือรั่ว**
แบตที่บวมหรือมีของเหลวออกมาอาจมีความเสี่ยง แจ้งมาตอนส่งรายการเพื่อให้ทางร้านพิจารณาก่อนว่ารับได้หรือไม่

**น้ำหนักของ UPS ขนาดใหญ่**
UPS 3000VA ขึ้นไปมักหนักมาก วางแผนการขนย้ายล่วงหน้า โดยเฉพาะถ้าอยู่ชั้นบนหรืออยู่ใน Rack ที่ต้องถอดออกก่อน

**UPS แบตเสื่อมแต่วงจรยังดี** — ตลาดมือสองมีผู้ซื้อที่ต้องการ UPS ตัวเปล่าเพื่อเปลี่ยนแบตเองเพราะถูกกว่าซื้อใหม่ ยังขายได้แต่ราคาต่ำกว่าชัดเจน

**UPS ที่อายุเกิน 5 ปี** — ยังรับพิจารณา ราคาต่ำกว่าตามอายุ แต่ถ้าเป็นแบรนด์ดีและ VA สูง ยังมีมูลค่าในตลาด

**UPS ที่ไม่รู้รุ่น** — ถ่ายรูป Label ด้านหลังหรือด้านล่างส่งมา ทางร้านช่วยระบุรุ่นได้ในหลายกรณี

**UPS ที่ Beep ไม่หยุด** — มักเป็น Low Battery หรือ Overload Alert ส่งอาการมาพร้อมรุ่น ทางร้านช่วยประเมินได้ว่าตัวเครื่องยังดีหรือไม่

## สถานการณ์ที่มักพบในงานรับซื้อ UPS มือสอง

### สำนักงานเปลี่ยน UPS ชุดใหม่พร้อมกัน

บริษัทที่ครบรอบ 3–5 ปีและเปลี่ยน UPS ทั้งชั้นหรือทั้งสำนักงานพร้อมกัน มักมี UPS เก่าที่ยังสภาพดีค้างอยู่ แจ้งจำนวนและรุ่นมาเพื่อประเมินรวมทั้งล็อต

### ห้อง Server ปรับ Infrastructure ใหม่

เมื่อ Server ถูก Decommission หรือย้ายไป Cloud Infrastructure UPS ขนาดใหญ่ที่ใช้ใน Server Room มักถูกทิ้งไว้ รับพิจารณาตามรุ่นและ VA

### UPS ที่มาพร้อมคอมสำนักงานยกล็อต

การขายคอมสำนักงานยกล็อตมักมี UPS มาพร้อมกัน รับพิจารณาประเมินรวมได้ในครั้งเดียว แจ้งรายการรวมทั้งคอมและ UPS มาพร้อมกันเลย

### หน่วยงานที่ต้องจำหน่ายครุภัณฑ์ตามระเบียบ

โรงเรียนหรือหน่วยงานราชการที่มี UPS เป็นรายการครุภัณฑ์ที่ต้องจำหน่ายออกตามขั้นตอน รับพิจารณาและออกเอกสารรับซื้อได้ถ้าแจ้งล่วงหน้า
