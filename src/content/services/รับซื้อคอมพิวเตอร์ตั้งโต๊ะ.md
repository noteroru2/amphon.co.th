### PC แบรนด์ Dell, HP, Lenovo, Acer

Desktop จากแบรนด์หลักที่มักพบในสำนักงานและบ้าน เช่น Dell OptiPlex, HP ProDesk, Lenovo ThinkCentre, Acer Veriton รับพิจารณาตามรุ่น Generation และสเปก

## เหมาะกับลูกค้ากลุ่มไหน

**คนที่ต้องการขายคอมบ้านที่ไม่ได้ใช้แล้ว** ซื้อมาแต่เปลี่ยนไปใช้โน๊ตบุ๊คแทน หรือเครื่องเก่าที่ไม่ได้ใช้มาหลายปี

**สำนักงานหรือบริษัทที่เปลี่ยนชุดคอมใหม่** มีคอมเก่าหลายเครื่องที่ต้องการขายออก รับพิจารณายกล็อตได้

**คนที่ได้รับเครื่องมาแต่ไม่ตรงความต้องการ** รับมา แต่ไม่ตรงกับสเปกที่ต้องการ ต้องการขายออกเพื่อเปลี่ยนรุ่น

**ผู้ดูแลทรัพย์สินที่ต้องการขายคอมพร้อมกัน** เช่น ผู้จัดการมรดก ผู้รับมอบอำนาจ รับพิจารณาทั้งล็อต

## ข้อมูลที่ต้องส่งเพื่อประเมิน

**รูปหน้าเครื่อง** — ถ่ายให้เห็น Port Panel ด้านหน้า สภาพ Case และ LED ถ้ามี

**รูปด้านหลังเครื่อง** — ถ่ายให้เห็น Port Panel ด้านหลัง ช่อง GPU ถ้ามี และ PSU

**รูปภายใน Case ถ้าเปิดได้** — ถ่ายให้เห็น Mainboard, RAM Slot, CPU Cooler และ Storage ที่ใส่อยู่

**สเปกหลัก** — CPU รุ่นและ Generation, RAM ขนาด GB, Storage ประเภทและขนาด (HDD/SSD)

**การ์ดจอถ้ามี** — แจ้งรุ่นการ์ดจอถ้าใส่แยก ถ้าใช้ Integrated แจ้งมาด้วย

**อาการหรือตำหนิ** — แจ้งทุกอาการที่ทราบ ดีกว่าปิดบัง เพราะทางร้านตรวจจริงอยู่ดี

**อุปกรณ์ที่มีมาด้วย** — จอ, คีย์บอร์ด, เมาส์, สายไฟ, กล่องบรรจุภัณฑ์ถ้ายังมี

**จำนวนเครื่อง** — ถ้ามีหลายเครื่องแจ้งจำนวนด้วย

**พื้นที่ตั้งสินค้า** — จังหวัดหรือพื้นที่คร่าวๆ เพื่อวางแผนการรับสินค้า

## วิธีถ่ายรูปและส่งสเปกให้ประเมินไว

ถ่ายภาพรวมของ Case ทั้งด้านหน้าและด้านหลัง จากนั้นถ่ายภายใน Case ให้เห็น Mainboard และ RAM ถ้าสามารถเปิดฝาได้

ถ้าเปิดคอมได้ ถ่ายหน้า System Information หรือ DxDiag ที่แสดง CPU, RAM และการ์ดจอ จะช่วยให้ประเมินได้แม่นขึ้นมาก

ถ้าเครื่องมีตำหนิ ถ่ายรูปใกล้ๆ ให้เห็นตำหนิชัดเจน การแจ้งตรงๆ ทำให้มูลค่าประเมินตรงกับสภาพจริงและลดการปรับราคาตอนตรวจจริง

## ราคาประเมินขึ้นกับอะไร

### CPU Generation มีผลมากที่สุด

Core i5/i7 Gen 8–12 มีมูลค่าสูงกว่า Gen 3–5 อย่างชัดเจน Ryzen 5/7 Gen 4–6 ยังมีตลาดรองรับดี

### ประเภท Storage

เครื่องที่ใส่ SSD มีมูลค่าสูงกว่า HDD ในสเปก CPU เดียวกัน เพราะ SSD ทำให้เครื่องใช้งานได้เร็วขึ้นและผู้ซื้อมือสองยอมจ่ายมากกว่า

### RAM

8GB ขึ้นไปมีมูลค่าดีกว่า 4GB อย่างชัดเจน 16GB ยิ่งดีขึ้นไปอีก

### แบรนด์และ Form Factor

Dell OptiPlex, HP EliteDesk มีมูลค่าดีกว่า PC No-Brand ในสเปก CPU เดียวกัน เพราะ Build Quality และ Service Support ที่ดีกว่า

### การ์ดจอแยก

เครื่องที่มีการ์ดจอแยกมีมูลค่าสูงกว่าเครื่องที่ใช้ Integrated Graphics ในสเปก CPU เดียวกัน โดยเฉพาะการ์ดจอ NVIDIA GTX หรือ RTX

### สภาพและอุปกรณ์ที่มาด้วย

เครื่องที่จอ คีย์บอร์ด เมาส์ และสายไฟครบมีมูลค่าเพิ่มเล็กน้อย แต่ไม่มีก็รับพิจารณาตัวเครื่อง

## เครื่องมีตำหนิหรืออาการเสีย ยังประเมินได้ไหม

**เครื่องเปิดไม่ติดหรือ POST ไม่ผ่าน** — รับพิจารณาตามรุ่นและอาการ ส่งรายละเอียดอาการมาก่อน ทางร้านประเมินตามสภาพจริง

**ไฟเข้าแต่ไม่ขึ้นภาพ** — อาจเป็นปัญหาการ์ดจอ RAM หรือ Cable รับพิจารณาตามอาการ ส่งรายละเอียดมาก่อน

**Case มีรอยขีดข่วนหรือบุบ** — รอยตามการใช้งานมีผลต่อมูลค่าน้อยกว่า Hardware ที่เสีย แจ้งมาด้วยเพื่อประเมินได้ตรง

**HDD เสียหรือ SSD มีปัญหา** — มีผลต่อมูลค่า แต่ไม่ทำให้ขายไม่ได้ แจ้งอาการมาก่อน

**เครื่องถอดชิ้นส่วนออกบางส่วน** — แจ้งว่าถอดอะไรออกบ้าง ทางร้านประเมินจากชิ้นส่วนที่ยังมีอยู่

## สิ่งที่ทราบแล้วว่ามีผลต่อมูลค่าคอมตั้งโต๊ะ

เครื่องที่มี SSD มีมูลค่าดีกว่า HDD อย่างชัดเจน ถ้าเครื่องยังใส่ HDD อยู่ มูลค่าจะต่ำกว่าเครื่องสเปก CPU เดียวกันที่ใส่ SSD ผู้ซื้อมือสองส่วนใหญ่ให้ความสำคัญกับความเร็วในการใช้งานซึ่งขึ้นกับ SSD มาก

RAM 8GB ขึ้นไปเป็นเกณฑ์ที่ผู้ซื้อมือสองส่วนใหญ่ต้องการ เครื่องที่มี RAM 4GB จะมีตลาดแคบกว่าอย่างเห็นได้ชัด

CPU Generation มีผลโดยตรงต่อราคา ทุก 2 Generation ที่ห่างออกไปมักมีผลต่อมูลค่า 20–40% ขึ้นกับแบรนด์และตลาดขณะนั้น
