## ปัจจัยที่มีผลต่อราคาประเมิน

ราคาโน๊ตบุ๊คมือสองขึ้นกับหลายปัจจัยพร้อมกัน ไม่ใช่แค่แบรนด์หรือรุ่นอย่างเดียว

**รุ่นและปีที่ผลิต** — รุ่นใหม่กว่าสเปกใกล้เคียงกันมักมีมูลค่าสูงกว่า เครื่องเก่าหลายปีราคาลดลงตามอายุ

**ซีพียู (CPU)** — Core i5 / i7 / Ryzen 5 / Ryzen 7 มักมีผลต่อราคาประเมินสูงกว่า Core i3 หรือ Celeron ในรุ่นที่ใกล้เคียงกัน

**RAM** — 8GB ขึ้นไปเป็นที่ต้องการ 4GB ในปัจจุบันตลาดมือสองความต้องการน้อยลงมาก

**SSD / HDD** — เครื่องที่มี SSD มักมีผลต่อราคาประเมินสูงกว่าเครื่องที่ใช้ HDD ขนาดและความจุ SSD ก็มีผลด้วย

**การ์ดจอ** — เครื่องที่มีการ์ดจอแยก (Dedicated GPU) มักมีผลต่อราคาประเมินสูงกว่าการ์ดจอในตัว (Integrated) โดยเฉพาะในกลุ่มเกมมิ่ง

**ขนาดจอ** — จอ 15 นิ้วขึ้นไปมักมีตลาดกว้างกว่า แต่จอ 13–14 นิ้วก็มีผู้ซื้อของตัวเอง

**สภาพภายนอก** — รอยขูดขีด รอยชน บุบ ฝาโก่ง มีผลต่อราคา เครื่องสภาพดีมักได้ราคาประเมินสูงกว่าชัดเจน

**แบตเตอรี่** — แบตที่ยังเก็บไฟได้ดีมีผลต่อราคาประเมินสูงกว่าเครื่องที่แบตเสื่อมมาก

**อุปกรณ์ที่มี** — Adapter / ที่ชาร์จ มีมาพร้อมกันดีกว่า

**ประกัน / กล่อง / ใบเสร็จ** — มีผลบ้าง เครื่องที่ยังมีประกันเหลืออยู่อาจมีผลต่อราคาประเมิน

**ประวัติการซ่อม** — เครื่องที่เคยซ่อมส่วนสำคัญ เช่น เมนบอร์ดหรือจอ มีผลต่อการประเมิน

**ความต้องการในตลาดมือสอง** — บางรุ่นมีผู้ซื้อต่อมาก บางรุ่นหาผู้ซื้อต่อยาก ราคาต่างกันตามสภาวะตลาด
