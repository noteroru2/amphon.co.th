## ร้านประเมินราคาคอมพิวเตอร์จากอะไรบ้าง

ราคาคอมพิวเตอร์มือสองขึ้นกับหลายปัจจัยพร้อมกัน การส่งสเปกมาครบจะช่วยให้ประเมินได้ใกล้เคียงกับราคาจริงมากขึ้น

**CPU** — รุ่นและ Generation ของซีพียูมีผลต่อราคาโดยตรง Core i5 / i7 / Ryzen 5 / Ryzen 7 มักประเมินได้สูงกว่ากลุ่ม Core i3 หรือ Celeron

**GPU / การ์ดจอ** — ชิ้นส่วนที่มีผลต่อราคาคอมมากที่สุดในกลุ่ม Gaming PC และ Workstation รุ่นและสภาพของการ์ดจอมีน้ำหนักในการประเมินสูง

**RAM** — ขนาดและ Generation ของ RAM มีผล 8GB ขึ้นไปเป็นที่ต้องการ 4GB ความต้องการตลาดน้อยลงมาก

**SSD / HDD** — เครื่องที่มี SSD มักประเมินได้สูงกว่าเครื่องที่ใช้ HDD ขนาดและความเร็ว SSD ก็มีผลด้วย

**Mainboard** — ชิป Socket และ Feature ของเมนบอร์ดมีผลต่อการประเมิน

**Power Supply** — Watt และแบรนด์ของ PSU มีผลบ้าง โดยเฉพาะในเครื่องที่ใช้ GPU แรง

**Case และชุดระบายความร้อน** — ขนาดและสภาพ Case รวมถึง Cooler ที่ติดมา

**Monitor / จอคอม** — ถ้ามีจอมาด้วยจะรับพิจารณาพร้อมกัน

**สภาพภายนอกและฝุ่น** — เครื่องสะอาด สภาพดี มักประเมินได้ง่ายกว่าเครื่องที่ฝุ่นหนาหรือเคสเสียหาย

**อุปกรณ์ครบไหม** — มีสายไฟ คีย์บอร์ด เมาส์ มาด้วยหรือเปล่า

**ประกัน / กล่อง / ใบเสร็จ** — มีผลบ้าง เครื่องหรือชิ้นส่วนที่ยังมีประกันมักช่วยให้ประเมินได้ง่ายขึ้น

**ความต้องการตลาดมือสอง** — บางรุ่นมีผู้ซื้อต่อมาก บางรุ่นหาผู้ซื้อต่อยาก ราคาต่างกันตามสภาวะตลาด
