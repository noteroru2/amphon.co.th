## รับซื้ออะไรบ้างเมื่อบริษัทปิดกิจการ

ออฟฟิศที่ปิดกิจการมักมีสินค้าหลายประเภทปะปนกัน รับพิจารณาได้ทั้งหมดในการประเมินเดียว ไม่ต้องแยกขายทีละประเภท

### คอมพิวเตอร์และโน๊ตบุ๊ค

Desktop และ Notebook ของพนักงานทุกแบรนด์ ทุกสเปก รับพิจารณาทั้งเครื่องที่ยังใช้งานได้ปกติและเครื่องที่มีอาการ เครื่องสเปกเก่าก็ยังมีมูลค่าในตลาดมือสอง ไม่ต้องทิ้งก่อนถามราคา

### จอมอนิเตอร์

จอมอนิเตอร์ทุกขนาดทุกแบรนด์ รับพิจารณาพร้อมกับชุดคอม ไม่จำเป็นต้องแยกขาย ถ้ามีหลายสิบจอรับพิจารณาเป็นล็อต

### UPS เครื่องสำรองไฟ

UPS ทุกขนาดและทุกแบรนด์ รับพิจารณาตามสภาพ แบตเตอรี่เสื่อมบางส่วนมีผลต่อราคา แต่ยังรับพิจารณาตามสภาพจริง

### Printer และอุปกรณ์พิมพ์

Printer ทั้งแบบ Inkjet และ Laser รับพิจารณาตามรุ่นและสภาพ แจ้งยี่ห้อและรุ่นมาพร้อมรายการอื่น

### Server และอุปกรณ์ Network

Server, Switch, Router, Firewall และอุปกรณ์ Network ที่ใช้ในออฟฟิศ รับพิจารณาตามสเปกและสภาพ แจ้งยี่ห้อและรุ่นมาในรายการ

### อุปกรณ์ไอทีเบ็ดเตล็ด

คีย์บอร์ด เมาส์ Hub Switch เล็ก และอุปกรณ์เสริมอื่น รับพิจารณารวมในล็อตได้ แม้จะมีมูลค่าต่อชิ้นไม่มาก แต่รวมกันแล้วยังมีมูลค่า

## กรณีไหนเหมาะกับบริการนี้

บริการนี้ออกแบบมาสำหรับสถานการณ์ที่ต้องเคลียร์ทรัพย์สินไอทีออกจากสถานที่ ไม่ใช่การขายรายเครื่องตามปกติ

**บริษัทที่กำลังปิดกิจการหรือเลิกกิจการ**
ต้องเคลียร์ทุกอย่างออกก่อนวันสิ้นสุดสัญญาเช่า หรือก่อนส่งมอบสถานที่คืน ยิ่งมีเส้นตายชัดเจนยิ่งควรติดต่อเร็ว เพราะการประเมินและนัดตรวจต้องใช้เวลา

**บริษัทที่ปิดสาขาหรือรวมสำนักงาน**
สาขาที่ปิดมักมีอุปกรณ์ครบชุดที่ไม่ได้ย้ายไปด้วย รับพิจารณาทั้งชุดหรือเฉพาะส่วนที่ไม่ต้องการ

**ร้านค้าหรือกิจการที่เลิกดำเนินการ**
ร้านที่มีคอม จอ และอุปกรณ์ไอทีที่ใช้ในการดำเนินงาน รับพิจารณาทั้งหมดหรือบางส่วน

**สำนักงานที่ย้ายที่ตั้งและไม่ต้องการขนอุปกรณ์ทั้งหมด**
บางครั้งการขนย้ายอุปกรณ์ไอทีเก่าไปที่ใหม่ไม่คุ้มกับค่าใช้จ่าย การขายออกก่อนย้ายเป็นทางเลือกที่สะดวกกว่า

**โรงเรียนหรือหน่วยงานที่เคลียร์ทรัพย์สินเก่า**
ห้องแล็บหรือห้องปฏิบัติการที่ต้องการเคลียร์อุปกรณ์เก่าออกก่อนจัดซื้อใหม่ รับพิจารณาเป็นล็อต

**เจ้าของที่ดูแลทรัพย์สินแทน**
ทนายความ ผู้จัดการมรดก หรือผู้รับมอบอำนาจที่ต้องจัดการทรัพย์สินไอทีแทนเจ้าของ รับพิจารณาได้เช่นกัน

## ต้องส่งข้อมูลอะไรเพื่อประเมินราคา

สถานการณ์ปิดกิจการมักมีสินค้าหลายประเภทปะปนกัน ข้อมูลที่ส่งมาครบยิ่งช่วยให้ประเมินได้ตรงและเร็วขึ้น

**จำนวนเครื่องและรายการสินค้าแยกประเภท**
แจ้งว่ามีคอมกี่เครื่อง จอกี่จอ UPS กี่ตัว Printer กี่เครื่อง ถ้ามี Inventory List หรือ Asset Register อยู่แล้วส่งมาได้เลย

**รูปถ่ายภาพรวมของห้องหรือกองสินค้า**
รูปที่มองเห็นทั้งห้องหรือทั้งพื้นที่ช่วยให้เห็นภาพรวมปริมาณสินค้าโดยรวม ถ่ายหลายมุมถ้าสินค้ากระจายอยู่หลายจุด

**รูปถ่ายแยกตามประเภทสินค้า**
ถ่ายแยกกันระหว่างคอม จอ UPS Printer และ Server เพื่อให้ประเมินได้ถูกต้องตามประเภท

**สภาพโดยรวมที่ทราบ**
บอกคร่าวๆ ว่าสินค้าส่วนใหญ่อยู่ในสภาพอย่างไร เช่น "ส่วนใหญ่ใช้งานได้ ไม่แน่ใจบางเครื่อง" หรือ "เก็บไว้หลายปีไม่แน่ใจสภาพ" ข้อมูลแบบนี้ช่วยให้ประเมินเบื้องต้นได้ใกล้เคียงกว่าการไม่บอกอะไรเลย

**พื้นที่ตั้งของสินค้า**
ระบุจังหวัดหรือที่อยู่คร่าวๆ เพื่อใช้วางแผนการรับของในขั้นตอนถัดไป

**ต้องการขายทั้งหมดหรือบางส่วน**
ถ้าต้องการขายเฉพาะบางประเภท เช่น ขายแค่คอมแต่ไม่ขายจอ แจ้งมาให้ชัดเจนตั้งแต่ต้น

**เงื่อนไขวันเคลียร์พื้นที่ถ้ามี**
ถ้ามีเส้นตายที่ต้องเคลียร์พื้นที่ภายในวันที่กำหนด แจ้งมาตอนติดต่อครั้งแรก เพื่อให้วางแผนได้ตรงกับสถานการณ์จริง

นอกจากราคาต่อรายการ สิ่งที่มีผลต่อราคารวมทั้งล็อตคือ สัดส่วนสินค้าที่ใช้งานได้ต่อสินค้าที่มีปัญหา ความหลากหลายของประเภทสินค้าในล็อต และเงื่อนไขด้านโลจิสติกส์ที่อาจมีต้นทุนเพิ่มขึ้นตามพื้นที่และจำนวน

## เรื่องข้อมูลในเครื่อง เอกสาร และทรัพย์สินบริษัท

เมื่อบริษัทปิดกิจการมีเรื่องที่ต้องจัดการนอกเหนือจากราคา

### ข้อมูลในเครื่อง

ควรจัดการเรื่องข้อมูลก่อนส่งมอบเครื่อง โดยเฉพาะถ้าเครื่องมีข้อมูลลูกค้า ข้อมูลบัญชี หรือข้อมูลความลับทางธุรกิจ วิธีที่ทำได้คือ Format ไดร์ฟหรือ Reset เครื่อง ไม่จำเป็นต้องทำก่อนส่งรูปหรือรายการมาประเมิน แต่ต้องเสร็จก่อนส่งมอบเครื่องจริง ถ้าไม่แน่ใจขั้นตอนสอบถามทางร้านได้ก่อนนัด

### เอกสารสำหรับทำบัญชีบริษัท

ถ้าบริษัทต้องการใบรับซื้อหรือเอกสารยืนยันการจำหน่ายทรัพย์สินเพื่อทำบัญชีตัดจำหน่าย แจ้งล่วงหน้าตอนติดต่อครั้งแรก เพื่อเตรียมเอกสารให้ถูกต้องตามที่ต้องการ

### ทรัพย์สินที่ไม่ต้องการขาย

ถ้ามีสินค้าบางส่วนที่ยังต้องการเก็บไว้หรือย้ายไปสถานที่อื่น แจ้งมาตั้งแต่ต้นว่าต้องการขายเฉพาะส่วนไหน ทางร้านประเมินเฉพาะส่วนที่ต้องการขายได้ ไม่บังคับให้ขายทั้งหมด

## การนัดตรวจ ขนย้าย หรือจัดส่งตามเงื่อนไข

วิธีรับของขึ้นกับจำนวนสินค้า ประเภทสินค้า พื้นที่ตั้ง และเงื่อนไขของแต่ละกรณี ไม่มีรูปแบบตายตัว

### สินค้าปริมาณน้อย นำมาที่ร้านได้

ถ้ามีสินค้าไม่มากและอยู่ในพื้นที่ที่ขนสะดวก นำมาที่ร้านที่อุบลราชธานีได้โดยตรง ไม่ต้องรอนัด

### สินค้าปริมาณมาก นัดตรวจเช็กที่สถานที่

ถ้าสินค้ามีจำนวนมากหรือมีอุปกรณ์หนักที่ขนลำบาก เช่น Server ราคสูง UPS หลายตัว หรือคอมหลายสิบเครื่อง สามารถนัดให้ทางร้านไปดูและตรวจเช็กที่สถานที่ได้ โดยเฉพาะในพื้นที่ที่เดินทางถึงได้สะดวก

### สินค้าอยู่ต่างจังหวัด

ส่งรายการและรูปถ่ายมาประเมินเบื้องต้นได้ก่อนเสมอ โดยไม่ต้องเคลื่อนย้ายของ จากนั้นหารือวิธีการรับของที่เหมาะกับพื้นที่และปริมาณสินค้า ซึ่งอาจรวมถึงการส่งสินค้าหรือการนัดรับที่จุดนัดหมาย ขึ้นกับแต่ละกรณี

### กรณีมีเส้นตายเคลียร์พื้นที่

แจ้งวันที่ต้องการเคลียร์มาตั้งแต่ต้น ทางร้านจะพยายามวางแผนให้ตรงกับเส้นตายนั้น การแจ้งล่วงหน้าเร็วช่วยให้วางแผนได้ดีกว่าการแจ้งกระชั้นชิด

## รับซื้อคอมบริษัทปิดกิจการจากลูกค้าทั่วประเทศ

อำพล เทรดดิ้ง มีหน้าร้านจริงที่อุบลราชธานี เป็นหลักฐานว่ามีตัวตนที่ตรวจสอบได้ ไม่ใช่แค่ LINE หรือเบอร์โทรที่ไม่รู้ว่าจะติดต่อใครได้จริง

การประเมินราคาเบื้องต้นทำได้จากรายการหรือรูปถ่าย ไม่ต้องนำสินค้ามาที่ร้านก่อน ลูกค้าจากทุกจังหวัดส่งข้อมูลมาทาง LINE ได้เลย จากนั้นหารือขั้นตอนการรับของตามสถานการณ์จริงของแต่ละเคส

สิ่งที่ทำให้บริการนี้เหมาะกับกรณีปิดกิจการโดยเฉพาะคือ รับสินค้าหลายประเภทในคราวเดียว ไม่ต้องหาผู้ซื้อแยกหลายราย และจ่ายเงินทันทีเมื่อตกลงราคา ซึ่งช่วยให้ปิดเรื่องทรัพย์สินไอทีได้ในการดำเนินการเดียว
