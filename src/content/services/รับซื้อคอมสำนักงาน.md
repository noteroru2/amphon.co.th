## รับซื้อคอมสำนักงานและอุปกรณ์ออฟฟิศแบบไหนบ้าง

### คอมตั้งโต๊ะสำนักงานทุกแบรนด์

รับพิจารณาคอมตั้งโต๊ะสายสำนักงานจากทุกแบรนด์หลัก ทั้ง Dell, HP, Lenovo, Acer และ Asus ตั้งแต่รุ่น Core i3 ขึ้นไป รับทั้งเครื่องที่ยังใช้งานได้ปกติและเครื่องที่มีปัญหาบางส่วน สภาพภายนอกมีรอยตามการใช้งานไม่ใช่ปัญหา เพราะประเมินตามสภาพจริงทุกเครื่อง

### โน๊ตบุ๊คบริษัทสายธุรกิจ

โน๊ตบุ๊คสายธุรกิจอย่าง Dell Latitude, HP EliteBook, Lenovo ThinkPad หรือ HP ProBook มักมีมูลค่าในตลาดมือสองที่ดี รับพิจารณาทั้งเครื่องสภาพดีและเครื่องที่มีรอยใช้งานตามปกติ ขอแค่ระบุสภาพมาให้ชัดเจนเพื่อประเมินราคาได้ตรง

### คอมสเปกต่ำหรือเครื่องเก่า

คอม Core i3 รุ่นเก่า หรือเครื่องที่ใช้งานมาหลายปีก็ยังรับพิจารณา ราคาขึ้นกับสภาพและรุ่นปี ถ้าไม่แน่ใจว่าเครื่องที่มีจะขายได้ไหม ส่งรูปพร้อมสเปกมาให้ดูก่อนได้เลย ไม่เสียอะไร

### อุปกรณ์สำนักงานเสริม

นอกจากคอมและโน๊ตบุ๊ค รับพิจารณาอุปกรณ์ที่มาพร้อมกันได้ด้วย เช่น จอมอนิเตอร์ คีย์บอร์ด เมาส์ และ UPS เครื่องสำรองไฟ สำหรับ Printer และ Server สามารถสอบถามทางร้านได้โดยตรง

### รับซื้อยกล็อตจากองค์กรและหน่วยงาน

บริษัทที่เปลี่ยนชุดคอมใหม่ทั้งสำนักงาน โรงเรียน โรงพยาบาล หรือหน่วยงานราชการที่ต้องการจำหน่ายครุภัณฑ์เก่าออก สามารถติดต่อรับซื้อยกล็อตได้เลย ไม่มีจำนวนขั้นต่ำตายตัว รับพิจารณาตั้งแต่ 5 เครื่องขึ้นไป ขึ้นกับสภาพและประเภทสินค้า

## ส่งข้อมูลอะไรมาให้ประเมินราคาได้เร็วขึ้น

### ข้อมูลที่ควรส่งมาพร้อมกัน

ยิ่งมีข้อมูลครบ ยิ่งได้ราคาประเมินที่แม่นยำขึ้น โดยทั่วไปควรแจ้ง:

- รูปถ่ายตัวเครื่อง ด้านหน้าและด้านหลัง รวมถึง label สเปกที่อยู่ใต้เครื่องหรือด้านข้าง
- แบรนด์และรุ่น เช่น Dell OptiPlex 7060, HP EliteDesk 800 G4
- CPU, RAM และประเภท Storage (HDD หรือ SSD)
- สภาพหรืออาการที่ทราบ เช่น เปิดติดปกติ จอมีรอย แบตเสื่อม
- จำนวนเครื่องทั้งหมด
- อุปกรณ์ที่มีมาด้วย เช่น สายไฟ กล่อง คู่มือ
- พื้นที่ของลูกค้า ว่าอยู่ที่ไหน

### วิธีถ่ายรูปให้ประเมินได้ไว

ไม่ต้องถ่ายสวย แค่ถ่ายให้เห็นชัด โดยเฉพาะ:

- Label สเปกที่ด้านล่างหรือด้านหลังเครื่อง (มักมีรุ่น, SN, สเปกครบ)
- หน้าจอขณะเปิดเครื่อง หรือหน้า System Information ถ้าเข้าได้
- ตำหนิหรือรอยที่มีตามจริง เพื่อให้ประเมินได้ตรงกับสภาพจริง

## ราคารับซื้อคอมสำนักงานขึ้นกับอะไรบ้าง

### ปัจจัยที่กำหนดราคา

ราคารับซื้อไม่มีสูตรตายตัว เพราะแต่ละเครื่องต่างกัน สิ่งที่มีผลมากที่สุดคือ:

- **แบรนด์และรุ่น** — แบรนด์สายธุรกิจอย่าง ThinkPad หรือ EliteBook มักมีราคาดีกว่ารุ่น Home ทั่วไป
- **ปีที่ผลิตและ Generation ของ CPU** — ยิ่งใหม่ยิ่งดี แต่รุ่นเก่าที่สภาพดีก็ยังมีราคา
- **สเปก** — RAM มากกว่าและมี SSD ราคาดีกว่า HDD
- **สภาพตัวเครื่อง** — รอย บวม จอมีปัญหา แบตเตอรี่เสื่อม มีผลต่อราคา
- **อุปกรณ์ที่มาด้วย** — ครบชุดราคาดีกว่า แต่ไม่ครบก็รับพิจารณา
- **จำนวนเครื่อง** — ยกล็อตมีความยืดหยุ่นในการเจรจาต่างจากรายเครื่อง
- **ความต้องการของตลาดในช่วงนั้น** — บางรุ่นมีความต้องการสูงกว่ารุ่นอื่น

### เครื่องเก่าหรือสเปกต่ำยังมีราคาไหม

มีครับ แม้เครื่องที่ CPU เก่าหรือสเปกไม่สูง ก็ยังมีมูลค่าในตลาด refurbish และสำหรับผู้ใช้งานทั่วไปที่ต้องการคอมราคาประหยัด อย่าเพิ่งทิ้ง ส่งรูปมาให้ดูก่อน จะได้รู้ว่าพอขายได้เท่าไหร่

## ก่อนขายคอมสำนักงาน ควรเตรียมอะไรบ้าง

หลายคนไม่แน่ใจว่าต้องทำอะไรก่อนขายคอมสำนักงาน โดยเฉพาะถ้าเป็นเครื่องขององค์กรที่มีข้อมูลสำคัญอยู่

**เช็กสเปกเครื่องก่อนส่งรูป**
เปิด System Information ใน Windows (กด Windows + R แล้วพิมพ์ `msinfo32`) จะเห็น CPU, RAM และข้อมูลเครื่องครบ หรือถ่าย label ที่ด้านล่างเครื่องมาก็ได้

**เรื่องข้อมูลในเครื่อง**
แนะนำให้ backup ไฟล์ที่ต้องการเก็บไว้ก่อน จากนั้นทำ Format หรือ Reset เครื่องก่อนส่งมอบ ไม่จำเป็นต้องทำก่อนส่งรูปประเมิน แต่ต้องเคลียร์ให้เรียบร้อยก่อนส่งมอบจริง

**เครื่องไม่มีอุปกรณ์ครบ**
ขายได้ครับ ของที่มาพร้อมเครื่องช่วยให้ราคาดีกว่า แต่ถ้าไม่มีกล่องหรือสายไฟก็ไม่ใช่ปัญหาหลัก

**ยกล็อตหรือขายรายเครื่อง**
ถ้ามีหลายเครื่อง การขายยกล็อตมักสะดวกกว่า เพราะจบในครั้งเดียว และบางกรณีอาจได้ราคารวมที่ดีกว่าขายทีละเครื่อง ขึ้นกับสภาพและรุ่นของเครื่อง
