## สิ่งที่มีผลต่อมูลค่า Desktop PC ในตลาดมือสองไทย

ผู้ซื้อ Desktop PC มือสองในไทยมักพิจารณาจาก CPU Generation ก่อนแล้วค่อยดู Storage Type CPU Gen 8–12 ยังมีความต้องการสูง ส่วน Gen 3–5 มูลค่าต่ำมากเพราะไม่รองรับ OS ใหม่ได้ดี

SSD เป็นปัจจัยที่ผู้ซื้อมือสองให้ความสำคัญมากขึ้นเรื่อยๆ เพราะประสบการณ์การใช้งานต่างกันชัดเจนระหว่าง SSD และ HDD Desktop ที่มี SSD ขายได้เร็วกว่าและมูลค่าดีกว่าอย่างเห็นได้ชัด
