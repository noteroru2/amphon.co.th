## รับซื้อ Server แบบไหนบ้าง

### Server Rack (1U, 2U, 4U)

Server แบบ Rack Mount ที่ติดตั้งในตู้ Rack เป็นประเภทที่พบบ่อยที่สุดในสำนักงานและ Data Center รับพิจารณาทุกขนาดตั้งแต่ 1U ไปจนถึง 4U ทั้งรุ่น Single Socket และ Dual Socket

### Server Tower

Server แบบตั้งพื้นที่มักใช้ใน SME และองค์กรขนาดกลาง รับพิจารณาตามรุ่นและสเปก Tower Server หลายรุ่นยังมีตลาดที่ชัดเจนแม้จะเป็นรุ่นเก่า

### Blade Server และ Chassis

Blade Server จาก Dell (M-Series), HP (C-Class Blade) และ IBM (BladeCenter) รับพิจารณาตามรุ่นและ Configuration รวมถึง Chassis และ Blade Module แยกหรือรวมกัน

### Server ที่ถอดชิ้นส่วนบางส่วนออกแล้ว

Server ที่ถอด Drive หรือ RAM ออกไปบ้างแล้วยังรับพิจารณา แต่ราคาต่างจาก Server ที่ครบชิ้นส่วน ยิ่งแจ้งรายละเอียดว่าถอดอะไรออกไปบ้าง ยิ่งประเมินได้แม่นขึ้น

### Server ปลดระวางและ Server เปิดไม่ติด

Server ที่หมดอายุการใช้งานตามนโยบายองค์กร หรือ Server ที่มีอาการ POST ไม่ผ่าน เปิดไม่ติด หรือมี Component บางส่วนเสีย ยังมีมูลค่าชิ้นส่วนในหลายกรณี รับพิจารณาตามสภาพจริง

### HP ProLiant

ThinkSystem เป็นรุ่นที่ต่อยอดจาก IBM x-Series มีตลาดรองรับในหมู่ผู้ดูแลระบบที่คุ้นเคยกับ Platform นี้ รับพิจารณาตามรุ่นและสเปก ThinkServer รุ่นเก่าก็รับพิจารณาตามสภาพ

### IBM และ Cisco UCS

IBM System x (ก่อน Lenovo เข้าซื้อ) และ Cisco UCS รับพิจารณาตามรุ่นและ License ที่ยังมีอยู่ Cisco UCS บางรุ่นมีมูลค่าสูงในตลาด Enterprise

### Server แบรนด์อื่นและ Server ประกอบ

## ข้อมูลที่ต้องส่งเพื่อประเมินราคา Server

Server มีรายละเอียดที่ซับซ้อนกว่าคอมทั่วไป ข้อมูลที่ครบช่วยให้ประเมินได้ตรงและเร็วขึ้นมาก

**รูปหน้าเครื่อง** — ถ่ายให้เห็น Model Name, Port Panel ด้านหน้า และสภาพโดยรวม

**รูปหลังเครื่อง** — ถ่ายให้เห็น PSU, Port ด้านหลัง, PCI Slot และ Cable ที่ต่ออยู่ถ้ามี

**รูป Service Tag หรือ Serial Number** — สติกเกอร์ด้านข้างหรือด้านหลังเครื่อง ช่วยให้ค้นหาสเปกได้แม่นยำโดยไม่ต้องเปิดเครื่อง

**รุ่นและ Platform** — เช่น Dell PowerEdge R740, HP ProLiant DL380 Gen10, Lenovo ThinkSystem SR650

**CPU** — รุ่น CPU และจำนวน Socket ที่ใส่จริง เช่น Dual Intel Xeon Gold 6130

**RAM** — ขนาดรวม และถ้าทราบว่าเป็น DDR4 ECC RDIMM หรือ LRDIMM จะช่วยให้ประเมินได้แม่นขึ้น

**Storage ที่ติดตั้งจริง** — จำนวน Bay ทั้งหมดและจำนวนที่ใส่ Drive อยู่ ประเภท Drive (SAS, SATA, NVMe) ขนาด GB/TB และรุ่น RAID Card ถ้าทราบ

**PSU** — จำนวน PSU ที่มีและ Watt เช่น 2x 750W Redundant PSU

**จำนวนเครื่อง** — ถ้ามีหลายเครื่อง แจ้งจำนวนรวมและว่ารุ่นเดียวกันหรือต่างรุ่น

**สภาพการใช้งาน** — POST ผ่านหรือไม่ มีอาการอะไรที่ทราบ หรือถ้าไม่ได้ทดสอบก็แจ้งว่าไม่ทราบสภาพ

**Rack Rail และอุปกรณ์เสริม** — มี Rail Kit, สายไฟ Server Grade, Bezel หรือ Management Cable มาด้วยหรือไม่

**พื้นที่ตั้งของสินค้า** — จังหวัดหรือพื้นที่คร่าวๆ เพื่อวางแผนการรับสินค้า

## วิธีถ่ายรูป Server ให้ประเมินได้ไว

Server ถ่ายรูปต่างจากคอมทั่วไปเพราะข้อมูลสำคัญกระจายอยู่หลายจุด

**4 รูปที่ควรถ่ายก่อน:**

ถ่ายด้านหน้าเครื่องให้เห็น Panel และ Drive Bay ทั้งหมด ถ่ายด้านหลังให้เห็น Port และ PSU ถ่าย Service Tag หรือ iDRAC Label ที่ด้านข้างเครื่องให้ชัด และถ่าย Label ของ Drive ที่ใส่อยู่แต่ละตัวถ้าต้องการให้ประเมินได้แม่นขึ้น

**ถ้า Server ยังอยู่ใน Rack:**

ถ่ายภาพรวมของ Rack ก่อน แล้วถ่ายเฉพาะตัว Server แต่ละเครื่องแยกกัน ไม่ต้องถอด Server ออกจาก Rack เพื่อถ่ายรูป

**ถ้ามีหลายเครื่อง:**

ถ่ายภาพรวมให้เห็นจำนวนทั้งหมดก่อน จากนั้นถ่ายตัวแทน 1–2 เครื่องให้เห็น Label รายละเอียด ถ้าทุกเครื่องรุ่นเดียวกันไม่ต้องถ่ายทุกตัว

## ราคา Server ขึ้นกับอะไร

ตลาด Server มือสองมีปัจจัยที่ซับซ้อนกว่าคอมทั่วไป เพราะ Configuration ในเครื่องเดียวกันอาจต่างกันมาก

### Platform และ Generation มีผลสูงสุด

Dell PowerEdge Generation 13 (R730) และ Generation 14 (R740) ยังมีอะไหล่หาได้และ Firmware Support ที่ดี ราคาต่างจาก Generation 11 (R710) อย่างชัดเจน HP ProLiant Gen9 และ Gen10 มีตลาดรองรับดีกว่า Gen7 หรือ Gen8

### CPU Configuration

Dual Socket ที่ใส่ CPU ครบทั้ง 2 ตัวมีราคาดีกว่า Single CPU บน Dual Socket Board CPU Xeon Scalable (Skylake ขึ้นไป) มีราคาดีกว่า E5 v3 หรือ E5 v2 รุ่นเก่า

### RAM และ Drive ที่ใส่จริง

RAM และ Drive คือ Component ที่มีมูลค่าแยกต่างหาก Server ที่มี RAM 256GB และ Drive NVMe ครบมีราคาต่างจาก Server ตัวเปล่าอย่างมีนัยสำคัญ

### ความครบสมบูรณ์ของ Configuration

Redundant PSU ที่ทำงานได้ทั้งคู่ RAID Card ที่ยังมีแบตเตอรี่ดี และ Rail Kit ที่ครบ ช่วยเพิ่มมูลค่าให้ Server ชัดเจน

### สภาพการใช้งานจริง

Server ที่ POST ผ่านและ Boot ได้ปกติมีราคาดีกว่า Server ที่มีอาการ แม้จะเป็นรุ่นเดียวกัน Server ที่ Power Supply ตัวใดตัวหนึ่งเสียแต่ตัวอื่น Redundant อยู่ยังมีมูลค่าที่ดี

## Server ปลดระวาง Server เก่า และ Server เปิดไม่ติด — ประเมินได้ไหม

ได้ครับ แต่วิธีประเมินต่างจาก Server สภาพดีเพราะมูลค่ามาจากคนละที่

**Server ปลดระวางตามนโยบายองค์กร**
Server ที่ยังใช้งานได้แต่ถูก Retire ตาม IT Policy เพราะหมดอายุ Support ตามสัญญา มักสภาพดีและมีมูลค่าในตลาด SME หรือ Dev/Test Environment ประเมินเหมือน Server ทั่วไปตามสเปก

**Server ที่ POST ไม่ผ่านหรือเปิดไม่ติด**
ราคาขึ้นกับอาการและรุ่น Server Dell R730 ที่ PSU เสียตัวหนึ่งมีมูลค่าแตกต่างจาก Server ที่ Motherboard เสียทั้งหมด ส่งอาการมาให้ดูก่อน ทางร้านช่วยประเมินได้ในหลายกรณี

**Server ที่ Drive ถูกถอดออกหมดแล้ว**
ยังมีมูลค่า CPU, RAM, RAID Card และ Chassis แต่ราคาต่ำกว่า Server ที่ครบ ยิ่งแจ้งว่า Component อะไรยังอยู่บ้าง ยิ่งประเมินได้ตรงขึ้น

**Server ที่อายุเกิน 10 ปี หรือ EOL แล้ว**
บางรุ่นยังมีมูลค่าชิ้นส่วนแม้ EOL แล้ว โดยเฉพาะ RAM และ CPU สำหรับผู้ที่ต้องการอัปเกรด Server รุ่นเดียวกัน ส่งรุ่นมาให้ดูก่อน

## เรื่องข้อมูลใน Storage และ Drive ก่อนขาย Server

Server มักมีข้อมูลสำคัญขององค์กรอยู่ใน Drive และ Configuration ก่อนส่งมอบควรดำเนินการเหล่านี้

**ลบข้อมูลจาก Drive ก่อนส่งมอบ**
วิธีที่เหมาะกับ Server คือ Secure Erase ผ่าน RAID Controller หรือใช้ Bootable Tool เช่น DBAN หรือ Blancco ถ้า Organization มี Standard สำหรับ Data Destruction ให้ปฏิบัติตามนั้น

**Reset BIOS และ iDRAC/iLO**
Factory Reset iDRAC (Dell), iLO (HP) หรือ XCC (Lenovo) ก่อนส่งมอบ เพื่อลบ User Account, Password และ Configuration ที่บันทึกไว้

**ถอด License และ Software**
Microsoft Windows Server License ที่เป็น Volume License หรือ OEM ควร Deactivate ก่อน License Management Software เช่น vCenter, Veeam ควร Remove Licence Key ออกก่อน

**ทางเลือก: ถอด Drive ออกก่อนขาย**
ถ้าไม่ต้องการจัดการ Data Sanitization เอง สามารถถอด Drive ออกก่อนขาย Server ตัวเปล่าได้ แต่ต้องแจ้งให้ทราบตอนส่งรายการมาประเมิน เพราะราคาต่างกันมาก

## การนัดตรวจ ขนย้าย และรับ Server ตามเงื่อนไข

Server หนักและใหญ่กว่าคอมทั่วไป การรับและขนย้ายจึงต้องวางแผนต่างออกไป

**Server 1–3 เครื่องที่เป็น Tower หรือเครื่องขนาดเล็ก**
ลูกค้าสามารถนำมาที่ร้านได้โดยตรง หรือนัดให้ทางร้านไปรับในพื้นที่ที่เดินทางได้สะดวก

**Server Rack หรือ Server จำนวนหลายเครื่อง**
ส่งรายการและรูปมาประเมินเบื้องต้นก่อน จากนั้นหารือวิธีรับสินค้าที่เหมาะกับสถานที่และจำนวน Server ที่อยู่ใน Rack อาจต้องประสานงานเรื่องการถอด Rack และการขนย้าย

**Server ที่อยู่ใน Data Room หรือห้องที่มีข้อกำหนดการเข้าถึง**
แจ้งเงื่อนไขการเข้าพื้นที่มาตั้งแต่ต้น เพื่อวางแผนวันและเวลานัดตรวจให้ตรงกับข้อกำหนดของสถานที่

**Server ที่อยู่ต่างจังหวัดหรือนอกพื้นที่**
ส่งรายการและรูปมาประเมินเบื้องต้นได้เสมอ โดยไม่ต้องขนมาก่อน จากนั้นหารือวิธีการรับสินค้าตามจำนวนและระยะทาง

## ความต่างระหว่าง Server, NAS และอุปกรณ์ Network

สามประเภทนี้มักอยู่ในห้อง IT เดียวกัน แต่มีวิธีประเมินและมูลค่าต่างกัน

| | **Server** | **NAS** | **Network Equipment** |
|---|---|---|---|
| **ตัวอย่าง** | Dell PowerEdge, HP ProLiant, Lenovo ThinkSystem | Synology, QNAP, NetApp | Cisco Switch, Ubiquiti Router, Aruba AP |
| **ปัจจัยราคาหลัก** | CPU Generation, RAM, Drive, PSU Redundancy | ความจุ Bay, RAM NAS, Model Number | Brand (Cisco ราคาดี), Port Count, Managed/Unmanaged |
| **สิ่งที่ต้องส่งมา** | Service Tag, CPU/RAM/Drive Configuration | Model Number, RAM, จำนวน Bay | Part Number, รุ่น, สภาพ Port |
| **เรื่องข้อมูล** | Secure Erase Drive, Reset iDRAC/iLO | Factory Reset หรือถอด Drive | Factory Reset ลบ Config และ Password |
| **น้ำหนักและการขนย้าย** | หนัก ต้องวางแผน | เบากว่า ขนสะดวกกว่า | เบา ขนได้ง่าย |

ทางร้านรับพิจารณาทั้งสามประเภท แจ้งรุ่นมาพร้อมกันในรายการเดียวได้เลย
