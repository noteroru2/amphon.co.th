- **Server Dell**
- **Server HP**
- **Server Cisco**
- **Server Lenovo**
- **Server Fujitsu**
- **Server Huawei**
- **Server Hitachi**
- **Server IBM**
- **Server Oracle**
- **Server Supermicro**

ไม่เพียงแค่นั้น เรายังรับเหมา**ตู้ Rack Server** ทุกขนาด รวมถึง**รับซื้อ Server เสีย** และ**ซาก Server** ที่ไม่สามารถใช้งานได้แล้ว เราก็พร้อมประเมินราคาให้คุณ

## ตารางประเมินราคารับซื้อ Server โดยประมาณ

| แบรนด์/ประเภท | สภาพ | ราคาประเมินเบื้องต้น |
| :--- | :--- | :--- |
| Server Dell / HP / IBM | ใช้งานได้ (Gen ใหม่ๆ) | 5,000 - 50,000+ บาท |
| Server ทุกแบรนด์ | ตกรุ่น / ปลดระวาง | 1,000 - 10,000 บาท |
| ซาก Server / Server เสีย | ชิ้นส่วนไม่ครบ/เปิดไม่ติด | 300 - 2,000 บาท |
| ตู้ Rack Server | สภาพดี | 1,000 - 5,000 บาท |

*(หมายเหตุ: ราคาประเมินขึ้นอยู่กับ Generation, สเปคของ CPU, RAM, HDD และสภาพของเครื่องเป็นหลัก)*
