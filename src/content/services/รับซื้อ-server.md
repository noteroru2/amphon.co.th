## สภาพ Server แบบไหนที่เรารับซื้อบ้าง?

หลายครั้งที่เซิร์ฟเวอร์ถูกปล่อยทิ้งจนฝุ่นเกาะ หรือมีชิ้นส่วนบางตัวพังเสียหาย เราขอยืนยันว่าเรารับประเมินราคาทุกสภาพ:

1. **สภาพสมบูรณ์ ใช้งานปกติ (ได้ประเมินตามสภาพจริง):** ถอดออกจาก Rack หมาดๆ เปิดติด ไฟเขียว รัน Windows Server หรือ VMware ได้ปกติ
2. **เครื่องผ่านการใช้งาน ต้องการลบข้อมูลก่อน:** เครื่องปกติแต่ทางบริษัทยังไม่กล้าขายเพราะมีข้อมูลค้างอยู่ เราเข้าไปรับซื้อและจัดการลบข้อมูลต่อหน้าท่านได้
3. **บางชิ้นส่วนมีปัญหา (Error):** เช่น เมนบอร์ดฟ้อง Error, การ์ด RAID Controller พัง, หรือพอร์ตแลนเสีย เราประเมินตามอะไหล่ที่เหลือ
4. **ไม่มีฮาร์ดดิสก์ (HDD Removed):** องค์กรถอดฮาร์ดดิสก์ทำลายทิ้งไปแล้ว เรารับซื้อเครื่องเปล่า (Barebone)
5. **Server เก่าหมดประกัน (EOL/EOSL):** แม้แบรนด์จะไม่รองรับการอัปเดตแล้ว แต่ชิ้นส่วนโลหะและแผงวงจรภายในยังสามารถนำไปรีไซเคิลเป็น E-Waste ได้

## 8 ปัจจัยหลักที่กำหนดราคาประเมินเซิร์ฟเวอร์

เพื่อให้ฝ่ายไอทีเข้าใจถึงการตั้งราคา นี่คือสิ่งที่เรานำมาคำนวณมูลค่าเซิร์ฟเวอร์ของคุณ:
1. **รุ่นและ Generation:** ยิ่งรุ่นใหม่ (เช่น Gen10, Gen11 หรือ PowerEdge 14G, 15G) ย่อมได้มูลค่าสูงกว่าแบบก้าวกระโดด
2. **โปรเซสเซอร์ (CPU):** เราดูตระกูล CPU เช่น Intel Xeon (Silver, Gold, Platinum) หรือ AMD EPYC ว่ามีจำนวน Core และ Clock Speed เท่าไหร่
3. **หน่วยความจำ (RAM):** จำนวนความจุทั้งหมด (เช่น 64GB, 128GB, 256GB) ชนิดเป็น DDR4 หรือ DDR5 ECC RDIMM
4. **ฮาร์ดดิสก์ (Storage):** จำนวนลูกดิสก์ ความจุต่อลูก (TB) และประเภท (SAS, SATA Enterprise, NVMe U.2)
5. **เพาเวอร์ซัพพลาย (PSU):** เป็นแบบตัวเดียว (Single) หรือมีระบบสำรอง (Redundant 1+1)
6. **ระบบการจัดการระยะไกล:** ติดตั้ง Enterprise License ของ iDRAC (Dell) หรือ iLO (HP) มาให้หรือไม่
7. **อุปกรณ์เสริมตู้แร็ค:** มี Rail Kit (รางสไลด์) และ Front Bezel (หน้ากาก) มาครบหรือไม่
8. **วันที่สิ้นสุดระยะเวลาสนับสนุน (EOL/EOSL):** สถานะการรับประกันจากผู้ผลิต

## Data Destruction บริการลบข้อมูลเซิร์ฟเวอร์ — ป้องกันข้อมูลรั่วไหลขั้นเด็ดขาด

สำหรับองค์กรที่กังวลเรื่อง PDPA (พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล) เราเข้าใจดีว่า Data Center ของคุณคือหัวใจหลักที่เก็บฐานข้อมูลทางการเงินและลูกค้าทั้งหมด เรามีบริการปกป้องข้อมูลให้คุณ 2 ระดับ:

* **Level 1: Secure Data Erase (NIST 800-88 / DoD 5220.22-M)**  
  การลบข้อมูลระดับลึกบนดิสก์อาร์เรย์ (RAID Arrays) ด้วยซอฟต์แวร์มาตรฐานสากล โดยเขียนข้อมูลทับ (Overwrite) หลายรอบ จนไม่สามารถใช้โปรแกรมกู้ข้อมูลใดๆ ดึงข้อมูลกลับมาได้ (วิธีนี้ทำให้ท่านยังสามารถขายฮาร์ดดิสก์พ่วงกับเซิร์ฟเวอร์เพื่อให้ได้ราคารวมสูงขึ้นได้)
* **Level 2: Physical Shredding (ทำลายเชิงกายภาพ)**  
  หากนโยบายองค์กรห้ามนำดิสก์กลับมาใช้ใหม่ เราบริการนำเครื่องทำลายหรือเจาะทำลายฮาร์ดดิสก์ให้เสียหายทางกายภาพอย่างถาวร 
* **ออกใบรับรอง Certificate of Data Destruction**  
  ระบุ Serial Number ของฮาร์ดดิสก์ทุกลูกที่ถูกทำลาย เพื่อให้บริษัทนำไปใช้แนบในรายงานการตรวจสอบ (Audit Report)

## 5 ขั้นตอนง่ายๆ ในการขาย Server กับ อำพล เทรดดิ้ง

1. **ส่งสเปคแจ้งราคา:** ส่งรายละเอียดยี่ห้อ รุ่น และสเปค (CPU, RAM, HDD) มาทาง Line [@webuy](https://line.me/ti/p/~@webuy) หากไม่ทราบสเปค ให้ถ่ายรูปหน้าจอ iDRAC/iLO หรือหน้าจอ BIOS มาให้เราดู
2. **ประเมินราคาเบื้องต้น:** เราจัดทำใบเสนอราคา (Quotation) ส่งให้ภายใน 24 ชั่วโมง
3. **เข้าดูหน้างาน (Site Audit):** สำหรับโปรเจกต์ 5 เครื่องขึ้นไป หรือการรื้อตู้ Rack ทีมช่างจะขอเข้าสำรวจหน้างานฟรี
4. **ลงนามสัญญาและรับเงิน:** ทางเราสั่งจ่ายเช็ค หรือ โอนเงินเข้าบัญชีนิติบุคคลของบริษัทเต็มจำนวนก่อนขนของ
5. **ถอดถอนและขนย้าย:** ทีมงานเข้ารื้อถอนเซิร์ฟเวอร์ แพ็คใส่พาเลท ยกขึ้นรถบรรทุกอย่างรวดเร็วและปลอดภัย
