## ก่อนล้างเครื่อง MacBook ควรสำรองข้อมูลก่อน

ข้อควรระวังที่สำคัญที่สุดก่อนทำอะไรกับเครื่องคือ การลบข้อมูลหรือ Erase เครื่องจะทำให้ข้อมูลทั้งหมดในเครื่องหายไป หากไม่ได้สำรองข้อมูลไว้ก่อน การกู้คืนภายหลังอาจทำไม่ได้

ก่อนทำขั้นตอนใด ๆ ควรสำรองไฟล์ รูปภาพ เอกสาร งานสำคัญ และข้อมูลอื่น ๆ ออกไปก่อน วิธีที่ทำได้คือคัดลอกไปยัง External Drive หรือ USB, สำรองผ่าน Time Machine ไปยัง External Drive, หรืออัปโหลดขึ้น iCloud หรือ Google Drive ตามที่สะดวก

อย่ากด Erase ถ้ายังไม่แน่ใจว่าสำรองข้อมูลครบแล้ว เพราะถ้าลืมไฟล์สำคัญไว้จะไม่มีทางเอากลับมาได้

## เช็กสเปกและข้อมูลเครื่องก่อนล้าง

ก่อนล้างเครื่อง ควรถ่ายรูปข้อมูลสเปกเก็บไว้ก่อน เพราะหลังล้างแล้วหน้า About This Mac ยังดูได้ แต่ถ้าถ่ายไว้ก่อนจะได้ส่งให้ร้านประเมินได้ทันที

ข้อมูลที่ควรถ่ายรูปเก็บไว้ได้แก่ หน้า About This Mac ที่แสดงรุ่น MacBook Air หรือ Pro, ชิปที่ใช้ว่าเป็น Intel, M1, M2, M3 หรือ M4, ขนาด RAM, ขนาด SSD, Serial Number สำหรับใช้ตรวจรุ่นในภายหลัง และ Cycle Count ของแบตเตอรี่

Cycle Count อาจดูได้จาก System Information > Power หรือเมนู Battery ใน macOS บางเวอร์ชัน หากหาไม่เจอให้ถ่ายรูปหน้า Battery Health หรือ About This Mac ส่งมาก่อนก็ได้ อ่านเพิ่มเติมเรื่อง Cycle Count ได้จากบทความ [MacBook Battery Cycle มีผลต่อราคาขายแค่ไหน](/blog/macbook-battery-cycle-มีผลต่อราคาขายแค่ไหน)

## วิธีออกจาก Apple ID และปิด Find My บน MacBook

ขั้นตอนนี้ต้องใช้รหัสผ่าน Apple ID ของเจ้าของเครื่อง ถ้าลืมรหัสควรกู้บัญชีกับ Apple ที่ iforgot.apple.com ก่อน อย่าพยายามข้ามขั้นตอนนี้ด้วยวิธีอื่น

สำหรับ macOS เวอร์ชันใหม่ เข้า System Settings แล้วคลิกชื่อ Apple ID ที่ด้านบนสุดของแถบด้านซ้าย เลื่อนลงมาที่ด้านล่างสุดของหน้า Apple ID จะเห็นตัวเลือก Find My ให้ปิดก่อน แล้วจึงกด Sign Out ออกจาก Apple ID ระบบจะถามว่าต้องการเก็บสำเนาข้อมูลบางอย่างไว้ในเครื่องไหม ตอบตามต้องการ แล้วยืนยันการออกจากระบบ

หลังออกจาก Apple ID แล้ว ควรตรวจสอบอีกครั้งว่า Find My ถูกปิดเรียบร้อยแล้ว

## Find My และ Activation Lock สำคัญอย่างไร

Find My เป็นระบบของ Apple ที่ใช้ติดตามเครื่องหายหรือเครื่องถูกขโมย เมื่อ Find My เปิดอยู่ เครื่องจะอยู่ภายใต้ Activation Lock ซึ่งทำให้ผู้ที่ได้เครื่องไปไม่สามารถเปิดใช้งานหรือติดตั้ง macOS ใหม่ได้โดยไม่ใช้ Apple ID ของเจ้าของเดิม

ร้านไม่สามารถรับซื้อเครื่องที่ยังติด Find My หรือ Activation Lock อยู่ได้ เพราะผู้ซื้อต่อจะไม่สามารถนำเครื่องไปใช้งานได้ตามปกติ และเครื่องที่สถานะไม่ชัดเจนก็ไม่สามารถยืนยันความเป็นเจ้าของได้

สำหรับเครื่องบริษัท โรงเรียน หรือเครื่องที่ติด MDM (Mobile Device Management) ซึ่งยังอยู่ภายใต้การจัดการของหน่วยงาน ต้องให้หน่วยงานนั้นปลดออกจากระบบก่อน ร้านไม่สามารถรับซื้อได้จนกว่าจะปลดล็อกแล้ว

## วิธีลบข้อมูล MacBook ก่อนส่งมอบ

หลังออกจาก Apple ID และปิด Find My แล้ว ขั้นตอนสุดท้ายก่อนส่งมอบเครื่องคือการลบข้อมูลออก

**MacBook ชิป Apple Silicon (M1, M2, M3, M4)** — macOS รุ่นใหม่มีตัวเลือก Erase All Content and Settings ใน System Settings > General > Transfer or Reset เลือก Erase All Content and Settings แล้วทำตามขั้นตอนที่ระบบแนะนำ วิธีนี้จะออกจาก Apple ID ลบข้อมูล และรีเซ็ตเครื่องให้พร้อมสำหรับเจ้าของใหม่โดยอัตโนมัติ

**MacBook Intel รุ่นเก่า** — อาจต้องรีสตาร์ทเข้า Recovery Mode แล้วใช้ Disk Utility ลบพาร์ทิชัน จากนั้นติดตั้ง macOS ใหม่ ขั้นตอนนี้ค่อนข้างละเอียดและอาจแตกต่างกันตามรุ่นของ macOS

ถ้าไม่แน่ใจว่าเครื่องของตัวเองควรทำแบบไหน หรือกลัวทำผิดพลาด สามารถถามร้านหรือช่างที่ไว้ใจได้ก่อน อย่าเพิ่งกดถ้าไม่แน่ใจ เพราะขั้นตอนนี้ถ้าทำผิดอาจทำให้เครื่องใช้งานไม่ได้ชั่วคราว
