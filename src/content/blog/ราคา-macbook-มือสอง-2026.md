## ทำไมราคา MacBook มือสองถึงยังดีในปี 2026

MacBook เป็นสินค้าไอทีที่มีราคามือสองค่อนข้างดีและตกช้ากว่าสินค้าประเภทอื่น เนื่องจากชิป Apple Silicon (M1 M2 M3) ยังคงมีประสิทธิภาพสูงและรองรับ macOS เวอร์ชันล่าสุดได้ดี MacBook Air M1 ที่ซื้อมาปี 2021 ในปี 2026 ก็ยังมีราคามือสองที่ดีอยู่

อย่างไรก็ตาม ราคา MacBook มือสองในแต่ละเครื่องต่างกันมากขึ้นอยู่กับรุ่น สเปก และสภาพ เครื่องที่ดูคล้ายกันภายนอกอาจราคาต่างกันหลักหมื่นถ้าสเปกต่างกัน

## ปัจจัยหลักที่มีผลต่อราคา MacBook มือสอง

### รุ่นชิปและปีที่ผลิต

ชิปเจนเนอเรชันที่ใหม่กว่ามักได้ราคาดีกว่า โดยลำดับความต้องการในตลาดมือสอง 2026 คือ M3 > M2 > M1 >> Intel (MacBook pre-2020 ชิป Intel ราคาตกเยอะแล้ว)

### RAM และ Storage

- RAM 8GB vs 16GB ต่างกันพอสมควร ตลาดมือสองนิยม 16GB มากกว่าในปี 2026 เพราะซอฟต์แวร์เริ่มกิน RAM มากขึ้น
- SSD 256GB vs 512GB ต่างกันพอสมควร
- MacBook Air M2 ที่มี RAM 16GB / 512GB ราคาสูงกว่า RAM 8GB / 256GB อย่างเห็นได้ชัด

### Battery Cycle Count

แตกต่างจาก iPhone ที่ดู Battery Health เป็น % MacBook ใช้ Battery Cycle Count แทน MacBook ออกแบบมารองรับ 1,000 cycles Apple ถือว่าหลัง 1,000 cycles แบตเตอรี่ยังทำงานได้แต่ประสิทธิภาพลดลง ดู Cycle Count ได้ที่ About This Mac > System Information > Power

- 0–300 cycles: สภาพดีมาก
- 300–600 cycles: สภาพดี ราคาดี
- 600–900 cycles: ใช้งานได้ ราคาลดลงบ้าง
- 900+ cycles: ควรแจ้งก่อนขาย ราคาต้องปรับ

### สภาพจอและ Palm Rest

รอยขีดข่วนบน Palm Rest (บริเวณที่วางมือ) เป็นเรื่องปกติมากในตลาดมือสอง แต่รอยบนจอหรือจอมีปัญหา (dead pixel, backlight bleeding) มีผลต่อราคาพอสมควร เพราะค่าซ่อมจอ MacBook แพง

### กล่องและอุปกรณ์ครบ

MacBook มีกล่องสวย กล่องครบและ adapter ช่วยเพิ่มราคาได้บ้าง แต่ไม่มีก็ยังขายได้

## ข้อมูลที่ต้องส่งก่อนประเมิน MacBook

ต่างจาก iPhone ที่มีข้อมูลอยู่ในตัวเครื่องง่ายๆ MacBook มีข้อมูลสำคัญที่ต้องตรวจเพิ่มเติม

### ตรวจ Battery Cycle Count

1. กด Apple menu > About This Mac
2. กด System Information (หรือ System Report)
3. ไปที่ Hardware > Power
4. ดูที่ Cycle Count

### ตรวจ RAM และ Storage

1. กด Apple menu > About This Mac
2. ดู Memory และ Storage

### ถ่ายรูปที่ควรส่ง

- รูปหน้าจอตอนเปิดใช้งาน (เห็น Desktop)
- รูป About This Mac แสดง chip, RAM, storage
- รูป Battery Cycle Count
- รูปสภาพตัวเครื่องด้านบน ด้านล่าง ด้านข้าง
- รูปรอยขีดข่วนหรือตำหนิที่พบ
- รูปจอ (ถ่ายบนพื้นหลังขาวเพื่อเห็น dead pixel)

## เตรียมขาย MacBook อย่างไรให้ได้ราคาดี

ก่อนส่งรูปประเมินหรือนำสินค้าไปขาย ควรดำเนินการดังนี้

1. **ออกจาก Apple ID** — System Settings > ชื่อของคุณ > Sign Out
2. **ออกจาก iCloud, iMessage, FaceTime** — ออกทุก service
3. **Erase All Content** — ทำ Erase Mac หรือ Factory Reset
4. **ทำความสะอาดเครื่อง** — โดยเฉพาะ Palm Rest และแป้นพิมพ์
5. **รวบรวมอุปกรณ์** — power adapter, กล่อง, ใบเสร็จ ถ้ามี

## MacBook Intel ยังขายได้ไหมในปี 2026

หากมี MacBook Intel แนะนำให้ส่งรูปมาประเมินก่อน เพื่อให้ทราบว่ายังมีมูลค่าเท่าไหร่ในตลาดปัจจุบัน
