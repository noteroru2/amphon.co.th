## การ์ดจอมีผลต่อราคาคอมมือสองแค่ไหน

ถ้าเคยส่งรูปคอมมาให้ร้านประเมิน สิ่งหนึ่งที่มักถูกถามกลับเร็วที่สุดคือ "การ์ดจอรุ่นอะไรครับ" ไม่ใช่แค่ถามเพื่อให้ข้อมูลครบ แต่เพราะการ์ดจอมีผลต่อราคาประเมินคอมมือสองโดยตรง โดยเฉพาะในกลุ่มคอมเกมมิ่งและคอมประกอบ

ลองนึกภาพคอมสองเครื่องที่ใช้ CPU ใกล้เคียงกัน RAM เท่ากัน SSD ขนาดใกล้กัน แต่เครื่องหนึ่งมีการ์ดจอแยกรุ่นกลางขึ้นไป อีกเครื่องมีแค่กราฟิกในตัว CPU สองเครื่องนั้นอาจได้ราคาประเมินต่างกันมาก และถ้าเป็นการ์ดจอที่ต่างกันสองรุ่น ราคาอาจต่างกันได้มากกว่านั้นอีก

## ทำไมการ์ดจอ GPU ถึงสำคัญกับราคาคอมมือสอง

GPU หรือการ์ดจอคือชิ้นส่วนที่รับผิดชอบการประมวลผลภาพและกราฟิก ใช้งานหนักในงานเล่นเกม ตัดต่อวิดีโอ ทำกราฟิก 3D และงาน AI บางประเภท

ในตลาดมือสอง คอมที่มีการ์ดจอแยกมักขายต่อได้ง่ายกว่าและได้ราคาดีกว่าคอมที่มีแต่กราฟิกในตัว CPU เพราะผู้ซื้อต่อส่วนใหญ่ต้องการนำไปใช้เล่นเกมหรืองานที่ต้องการ GPU จริง

รุ่นของการ์ดจอก็สำคัญมาก เช่น การ์ดจอ RTX รุ่นใหม่กว่ามักได้ราคาดีกว่า GTX รุ่นเก่าในระดับราคาเดิม และการ์ดจอที่มี VRAM สูงกว่าก็มีความต้องการจากกลุ่มงานสร้างสรรค์ด้วย

## การ์ดจอแบบไหนที่ตลาดมือสองให้ความสนใจ

### NVIDIA GTX และ RTX

การ์ดจอฝั่ง NVIDIA แบ่งออกเป็นสองซีรีส์หลักที่พบบ่อยในตลาดมือสอง

GTX เป็นซีรีส์เก่ากว่า ยังมีตลาดในกลุ่มที่ต้องการคอมสำหรับเล่นเกมระดับกลางหรืองานเบาและไม่ได้ต้องการสเปกสูงมาก แต่ความต้องการลดลงเรื่อย ๆ ตามอายุของรุ่น

RTX เป็นซีรีส์ที่ยังคงมีความต้องการในตลาดสูงกว่า โดยเฉพาะรุ่นกลางขึ้นไปที่รองรับงานเกมและกราฟิกสมัยใหม่ได้ดี RTX ที่สภาพดีและไม่เคยใช้งานหนักผิดปกติมักให้ราคาประเมินได้ดีกว่าในกลุ่มคอมเกมมิ่ง

### AMD Radeon

การ์ดจอ AMD Radeon มีตลาดในกลุ่มคอมเกมมิ่งและคนที่ประกอบคอมเองซึ่งเลือกฝั่ง AMD สภาพและรุ่นของ Radeon มีผลต่อราคาเช่นเดียวกับ NVIDIA โดยรุ่นใหม่กว่าและ VRAM สูงกว่ามักได้ราคาดีกว่า ประวัติการใช้งานก็มีผลในกลุ่มนี้เช่นกัน

### การ์ดจอออนบอร์ดหรือ iGPU

iGPU คือกราฟิกที่ฝังมาใน CPU อยู่แล้ว เช่น Intel UHD Graphics หรือ AMD Radeon Graphics ที่มาพร้อม Ryzen บางรุ่น ใช้งานทั่วไปได้ดี เช่น เปิดเว็บ ทำงาน Office ดูหนัง แต่ไม่เหมาะสำหรับเล่นเกมหนักหรืองานกราฟิกจริงจัง

คอมที่มีแค่ iGPU มักอยู่ในกลุ่มคอมสำนักงานหรือคอมใช้งานทั่วไป ราคาประเมินมักต่างจากคอมที่มี GPU แยกในสเปกอื่นที่ใกล้เคียงกัน

## ปัจจัยที่ทำให้การ์ดจอได้ราคาดีหรือราคาตก

**รุ่นและซีรีส์** — RTX รุ่นใหม่กว่า GTX รุ่นเก่าในระดับเดิม รุ่นที่ยังมีการสนับสนุนจากผู้ผลิตและ driver ใหม่อยู่มักได้ราคาดีกว่า

**VRAM** — ขนาด VRAM มีผลต่อประสิทธิภาพงานกราฟิก 6GB ขึ้นไปเริ่มเป็นที่ต้องการสำหรับงานเกมและกราฟิก 8GB และ 12GB ยิ่งดีกว่าในงานระดับสูง

**สภาพพัดลมและอุณหภูมิ** — พัดลมที่หมุนได้ปกติและเงียบ บ่งชี้ว่าดูแลมาดีและระบายความร้อนได้ปกติ

**ฝุ่น สนิม คราบ ความชื้น** — การ์ดจอที่มีฝุ่นหนาหรือมีร่องรอยของความชื้นมีผลต่อความน่าเชื่อถือและราคา

**ประวัติการซ่อม** — การ์ดจอที่เคยซ่อมมาก่อน โดยเฉพาะส่วนที่เกี่ยวกับชิปหลักหรือ VRAM มีผลต่อการประเมิน

**ประวัติการใช้งาน** — การ์ดจอที่เคยใช้ขุดเหรียญคริปโตมักทำงานหนักต่อเนื่องเป็นเวลานาน ซึ่งมีผลต่อความทนทานของชิ้นส่วน อ่านเพิ่มได้จากบทความ [การ์ดจอเคยขุดหรือใช้งานหนัก ขายได้ไหม](/blog/การ์ดจอเคยขุด-ขายได้ไหม-เช็กอย่างไร)

**กล่อง ใบเสร็จ ประกัน** — มีผลบ้าง การ์ดจอที่ยังมีประกันเหลือให้ราคาดีขึ้น

**ความต้องการตลาดในช่วงนั้น** — ราคาการ์ดจอมือสองเปลี่ยนแปลงค่อนข้างเร็ว ขึ้นกับรุ่นใหม่ที่ออกมา ราคาการ์ดจอใหม่ และสภาพเศรษฐกิจโดยรวม

**PSU และสเปกเครื่องโดยรวม** — สำหรับคอมประกอบ PSU ที่รองรับการ์ดจอได้พอดีและสเปกเครื่องที่ balance กันมีผลต่อมูลค่ารวมของชุดคอม

## คอมเกมมิ่งกับคอมประกอบ ทำไมต้องดู GPU ก่อน

คอมเกมมิ่งส่วนใหญ่ราคาประเมินขึ้นกับ GPU มากกว่าชิ้นส่วนอื่น บางเครื่องที่ใช้ CPU ระดับกลางแต่มีการ์ดจอที่ตลาดต้องการยังขายต่อได้ง่ายกว่าเครื่องที่ CPU ดีแต่การ์ดจอเก่ามาก

สิ่งที่ทำให้เกิดความเข้าใจผิดบ่อยคือการบอกว่า "ขายคอมเกมมิ่ง" แล้วหยุด โดยไม่ได้แจ้งรุ่น GPU มาด้วย ร้านจะยังตีราคาไม่ได้จนกว่าจะรู้ว่าการ์ดจอรุ่นอะไร สภาพเป็นอย่างไร และ PSU Watt เท่าไหร่

การส่งสเปกมาครบตั้งแต่ต้น ทั้ง CPU, RAM, SSD, GPU และ PSU จะช่วยให้ได้ราคาเบื้องต้นที่ใกล้เคียงกับราคาจริงมากขึ้น ดูรายละเอียดเพิ่มที่หน้า [รับซื้อคอมเกมมิ่ง](/บริการ/รับซื้อคอมเกมมิ่ง) และ [รับซื้อคอมประกอบ](/บริการ/รับซื้อคอมประกอบ)

## วิธีเช็กรุ่นการ์ดจอก่อนส่งประเมิน

### ดูจาก Task Manager

กด Ctrl + Shift + Esc เพื่อเปิด Task Manager คลิกแท็บ Performance แล้วเลือก GPU ที่แถบด้านซ้าย จะเห็นชื่อรุ่นการ์ดจอและข้อมูล VRAM ถ่ายรูปหน้าจอส่งมาได้เลย ถ้าเครื่องมีการ์ดจอสองตัว จะเห็น GPU 0 และ GPU 1 ให้ถ่ายทั้งสองหน้า

### ดูจาก Device Manager

คลิกขวาที่ปุ่ม Start เลือก Device Manager แล้วขยาย Display adapters จะเห็นชื่อการ์ดจอทั้งหมดที่ติดตั้งอยู่ ถ้าเครื่องมีทั้งกราฟิกในตัว CPU เช่น Intel UHD หรือ AMD Radeon Graphics และการ์ดจอแยก เช่น NVIDIA RTX หรือ AMD Radeon RX ให้ถ่ายรูปให้เห็นชื่อทั้งสองตัว เพราะการ์ดจอแยกคือตัวที่มีผลต่อราคา

### ดูจากตัวการ์ดจอในเคส

สำหรับคอมประกอบที่เปิดเคสได้ ให้ถ่ายรูปตัวการ์ดจอด้านข้างหรือด้านบน มักจะเห็นชื่อรุ่นอยู่บนแผ่นระบายความร้อนหรือพัดลม ถ้ามีสติกเกอร์รุ่นอยู่ที่ตัวการ์ดให้ถ่ายด้วย ไม่จำเป็นต้องถอดการ์ดจอออกมาเองถ้าไม่แน่ใจ ถ่ายรูปในเคสก็เพียงพอสำหรับการประเมินเบื้องต้น

### ใช้โปรแกรม GPU-Z ถ้าทำได้

GPU-Z เป็นโปรแกรมฟรีที่แสดงข้อมูลการ์ดจอละเอียด เช่น ชื่อรุ่นเต็ม ขนาด Memory และ Bus interface แต่ไม่จำเป็นต้องติดตั้งสำหรับลูกค้าทั่วไป ข้อมูลจาก Task Manager หรือ Device Manager ก็เพียงพอสำหรับการประเมินเบื้องต้น

## รูปและข้อมูลที่ควรส่งให้ร้านประเมินการ์ดจอ

- รูปตัวเครื่องหรือเคสด้านหน้า
- รูปด้านในเคสที่เห็นการ์ดจออย่างชัดเจน
- รูปชื่อ GPU จาก Task Manager > Performance > GPU
- รูป Device Manager > Display adapters
- รูปตัวการ์ดจอโดยตรง ถ้าถ่ายได้โดยไม่ต้องถอด
- แจ้งว่าการ์ดจอเคยซ่อมมาก่อนไหม
- แจ้งว่าเคยใช้ขุดเหรียญหรือใช้งานต่อเนื่องยาวนานผิดปกติไหม ถ้าทราบ
- แจ้งว่ามีกล่อง ใบเสร็จ หรือประกันเหลือไหม
- แจ้งอาการผิดปกติ เช่น ภาพลาย จอดับบางครั้ง พัดลมดังผิดปกติ หรือเครื่องร้อนมากกว่าปกติ
