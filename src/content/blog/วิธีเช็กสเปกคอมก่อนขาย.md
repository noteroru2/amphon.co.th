## ข้อมูลสเปกอะไรที่ร้านดูเป็นหลัก

### CPU

CPU หรือโปรเซสเซอร์คือ "สมอง" ของเครื่อง และเป็นหนึ่งในปัจจัยที่มีน้ำหนักมากที่สุดในการประเมินราคา

ฝั่ง Intel จะใช้ชื่ออย่าง Core i3, Core i5, Core i7, Core i9 หรือรุ่นใหม่ที่เรียกว่า Core Ultra ฝั่ง AMD จะใช้ Ryzen 3, Ryzen 5, Ryzen 7, Ryzen 9

สิ่งที่สำคัญไม่แพ้ชื่อรุ่นคือ "Generation" หรือรุ่นปี ตัวเลขหลัง i5 หรือ Ryzen 5 บ่งบอก Gen ได้ เช่น i5-12400 คือ Gen 12 ซึ่งใหม่กว่า i7-8700 ที่เป็น Gen 8 ในบางกรณี i5 รุ่นใหม่อาจให้ราคามือสองดีกว่า i7 รุ่นเก่าได้ เพราะผู้ซื้อต่อดูประสิทธิภาพจริง ไม่ได้ดูแค่ชื่อรุ่น

สิ่งที่ควรส่งมา คือชื่อ CPU เต็ม เช่น "Intel Core i5-12400F" หรือ "AMD Ryzen 5 5600X" ถ่ายรูปจากหน้า Settings หรือ Task Manager ก็พอ

### RAM

RAM คือหน่วยความจำที่ใช้ขณะเครื่องทำงาน แบ่งเป็นสองสิ่งที่ควรรู้ก่อนส่งประเมิน คือขนาดและประเภท

ขนาด 4GB ในปัจจุบันตลาดมือสองความต้องการน้อยลงมากแล้ว 8GB พอใช้งานทั่วไปได้ และ 16GB ขึ้นไปเป็นที่ต้องการในกลุ่มคนทำงานและเกม

ประเภท DDR4 และ DDR5 ก็มีผลต่อราคา เครื่องที่ใช้ DDR5 มักเป็นรุ่นใหม่กว่าและให้ราคาดีกว่าในสเปกที่คล้ายกัน

สำหรับโน๊ตบุ๊ค บางรุ่น RAM บัดกรีติดเมนบอร์ดมา ไม่สามารถเพิ่มหรือเปลี่ยนได้ ถ้าทราบให้แจ้งมาด้วย เพราะมีผลต่อการตัดสินใจซื้อต่อ

### SSD / HDD

เครื่องที่มี SSD ให้ราคาดีกว่าเครื่องที่ยังใช้ HDD ในหลายกรณี เพราะผู้ซื้อต่อมักต้องการเครื่องที่เปิดเร็วและโหลดโปรแกรมไวกว่า

ขนาดความจุก็มีผล 256GB ถือว่าน้อยสำหรับการใช้งานปัจจุบัน 512GB ขึ้นไปเป็นที่ต้องการมากกว่า 1TB ขึ้นไปให้ราคาดีกว่าในกลุ่มเดียวกัน

ถ้าเครื่องใช้ SSD มาหลายปีและเขียนข้อมูลเยอะมาก สุขภาพของ SSD อาจมีผลต่อราคาด้วย อ่านเพิ่มได้จากบทความ [SSD สุขภาพเหลือน้อย เขียนข้อมูลเยอะ ขายได้ไหม](/blog/ssd-สุขภาพเหลือน้อย-เขียนข้อมูลเยอะ-ขายได้ไหม)

### การ์ดจอ GPU

สำหรับคอมใช้งานทั่วไปหรือโน๊ตบุ๊คที่มีแค่การ์ดจอในตัว (Integrated Graphics) ปัจจัยนี้มีน้ำหนักน้อย แต่สำหรับคอมเกมมิ่งและคอมประกอบ การ์ดจอคือปัจจัยที่มีน้ำหนักมากที่สุดในการประเมินราคา

การ์ดจอที่พบบ่อยคือฝั่ง NVIDIA ซีรีส์ GTX และ RTX และฝั่ง AMD ซีรีส์ Radeon รุ่นการ์ดจอและสภาพการใช้งานมีผลต่อราคา เช่น เครื่องที่เคยใช้ขุดเหรียญมาอาจได้ราคาต่างจากเครื่องที่ใช้งานเล่นเกมปกติ

อ่านเพิ่มเติมได้จากบทความ [การ์ดจอเคยขุดหรือใช้งานหนัก ขายได้ไหม](/blog/การ์ดจอเคยขุด-ขายได้ไหม-เช็กอย่างไร)

### เมนบอร์ด PSU และเคส สำหรับคอมประกอบ

ถ้าเป็นคอมประกอบหรือคอมเกมมิ่งที่ประกอบเอง นอกจาก CPU, RAM, SSD, GPU แล้ว ยังมีชิ้นส่วนอื่นที่ควรแจ้ง ได้แก่

เมนบอร์ด Socket และ Chipset มีผลต่อความเข้ากันได้กับ CPU PSU (Power Supply Unit) Watt และแบรนด์มีผลต่อความน่าเชื่อถือของชุดคอม เคสขนาดและสภาพ รวมถึงชุดระบายความร้อน ไม่ว่าจะเป็น Air Cooler หรือระบบน้ำ

สำหรับชิ้นส่วนเหล่านี้ ถ่ายรูปด้านในเคสให้เห็นการ์ดจอ PSU และสายต่างๆ ก็เพียงพอสำหรับการประเมินเบื้องต้น

## วิธีเช็กสเปก Windows 10/11 แบบง่าย

### ดูจาก Settings

วิธีนี้เหมาะกับการดูข้อมูลพื้นฐานอย่าง CPU และ RAM อย่างรวดเร็ว

กด Start แล้วเข้า Settings จากนั้นเลือก System แล้วเลื่อนลงไปคลิก About จะเห็นข้อมูล Processor (CPU), Installed RAM และ Device name (รุ่นเครื่อง) ถ่ายรูปหน้านี้ส่งมาได้เลย

### ดูจาก Task Manager

วิธีนี้เหมาะสำหรับถ่ายรูปส่งให้ร้าน เพราะเห็นข้อมูลได้ครบในหน้าเดียว

กด Ctrl + Shift + Esc เพื่อเปิด Task Manager จากนั้นคลิกแท็บ Performance ด้านซ้ายจะเห็นรายการ CPU, Memory, Disk และ GPU คลิกทีละอันแล้วถ่ายรูปหน้าจอส่งมา จะได้ข้อมูลครบทั้ง CPU รุ่น, RAM ขนาด, SSD/HDD ความจุ และการ์ดจอชื่อ

### ดูจาก System Information

เหมาะสำหรับโน๊ตบุ๊คแบรนด์ที่ต้องการรู้รุ่นเต็มของเครื่อง

กด Windows + R แล้วพิมพ์ msinfo32 กด Enter จะเห็น System Model (รุ่นเครื่อง), Processor (CPU) และ Installed Physical Memory (RAM) รุ่นเครื่องที่แสดงตรงนี้ช่วยให้ค้นหาสเปกเพิ่มเติมได้ง่ายขึ้น

### ดูการ์ดจอจาก Device Manager

วิธีนี้ช่วยให้เห็นชื่อการ์ดจอทั้งหมดในเครื่อง

คลิกขวาที่ปุ่ม Start แล้วเลือก Device Manager เลื่อนหาหัวข้อ Display adapters แล้วคลิกขยาย จะเห็นชื่อการ์ดจอที่ติดตั้งอยู่

ถ้าเครื่องมีทั้งการ์ดจอในตัว (เช่น Intel UHD หรือ AMD Radeon Graphics) และการ์ดจอแยก (เช่น NVIDIA RTX หรือ AMD Radeon RX) ให้ถ่ายรูปให้เห็นทั้งสองตัว เพราะการ์ดจอแยกคือสิ่งที่มีผลต่อราคา

## วิธีเช็กสเปกโน๊ตบุ๊คก่อนขาย

โน๊ตบุ๊คมีข้อมูลที่ควรดูเพิ่มจากคอมตั้งโต๊ะ

ดูรุ่นเครื่องจากสติกเกอร์ใต้เครื่องหรือข้างเครื่อง บางรุ่นมีแผ่นสติกเกอร์ที่ระบุรุ่น CPU, RAM, SSD ไว้ชัดเจน ดูกล่องหรือใบเสร็จถ้าเก็บไว้ มักระบุสเปกครบ ถ่ายรูปหน้า Settings > About เพื่อให้เห็น CPU และ RAM แจ้งสภาพจอ คีย์บอร์ด ฝาพับ และมุมเครื่อง แจ้งว่าแบตยังเก็บไฟได้ดีไหม และชาร์จติดปกติไหม

## วิธีเช็กสเปกคอมประกอบและคอมเกมมิ่ง

คอมประกอบหรือคอมเกมมิ่งที่ประกอบเองมีชิ้นส่วนหลายอย่าง ควรแจ้งแยกชิ้นเพื่อให้ประเมินได้ครบ

ชิ้นส่วนที่ควรแจ้ง ได้แก่ CPU รุ่นเต็ม, Mainboard รุ่นและ Socket, RAM ขนาดและ Gen, SSD/HDD ความจุ, GPU ชื่อรุ่นเต็ม, PSU Watt และแบรนด์, เคสขนาด, ชุดน้ำหรือพัดลมถ้ามี

รูปที่ควรถ่าย คือรูปด้านในเคสที่เห็นการ์ดจอ PSU และการจัดสายครบ รูปการ์ดจอโดยเฉพาะ รูปหน้า Task Manager Performance ที่แสดง CPU, Memory, GPU และ Disk

ดูรายละเอียดเพิ่มได้ที่หน้า [รับซื้อคอมประกอบ](/บริการ/รับซื้อคอมประกอบ), [รับซื้อคอมเกมมิ่ง](/บริการ/รับซื้อคอมเกมมิ่ง) และ [รับซื้อการ์ดจอ](/บริการ/รับซื้อการ์ดจอ)

## รูปและข้อมูลที่ควรส่งให้ร้านประเมิน

รวบรวมไว้ในรายการเดียว ถ้าส่งมาครบตามนี้จะประเมินได้เร็วที่สุด

- รูปหน้าเครื่องหรือหน้าเคส
- รูปด้านข้างและด้านหลัง
- รูปหน้าจอขณะเปิดติด
- รูปหน้า Settings > System > About
- รูป Task Manager > Performance (CPU, Memory, Disk, GPU)
- รูป Device Manager > Display adapters ถ้าเครื่องมีการ์ดจอ
- รูป SSD หรือ HDD ถ้าถอดออกมาได้หรือเห็นได้ในเคส
- รูปการ์ดจอและ PSU สำหรับคอมประกอบ
- แจ้งอาการเสียหรือตำหนิ ถ้ามี
- แจ้งว่ามีกล่อง ใบเสร็จ อุปกรณ์ หรือสายชาร์จไหม
- แจ้งว่าต้องการขายทั้งชุดหรือแยกชิ้น

## ถ้าจำสเปกไม่ได้หรือเปิดเครื่องไม่ติด ทำอย่างไร

ไม่ต้องกังวลครับ ถ้าไม่รู้สเปกหรือจำไม่ได้ ส่งรูปตัวเครื่อง รูปสติกเกอร์ข้างเครื่อง หรือแจ้งรุ่นเครื่องที่พอทราบมาก่อนได้เลย ร้านจะช่วยค้นหาสเปกเพิ่มเติมให้ในกรณีที่ทำได้

ถ้าเครื่องยังเปิดติดอยู่ ร้านอาจแนะนำให้ถ่ายรูปหน้าสเปกเพิ่มตามที่บอกไว้ด้านบน

ถ้าเปิดเครื่องไม่ติด ให้แจ้งอาการว่าเป็นอย่างไร เช่น มีไฟเข้าแต่ไม่ขึ้นภาพ, เปิดแล้วดับทันที, จอฟ้าค้าง, หรือไม่มีสัญญาณภาพเลย ข้อมูลเหล่านี้ช่วยให้ร้านพิจารณาได้ว่าน่าจะมีราคาเท่าไหร่แม้ไม่ได้ดูสเปกจากหน้าจอโดยตรง

อ่านเพิ่มเติมจากบทความ [โน๊ตบุ๊คเปิดไม่ติด ขายได้ไหม](/blog/โน๊ตบุ๊คเปิดไม่ติด-ขายได้ไหม) และถ้าเครื่องเสียหนักหรือต้องการขายเป็นอะไหล่ ดูได้ที่หน้า [รับซื้อของไอทีเสีย](/บริการ/รับซื้อของไอทีเสีย)
